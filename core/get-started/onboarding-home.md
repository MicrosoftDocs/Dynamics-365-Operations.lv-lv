# <a name="onboarding-home-page"></a>Pievienošanās sākumlapa

[!include[banner](../includes/banner.md)]


Lai sāktu lietot sistēmu Microsoft Dynamics 365 for Operations, varat izvēlēties sev vispiemērotāko abonementa līmeni, varat atrast partneri ar pieredzi jūsu reģionā vai nozarē, un varat lietot programmu Microsoft FastTrack, lai ātri uzsāktu darbu.  

Tālāk norādītajos resursos ir sniegts apskats par Lifecycle Services un veidu, kā sadarboties ar partneri, lai jūs tiktu pievienots sistēmā Dynamics 365 for Operations.
