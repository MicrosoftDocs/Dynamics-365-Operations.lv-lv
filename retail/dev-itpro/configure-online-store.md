# <a name="configure-an-online-store"></a>Tiešsaistes veikala konfigurēšana

Šis raksts sniedz saites uz tēmām, kas palīdzēs jums centralizēti konfigurēt un pārvaldīt un glabāt tiešsaistē no Microsoft Dynamics 365 operācijām.

Tabulā minētajiem jautājumiem palīdz konfigurēt Dynamics 365 operācijām - mazumtirdzniecības detaļām un tiešsaistes veikala programmā Dynamics 365 operācijām klientu.

| Konfigurēt mazumtirdzniecības komponentus.                        | Iestatītu un uzturētu informāciju par mazumtirdzniecības operācijas. | Konfigurēt mazumtirdzniecības kanāla navigācijas hierarhijā.    | Jāizveido mazumtirdzniecības kanāla navigācijas kategoriju hierarhija, lai iestatītu kategoriju struktūru attiecībā uz produktiem, ko jūs piedāvājat, izmantojot tiešsaistes veikalu. Definējiet kategorijas hierarhija un piešķirt kategorijas produktiem, produktu atribūtu grupām un atribūtu vērtības. Pēc tam piešķirt kategoriju hierarhija tiešsaistes veikalā.                            | [Iestatītu mazumtirdzniecības hierarhijas](https://technet.microsoft.com/en-us/library/hh580593.aspx) (TechNet saturu Microsoft Dynamics AX 2012) [iestatīt atribūtu un atribūta tipu](https://technet.microsoft.com/en-us/library/hh227548.aspx) (TechNet saturu Microsoft Dynamics AX 2012) [iestatīt atribūtu mazumtirdzniecības grupas](https://technet.microsoft.com/en-us/library/jj728713.aspx) (TechNet saturu Microsoft Dynamics AX 2012) |
| Tiešsaistes veikalu pievienotu organizācijas hierarhijā. | Pirms var piešķirt preču sortiments vai izpildītu pasūtījumus tiešsaistes veikalu, kas izveidota vai izveidot pārskatus, kas ietver informāciju, ka tiešsaistes veikalā, veikals ir jāpiešķir vienu vai vairākas organizācijas hierarhijām.
