# <a name="configure-an-online-store"></a>Tiešsaistes veikala konfigurēšana

[!include[banner](../includes/banner.md)]


Šajā rakstā ir saites uz tēmām, kas jums palīdzēs centrāli konfigurēt un pārvaldīt tiešsaistes veikalu no sistēmas Microsoft Dynamics 365 for Operations.

Nākamajā tabulā uzskaitītās tēmas jums palīdzēs konfigurēt Dynamics 365 for Operations - Mazumtirdzniecība komponentus un mazumtirdzniecības tiešsaistes veikalu Dynamics 365 for Operations klientā.
