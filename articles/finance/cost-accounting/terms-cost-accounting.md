[!include [banner](../includes/banner.md)]

<span data-ttu-id="19c50-104">Šajā tēmā tiek definēti galvenie termini, kas tiek izmantoti Izmaksu uzskaitē.</span><span class="sxs-lookup"><span data-stu-id="19c50-104">This topic defines the key terms that are used in Cost accounting.</span></span>

<span data-ttu-id="19c50-105">**Sadalījuma pamats**</span><span class="sxs-lookup"><span data-stu-id="19c50-105">**Allocation base**</span></span>

<span data-ttu-id="19c50-106">Sadalījuma pamats tiek izmantots, lai mērītu aktivitātes, piemēram, izmantotās mašīnstundas, patērētās kilovatstundas (kWh) vai aizņemto platību, un lai noteiktu šo aktivitāšu daudzumu.</span><span class="sxs-lookup"><span data-stu-id="19c50-106">The allocation base is used to measure and quantify activities, such as machine hours that are used, kilowatt hours that are consumed, or square footage that is occupied.</span></span> <span data-ttu-id="19c50-107">Tas tiek izmantots kā bāze izmaksu sadalīšanai vienam vai vairākiem izmaksu objektiem</span><span class="sxs-lookup"><span data-stu-id="19c50-107">It's used as basis for allocating costs to one or more cost objects</span></span>

<span data-ttu-id="19c50-108">**Izmaksu uzskaite**</span><span class="sxs-lookup"><span data-stu-id="19c50-108">**Cost accounting**</span></span>

<span data-ttu-id="19c50-109">Izmaksu uzskaite ļauj jums apkopot datus no dažādiem avotiem, piemēram, virsgrāmata, apakšgrāmata, budžeti un statistiska informācija.</span><span information.</span></span> <span data-ttu-id="19c50-110">Pēc tam jūs varat analizēt, apkopot un novērtēt izmaksu datus, tādējādi vadība var pieņemt vislabākos lēmumus cenas pielāgošanai, budžetiem, izmaksu kontrolei un tā tālāk.</span><span class="sxs-lookup"><span data-stu-id="19c50-110">You can then analyze, summarize, and evaluate cost data, so that management can make the best possible decisions for price updates, budgets, cost control, and so on.</span></span> <span data-ttu-id="19c50-111">Datu avots, kas tiek izmantots izmaksu analīzei, tiek izmantots neatkarīgi Izmaksu uzskaitē.</span><span class="sxs-lookup"><span data-stu-id="19c50-111">The source data that is used for cost analysis is treated independently in Cost accounting.</span></span> <span data-ttu-id="19c50-112">Tāpēc atjauninājumus Izmaksu uzskaitē neietekmē datu avots.</span><span class="sxs-lookup"><span data-stu-id="19c50-112">Therefore, updates in Cost accounting don’t affect the source data.</span></span> <span data-ttu-id="19c50-113">Tomēr, apkopojot izmaksu datus no dažādiem avotiem, un jo īpaši, importējot galvenos kontus no virsgrāmatas kā izmaksu elementus, rodas datu <span data-ttu-id="19c50-114">Šāda dublēšana ir nepieciešama, jo jūs izmantojat finanšu pārvaldību ārēju pārskatu veidošanai, un Izmaksu uzskaiti iekšēju pārskatu veidošanai.</span><span class="sxs-lookup"><span data-stu-id="19c50-114">This redundancy is required, because you use financial management for external reporting and Cost accounting for internal reporting.</span></span>

<span data-ttu-id="19c50-115">**Izmaksu uzskaites virsgrāmata**</span><span class="sxs-lookup"><span data-stu-id="19c50-115">**Cost accounting ledger**</span></span>

<span data-ttu-id="19c50-116">To nosaka pēc kalendāra, valūtas un izmaksu elementa dimensijas, un tā kontrolē procesus un politikas izmaksu mērīšanai.</span><span class="sxs-lookup"><span data-stu-id="19c50-116">Defined by calendar, currency, and cost element dimension, it controls processes and policies for measuring costs.</span></span> 

<span data-ttu-id="19c50-117">**Izmaksu ieraksts**</span><span class="sxs-lookup"><span data-stu-id="19c50-117">**Cost entry**</span></span>

<span data-ttu-id="19c50-118">Izmaksu ieraksti ir pārsūtīšanas rezultāts, izmantojot datu savienotājus no virsgrāmatas ierakstiem, izmaksu sadalījumus un grāmatotos izmaksu ierakstus izmaksu žurnālos.</span><span class="sxs-lookup"><span data-stu-id="19c50-118">Cost entries are the result of a transfer via data connectors from general ledger entries, cost allocations, and posted cost entries in cost journals.</span></span>

-  <span data-ttu-id="19c50-124">Projekti</span><span class="sxs-lookup"><span data-stu-id="19c50-124">Projects</span></span>
-  <span data-ttu-id="19c50-125">Nodaļas</span><span class="sxs-lookup"><span data-stu-id="19c50-125">Departments</span></span>
-  <span data-ttu-id="19c50-126">Izmaksu centri</span><span class="sxs-lookup"><span data-stu-id="19c50-126">Cost centers</span></span>

<span data-ttu-id="19c50-127">Vadība izmanto izmaksu objektus, lai aprēķinātu izmaksu daudzumu, kā arī veiktu ienesīguma analīzi.</span><span class="sxs-lookup"><span data-stu-id="19c50-127">Management uses cost objects to quantify costs, but also to drive profitability analysis.</span></span>

<span data-ttu-id="19c50-128">**Izmaksu elements**</span><span class="sxs-lookup"><span data-stu-id="19c50-128">**Cost element**</span></span>

<span data-ttu-id="19c50-129">Lietots kā funkcija izmaksu izsekošanai un kategorizēšanai.</span><span class="sxs-lookup"><span data-stu-id="19c50-129">Used as a function to track and categorize costs.</span></span> <span data-ttu-id="19c50-130">Ir divi izmaksu elementu tipi: primārais un sekundārais.</span><span class="sxs-lookup"><span data-stu-id="19c50-130">There are two types of cost elements: primary and secondary.</span></span>

<span data-ttu-id="19c50-134">Sekundārie izmaksu elementi pārstāv iekšējo izmaksu plūsmu, jo šīs izmaksas tiek izmantotas tikai izmaksu uzskaitē.</span><span class="sxs-lookup"><span data-stu-id="19c50-134">Secondary cost elements represent the internal cost flow because these costs are only used in Cost accounting.</span></span> <span data-ttu-id="19c50-135">Tie tiek izmantoti izmaksu apkopojumu kārtulās, lai izmaksas apkopotu saprotamos intervālos, kurus izmanto pieskaitāmo izmaksu aprēķināšanai.</span><span class="sxs-lookup"><span data-stu-id="19c50-135">They are used in cost roll-up rules to aggregate costs into meaningful buckets used by overhead calculation.</span></span> 

<span data-ttu-id="19c50-136">**Izmaksu klasifikācija**</span><span class="sxs-lookup"><span data-stu-id="19c50-136">**Cost classification**</span></span>

<span data-ttu-id="19c50-137">Izmaksu klasifikācijas grupu izmaksas atbilstoši to koplietojamajām iezīmēm.</span><span class="sxs-lookup"><span data-stu-id="19c50-137">Cost classification groups costs according to their shared characteristics.</span></span> <span data-ttu-id="19c50-138">Piemēram, izmaksas var grupēt pēc elementiem, izsekojamības un uzvedības.</span><span class="sxs-lookup"><span data-stu-id="19c50-138">For example, costs can be grouped by elements, traceability, and behavior.</span></span>

-   <span data-ttu-id="19c50-139">**Pēc elementiem** – materiāli, darbaspēks un izdevumi.</span><span class="sxs-lookup"><span data-stu-id="19c50-139">**By elements** – Materials, labor, and expenses.</span></span>

-   <span data-ttu-id="19c50-144">**Pēc uzvedības** – fiksēts, mainīgs un daļēji mainīgs.</span><span class="sxs-lookup"><span data-stu-id="19c50-144">**By behavior** – Fixed, variable, and semi-variable.</span></span>

<span data-ttu-id="19c50-145">**Izmaksu izturēšanās**</span><span class="sxs-lookup"><span data-stu-id="19c50-145">**Cost behavior**</span></span>

<span data-ttu-id="19c50-146">Izmaksu darbību klasificē izmaksas atbilstoši to uzvedībai saistībā ar galvenajām biznesa aktivitāšu izmaiņām.</span><span class="sxs-lookup"><span data-stu-id="19c50-146">Cost behavior classifies costs according to their behavior in relation to changes in key business activities.</span></span> <span data-ttu-id="19c50-147">Lai efektīvi kontrolētu izmaksas, vadībai jāsaprot izmaksu uzvedību.</span><span class="sxs-lookup"><span data-stu-id="19c50-147">To control costs effectively, management must understand the cost behavior.</span></span> <span data-ttu-id="19c50-148">Pastāv trīs veidu izmaksu uzvedības modeļi: fiksēts, mainīgs un daļēji mainīgs.</span><span class="sxs-lookup"><span data-stu-id="19c50-148">There are three types of cost behavior pattern: fixed, variable, and semi-variable.</span></span>

- <span data-ttu-id="19c50-149">**Fiksētās izmaksas** — fiksētās izmaksas ir izmaksas, kas neatšķiras īstermiņā neatkarīgi no izmaiņām aktivitātes līmenī.</span><span class="sxs-lookup"><span data-stu-id="19c50-149">**Fixed cost** - A fixed cost is a cost that doesn't vary in the short term, regardless of changes in activity level.</span></span> <span data-ttu-id="19c50-150">Fiksētas izmaksas var būt, piemēram, biznesa pamata saimnieciskās darbības izdevumi, piemēram, noma, ka netiks ietekmēta pat tad, ja aktivitātes līmenis palielinās vai samazinās.</span><span class="sxs-lookup"><span data-stu-id="19c50-150">For example, a fixed cost can be a basic operating expense of a business, such as rent, that won't be affected even if the activity level increases or decreases.</span></span>

- <span data-ttu-id="19c50-154">**Daļēji mainīgās izmaksas** — daļēji mainīgās izmaksas ir daļēji fiksētas un daļēji mainīgas izmaksas.</span><span class="sxs-lookup"><span data-stu-id="19c50-154">**Semi-variable cost** - Semi-variable costs are partly fixed and partly variable costs.</span></span> <span data-ttu-id="19c50-155">Piemēram, interneta piekļuves maksa ietver standarta mēneša piekļuves maksu un platjoslas lietošanas maksu.</span><span class="sxs-lookup"><span data-stu-id="19c50-155">For example, an Internet access fee includes a standard monthly access fee and a broadband usage fee.</span></span> <span data-ttu-id="19c50-156">Standarta mēneša piekļuves maksa ir fiksēta izmaksa, bet platjoslas lietošanas maksa ir mainīga izmaksa.</span><span class="sxs-lookup"><span data-stu-id="19c50-156">The standard monthly access fee is a fixed cost, whereas the broadband usage fee is a variable cost.</span></span>

<span data-ttu-id="19c50-157">**Izmaksu vadības ierīce**</span><span class="sxs-lookup"><span data-stu-id="19c50-157">**Cost control unit**</span></span>

<span data-ttu-id="19c50-158">Izmaksu vadības ierīce atspoguļo izmaksu struktūru.</span><span class="sxs-lookup"><span data-stu-id="19c50-158">The cost control unit represents the cost structure.</span></span> <span data-ttu-id="19c50-159">Šī struktūra nosaka, kā izmaksas hierarhiskā secībā plūst starp izmaksu objektu dimensijām un to attiecīgajiem izmaksu objektiem.</span><span class="sxs-lookup"><span data-stu-id="19c50-159">The structure determines how cost flows in a hierarchical order between cost object dimensions and their respective cost objects.</span></span>

<span data-ttu-id="19c50-163">**Izmaksu sadalījums**</span><span class="sxs-lookup"><span data-stu-id="19c50-163">**Cost allocation**</span></span>

<span data-ttu-id="19c50-164">Tiek izmantots, lai izmaksu objekta bilanci piešķirtu citiem izmaksu objektiem, lietojot sadalījuma pamatu.</span><span class="sxs-lookup"><span data-stu-id="19c50-164">Is used to allocate the balance of a cost object to other cost objects by applying an allocation base.</span></span> <span data-ttu-id="19c50-165">Finance atbalsta savstarpējā sadalījuma metodi.</span><span class="sxs-lookup"><span data-stu-id="19c50-165">Finance supports the reciprocal allocation method.</span></span> <span data-ttu-id="19c50-166">Savstarpējā sadalījuma metodē tiek pilnīgi atpazīti savstarpējie pakalpojumi, ar kuriem izmaksu objekti apmainās.</span><span class="sxs-lookup"><span data-stu-id="19c50-166">In the reciprocal allocation method, the mutual services that auxiliary cost objects exchange are fully recognized.</span></span> <span data-ttu-id="19c50-167">Sistēma automātiski nosaka secību, kādā veikt sadalījumus un veikt tās iterāciju.</span><span class="sxs-lookup"><span data-stu-id="19c50-167">The system automatically determines the order to perform the allocations in and iterate over it.</span></span> <span data-ttu-id="19c50-168">Izmaksu objekta bilance tiek sadalīta pēc viena sadalījuma pamata.</span><span class="sxs-lookup"><span data-stu-id="19c50-168">The balance of a cost object is allocated by a single allocation base.</span></span> <span data-ttu-id="19c50-169">Tiek atbalstīti sadalījumi dažādās izmaksu objektu dimensijās un to attiecīgajos elementos.</span><span class="sxs-lookup"><span data-stu-id="19c50-169">Allocations across cost object dimensions and their respective members are supported.</span></span> <span data-ttu-id="19c50-170">Sadalījuma secību kontrolē izmaksu kontroles vienība.</span><span class="sxs-lookup"><span data-stu-id="19c50-170">The allocation order is controlled by the cost control unit.</span></span>

<span data-ttu-id="19c50-171">**Izmaksu sadalījuma politika**</span><span class="sxs-lookup"><span data-stu-id="19c50-171">**Cost allocation policy**</span></span>
