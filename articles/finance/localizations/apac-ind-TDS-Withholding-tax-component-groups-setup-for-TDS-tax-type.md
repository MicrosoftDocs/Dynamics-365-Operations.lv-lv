# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="93ee4-103">ieturētā nodokļa komponentu grupu iestatīšana TDS nodokļu tipam</span><span class="sxs-lookup"><span data-stu-id="93ee4-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="93ee4-104">Šajā tēmā skaidrots, kā iestatīt ieturētā nodokļa komponentu grupas, piemēram, **Noma** un **Līgumdarbinieks** No kopējās ienākumu summas atskaitītā nodokļa (TDS) veidam.</span><span class="sxs-lookup"><span data-stu-id="93ee4-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="93ee4-105">Atveriet **Nodoklis \> Iestatīšana \> Ieturētais nodoklis \> ieturētā nodokļa komponentu grupas**.</span><span class="sxs-lookup"><span data-stu-id="93ee4-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="93ee4-106">[![ieturētā nodokļa komponentu grupu lapa](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="93ee4-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="93ee4-107">Laukā **Nodokļu tips** atlasiet **TDS**, lai iestatītu ieturētā nodokļa komponentu grupas TDS nodokļa tipam.</span><span class="sxs-lookup"><span data-stu-id="93ee4-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="93ee4-108">Lai izveidotu rindu, darbību rūtī atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="93ee4-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="93ee4-109">Laukā **ieturētā nodokļa komponentu grupa** ievadiet TDS komponenta grupas nosaukumu.</span><span class="sxs-lookup"><span data-stu-id="93ee4-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="93ee4-110">Ievadiet aprakstu laukā **Apraksts**.</span><span class="sxs-lookup"><span data-stu-id="93ee4-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="93ee4-111">Kopsavilkuma cilnes **Vispārīgi** laukā **Statuss** atlasiet TDS komponentu grupas dzīvesvietas statusu.</span><span class="sxs-lookup"><span data-stu-id="93ee4-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="93ee4-112">Iespējas ir **Rezidents** un **Nerezidents**.</span><span class="sxs-lookup"><span data-stu-id="93ee4-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="93ee4-113">Laukā **Sadaļas kods** ievadiet sadaļas kodu, kas tiek pielietots TDS komponentu grupai.</span><span class="sxs-lookup"><span data-stu-id="93ee4-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="93ee4-114">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="93ee4-114">Close the page.</span></span>
