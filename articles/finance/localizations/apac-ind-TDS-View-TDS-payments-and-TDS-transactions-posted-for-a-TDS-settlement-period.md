# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a><span data-ttu-id="a2e07-103">Skatīt grāmatotos TDS maksājumus un darījumus TDS norēķinu periodam</span><span class="sxs-lookup"><span data-stu-id="a2e07-103">View posted TDS payments and transactions for a TDS settlement period</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a2e07-104">Šajā tēmā skaidrots, kā skatīt No kopējo ienākumu summas atskaitītā nodokļa (TDS) maksājumus un darījumus, kas tika grāmatoti norēķinu periodam.</span><span class="sxs-lookup"><span data-stu-id="a2e07-104">This topic explains how to view the Tax Deducted at Source (TDS) payments and transactions that were posted for a settlement period.</span></span>

1. <span data-ttu-id="a2e07-105">Pārejiet uz sadaļu **Nodokļi \> Netiešie nodokļi \> Ieturētais nodoklis \> ieturētā nodokļa norēķinu periodi**.</span><span class="sxs-lookup"><span data-stu-id="a2e07-105">Go to **Tax \> Indirect taxes \> Withholding tax \> Withholding tax settlement periods**.</span></span>

    <span data-ttu-id="a2e07-106">[![ieturētā nodokļa norēķinu periodu lapa](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span><span class="sxs-lookup"><span data-stu-id="a2e07-106">[![Withholding tax settlement periods page](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span></span>

2. <span data-ttu-id="a2e07-107">Lapā **ieturētā nodokļa norēķinu periodi** atlasiet **ieturētā nodokļa maksājumi**, lai atvērtu lapu **ieturētā nodokļa maksājumi**, kur varat skatīt TDS segšanu, kas tika veiktas noteiktam TDS apmaksas periodam.</span><span class="sxs-lookup"><span data-stu-id="a2e07-107">On the **Withholding tax settlement periods** page, select **Withholding tax payments** to open the **Withholding tax payments** page, where you can view the TDS settlements that were made for a specific TDS settlement period.</span></span>

    <span data-ttu-id="a2e07-108">Cilne **Pārskats** parāda tālāk norādīto informāciju:</span><span class="sxs-lookup"><span data-stu-id="a2e07-108">The **Overview** tab shows the following information:</span></span>

    - <span data-ttu-id="a2e07-109">**Datums** - TDS norēķinu datums.</span><span class="sxs-lookup"><span data-stu-id="a2e07-109">**Date** – The date of TDS settlement.</span></span>
    - <span data-ttu-id="a2e07-110">**Dokuments** – TDS norēķinu darījuma dokumenta numurs.</span><span class="sxs-lookup"><span data-stu-id="a2e07-110">**Voucher** – The voucher number of the TDS settlement transaction.</span></span>
    - <span data-ttu-id="a2e07-111">**Nodokļu tips** – nodokļu tips, kam tiek izpildīts norēķinu process.</span><span class="sxs-lookup"><span data-stu-id="a2e07-111">**Tax type** – The tax type that the settlement process is run for.</span></span>
    - <span data-ttu-id="a2e07-112">**Nodokļu konta numurs (TAN)** – TDS norēķinu darījuma Nodokļa konta numurs (TAN).</span><span class="sxs-lookup"><span data-stu-id="a2e07-112">**Tax Account Number (TAN)** – The Tax Account Number (TAN) of the settled TDS transaction.</span></span>
    - <span data-ttu-id="a2e07-113">Laukā **Norēķinu periods** - norēķinu periods, kuram tiek palaists TDS norēķinu process.</span><span class="sxs-lookup"><span data-stu-id="a2e07-113">**Settlement period** – The settlement period that the TDS settlement process is run for.</span></span>
    - <span data-ttu-id="a2e07-114">**Sākuma datums** - norēķinu perioda sākuma datums.</span><span class="sxs-lookup"><span data-stu-id="a2e07-114">**From date** – The start date of the settlement period.</span></span>
    - <span data-ttu-id="a2e07-115">**Beigu datums** - norēķinu perioda beigu datums.</span><span class="sxs-lookup"><span data-stu-id="a2e07-115">**To date** – The end date of the settlement period.</span></span>
    - <span data-ttu-id="a2e07-116">**ieturētā nodokļa maksājuma versija** – ieturētā nodokļa maksājuma versija: **Sākotnējais**, **Pēdējie labojumi** vai **Kopsummas saraksts**.</span><span class="sxs-lookup"><span data-stu-id="a2e07-116">**Withholding tax payment version** – The version of the withholding tax payment: **Original**, **Latest corrections**, or **Total list**.</span></span>

5. <span data-ttu-id="a2e07-117">Atlasiet **Dokuments**, lai apskatītu dokumentu ierakstus TDS darījumam.</span><span class="sxs-lookup"><span data-stu-id="a2e07-117">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span>
6. <span data-ttu-id="a2e07-118">Atlasiet **Ieturētā nodokļa darījumi**, lai skatītu detalizētu informāciju par TDS darījumiem, kas nosegti noteiktam periodam norēķinu periodā.</span><span class="sxs-lookup"><span data-stu-id="a2e07-118">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for a specific period during a settlement period.</span></span> <span data-ttu-id="a2e07-119">Atlasiet **Dokuments**, lai apskatītu dokumentu ierakstus TDS darījumam.</span><span class="sxs-lookup"><span data-stu-id="a2e07-119">Select **Voucher** to view the voucher entries for the TDS transaction.</span></span> <span data-ttu-id="a2e07-120">Atlasiet **Ieturētā nodokļa komponentus**, lai skatītu TDS, kas tika aprēķināti uz TDS nodokļa komponentu noteiktam TDS nodokļa kodam.</span><span class="sxs-lookup"><span data-stu-id="a2e07-120">Select **Withholding tax components** to view the TDS that was calculated per TDS tax component for a specific TDS tax code.</span></span>
7. <span data-ttu-id="a2e07-121">Aizveriet lapu **ieturētā nodokļa maksājumi**, lai atgrieztos **ieturētā nodokļa norēķinu periodu** lapā.</span><span class="sxs-lookup"><span data-stu-id="a2e07-121">Close the **Withholding tax payments** page to return to the **Withholding tax settlement periods** page.</span></span>
8. <span data-ttu-id="a2e07-122">Atlasiet **ieturētā nodokļa darījumi**, lai skatītu detalizētu informāciju par TDS darījumiem, kas nosegti norēķinu periodā.</span><span class="sxs-lookup"><span data-stu-id="a2e07-122">Select **Withholding tax transactions** to view the details of the TDS transactions that were settled for the settlement period.</span></span>
