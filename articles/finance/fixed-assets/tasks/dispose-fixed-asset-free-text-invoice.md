# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="f6f96-103">Pamatlīdzekļa izslēgšana, izmantojot brīvā teksta rēķinu</span><span class="sxs-lookup"><span data-stu-id="f6f96-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f6f96-104">Šajā tēmā ir paskaidrots, kā izslēgt pamatlīdzekli, izmantojot brīva teksta rēķinu.</span><span class="sxs-lookup"><span data-stu-id="f6f96-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="f6f96-105">Navigācijas rūtī pārejiet uz sadaļu **Moduļi > Debitori > Rēķini > Visi brīva teksta rēķini**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="f6f96-106">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-106">Select **New**.</span></span>
3. <span data-ttu-id="f6f96-107">Ievadiet vai atlasiet vērtību laukā **Klienta konts**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="f6f96-108">Validējiet noklusējuma **Rēķina** datumu un rediģējiet, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="f6f96-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="f6f96-109">Validējiet atlikušos noklusējuma virsraksta laukus, tādus kā **Valūta**, un rediģējiet, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="f6f96-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="f6f96-110">Sadaļā **Rēķina rindas** validējiet laukus **Apraksts** un **Galvenais konts**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="f6f96-111">Validējiet noklusējuma laukus **PVN grupa** un **Vienuma PVN grupa**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="f6f96-112">Ievadiet **Vienības cenu** vai pamatlīdzekļa pārdošanas **Daudzumu**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="f6f96-113">Atlasiet sadaļu **Rindas informācija**:</span><span class="sxs-lookup"><span data-stu-id="f6f96-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="f6f96-114">Ievadiet vai atlasiet vērtību laukā **Pamatlīdzeklis**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="f6f96-115">Atlasiet **Grāmatot**.</span><span class="sxs-lookup"><span data-stu-id="f6f96-115">Select **Post**.</span></span>
