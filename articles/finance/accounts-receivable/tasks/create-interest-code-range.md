# <a name="create-an-interest-code-with-a-range"></a>Procentu koda ar diapazonu izveide

[!include [banner](../../includes/banner.md)]
Procentu kodus var iestatīt, lai tiktu aprēķinātas dažādas procentu summas, balstoties uz vērtību diapazonu. Šajā procedūrā ir parādīts, kā pievienot procentu kodu un tam pievienot diapazonu.

1. Pārejiet uz **Sadaļu Kredīts un > soda > Iestatiet soda naudas kodus**.
2. Klikšķiniet **Jauns**.
3. Laukā **Procentu** kods ievadiet soda naudas koda nosaukumu.
4. Laukā **Apraksts** ievadiet soda naudas koda aprakstu.
5. Atlasiet **mēnesi**.
6. Izvērsiet sadaļu **Ienākumi**.
7. Izvērsiet sadaļu **Peļņa pēc** valūtas.
8. Laukā **Virsgrāmatas grāmatošanas** konts norādiet vēlamās vērtības.
9. Laukā **Procenti pēc diapazona** atlasiet 'Mēneši'.
10. Noklikšķiniet uz **Pievienot**.
11. Laukā **Apraksts** ievadiet šīs valūtas un diapazona aprakstu.
12. Noklikšķiniet uz **Saglabāt**.
13. Noklikšķiniet **uz Diapazoni**.
14. Klikšķiniet **Jauns**.
15. Ievadiet **vērtību No** kā 0 un pēc tam ievadiet soda procentus mēnesī, kas tiks izmantoti soda naudas aprēķinam. Mūsu piemērā tā ir 1,5.
16. Klikšķiniet **Jauns**.
17. Kā nākamo vērtību No ievadiet 4, kas ir pirmais mēnesis, kurā tiks aprēķināta jauna procentu summa.
18. Ievadiet procentu likmi mēnesī, kas tiks izmantota, lai aprēķinātu procentus, sākot ar 4. mēnesi. Šajā piemērā atlasiet vērtību 2.0.
19. Klikšķiniet **Jauns**.
20. Kā nākamo vērtību No ievadiet 7, kas ir nākamais mēnesis, kurā tiks aprēķināta jauna procentu summa.
21. Ievadiet procentu likmi mēnesī, kas tiks izmantota, lai aprēķinātu procentus, sākot ar 7. mēnesi. Šajā piemērā atlasiet vērtību 2.5.
22. Noklikšķiniet **uz Aizvērt,** lai pabeigtu iestatīšanu.
