[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="7a2c8-106">Navigācijas rūtī dodieties uz **Moduļi > Kredīts un iekasēšana > Iestatīšana > Iestatīt atgādinājuma vēstuļu sēriju**.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-106">In the Navigation pane, go to **Modules > Credit and collections > Setup > Set up collection letter sequence**.</span></span>

3. <span data-ttu-id="7a2c8-108">Laukā **Atgādinājuma vēstuļu sērija** ievadiet sērijas ID, kas pārstāvēs sēriju.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-108">In the **Collection letter sequence** field, enter a sequence ID that will represent the sequence.</span></span> <span data-ttu-id="7a2c8-109">Tas tiks izmantots, kad iestatīsit grāmatošanas metodi.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-109">It will be used when you set up a posting profile.</span></span>
4. <span data-ttu-id="7a2c8-110">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-110">In the **Description** field, type a value.</span></span>  <span data-ttu-id="7a2c8-111">Maksāšanas nosacījumi nav obligāti.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-111">The terms of payment is optional.</span></span> <span data-ttu-id="7a2c8-112">Ja ievadāt vērtību šeit, atgādinājuma vēstules papildmaksas rēķinā tiks izmantoti šie maksāšanas nosacījumi, nevis debitoram saglabātie maksāšanas nosacījumi.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-112">If you enter a value here, the collection letter fee invoice will use these terms of payment instead of the terms of payment stored with the customer.</span></span>  
5. <span data-ttu-id="7a2c8-113">Laukā **Atgādinājuma vēstules kods** atlasiet kodu pirmajai atgādinājuma vēstulei, ko vēlaties nosūtīt.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-113">In the **Collection letter code** field, select the code for the first collection letter that you want to send.</span></span> <span data-ttu-id="7a2c8-114">Pirmā atgādinājuma vēstule tiek izveidota atbilstoši rēķina apmaksas termiņam, vērtībai, ko šai rindai ievadāt pagarinājuma periodam laukā Dienas, un atbilstoši citai informācijai, ko ievadāt šai rindai.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-114">The first collection letter is created according to the due date on the invoice, the value that you enter for the grace period in the Days field on this line, and other information that you enter on this line.</span></span>  
6. <span data-ttu-id="7a2c8-115">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="7a2c8-116">Papildmaksas valūta pēc noklusējuma tiek ņemta no debitora valūtas norādes.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="7a2c8-117">Šis valūtas kods var atšķirties no rēķina valūtas.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-117">This currency code can be different than the invoice currency.</span></span>  
7. <span data-ttu-id="7a2c8-118">Noklikšķiniet uz **Pievienot**, lai pievienotu nākamo atgādinājuma vēstuli, kas tiks nosūtīta šajā sērijā.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-118">Click **Add** to add the next collection letter that will be sent in the sequence.</span></span> <span data-ttu-id="7a2c8-119">Daudzos gadījumos pirmā atgādinājuma vēstule ir tikai brīdinājums.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-119">In many cases, the first collection letter is just a warning.</span></span> <span data-ttu-id="7a2c8-120">Varat pievienot papildmaksas, ja tādas nepieciešamas.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-120">You can add fees if needed.</span></span>  
8. <span data-ttu-id="7a2c8-121">Laukā Atgādinājuma vēstules kods atlasiet nākamo atgādinājuma vēstuli, kura tiks nosūtīta šajā sērijā.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-121">In the collection letter code field, select the next collection letter that will be sent in the sequence.</span></span>
9. <span data-ttu-id="7a2c8-122">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-122">In the **Description** field, type a value.</span></span>
10. <span data-ttu-id="7a2c8-123">Laukā **Galvenais konts** atlasiet ieņēmumu kontu, kas tiks izmantots papildmaksām.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-123">In the **Main account** field, select the revenue account that will be used for fees.</span></span>
11. <span data-ttu-id="7a2c8-124">Ievadiet papildmaksu, kas jāattiecina, grāmatojot šo atgādinājuma vēstuli.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-124">Enter the fee that will be charged when this collection letter is posted.</span></span>
12. <span data-ttu-id="7a2c8-125">Laukā **Krājumu PVN grupa** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-125">In the **Item sales tax group** field, click the drop down button to open the lookup.</span></span> <span data-ttu-id="7a2c8-126">Atlasiet krājuma PVN grupu, ja papildmaksai jāaprēķina PVN.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-126">Select an item sales tax group if sales taxes must be calculated on the fee.</span></span>  
13. <span data-ttu-id="7a2c8-127">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-127">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="7a2c8-128">Laukā **Minimālais nokavētais atlikums** ievadiet minimālo nokavējuma bilanci, kas nepieciešama pirms inkasācijas vēstules nosūtīšanas.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>

16. <span data-ttu-id="7a2c8-134">Noklikšķiniet uz **Pievienot**, lai pievienotu pēdējo atgādinājuma vēstuli, kas tiks nosūtīta šajā sērijā.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-134">Click **Add** to add the last collection letter in the sequence.</span></span> <span data-ttu-id="7a2c8-135">Atgādinājuma vēstuļu sērijai varat pievienot līdz pieciem atgādinājuma vēstuļu kodiem.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-135">You can add up to five collection letter codes for a collection letter sequence.</span></span>

18. <span data-ttu-id="7a2c8-137">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-137">In the **Description** field, type a value.</span></span>
19. <span data-ttu-id="7a2c8-138">Laukā **Galvenais konts** norādiet vēlamās vērtības.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-138">In the **Main account** field, specify the desired values.</span></span>

21. <span data-ttu-id="7a2c8-140">Laukā **Krājumu PVN grupa** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-140">In the **Item sales tax group** field, click the drop-down button to open the lookup.</span></span>
22. <span data-ttu-id="7a2c8-141">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="7a2c8-142">Ievadiet skaitli laukā **Minimālā nokavētā bilance**.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="7a2c8-143">Ievadiet skaitli laukā **Dienas**.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-143">In the **Days** field, enter a number.</span></span>
25. <span data-ttu-id="7a2c8-144">Atlasiet izvēles rūtiņu **Bloķēt**, lai apturētu papildu piegādes un rēķinu piesūtīšanu klientam.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-144">Select the **Block** check box to stop the customer from additional deliveries and invoicing.</span></span> <span data-ttu-id="7a2c8-145">Lai kontu atbloķētu, rēķinu izrakstīšanā atlasiet **Nē** un debitoru lapā atlasiet piegādes aizturēšanas lauku.</span><span class="sxs-lookup"><span data-stu-id="7a2c8-145">To unblock the account, select **No** in the Invoicing and delivery on hold field in the Customers page.</span></span>
