# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="32477-103">Kredīta notas izveide projektu rēķiniem</span><span class="sxs-lookup"><span data-stu-id="32477-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="32477-104">Dodieties uz Projektu vadība un uzskaite > Projekti > Visi projekti.</span><span class="sxs-lookup"><span data-stu-id="32477-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="32477-105">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="32477-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="32477-106">Atlasiet kredīta notai izmantoto projekta ID.</span><span class="sxs-lookup"><span data-stu-id="32477-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="32477-107">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="32477-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="32477-108">Darbību rūtī noklikšķiniet uz Pārvaldīt.</span><span class="sxs-lookup"><span data-stu-id="32477-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="32477-109">Noklikšķiniet uz Rēķinu žurnāli.</span><span class="sxs-lookup"><span data-stu-id="32477-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="32477-110">Atlasiet kredīta notas transakciju.</span><span class="sxs-lookup"><span data-stu-id="32477-110">Select the transaction for credit note.</span></span> <span data-ttu-id="32477-111">Sarakstā atzīmējiet atlasīto rindu.</span><span class="sxs-lookup"><span data-stu-id="32477-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="32477-112">Noklikšķiniet uz Funkcijas.</span><span class="sxs-lookup"><span data-stu-id="32477-112">Click Functions.</span></span> 
8. <span data-ttu-id="32477-113">Noklikšķiniet uz Atlasīt kredīta notai</span><span class="sxs-lookup"><span data-stu-id="32477-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="32477-114">Atlasiet kredīta notas transakciju, atzīmējot izvēles rūtiņu.</span><span class="sxs-lookup"><span data-stu-id="32477-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="32477-115">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="32477-115">Click OK.</span></span> 

<span data-ttu-id="32477-116">Pēc šīs darbības veikšanas kredīta nota ir pieejama rēķina priekšlikumam.</span><span class="sxs-lookup"><span data-stu-id="32477-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
