# <a name="confirm-update-and-send-a-project-quotation"></a><span data-ttu-id="6746d-103">Projekta piedāvājuma apstiprināšana, atjaunināšana un sūtīšana</span><span class="sxs-lookup"><span data-stu-id="6746d-103">Confirm, update, and send a project quotation</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="6746d-104">Pēc tam, kad esat izveidojis un nosūtījis projekta piedāvājumu debitoram, jums jāsaņem debitora apstiprinājums, pirms atjaunināt piedāvājuma statusu uz **Nosūtīts**.</span><span class="sxs-lookup"><span data-stu-id="6746d-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="6746d-105">Visas debitora pieprasītās modifikācijas var tikt atjauninātas piedāvājumā.</span><span class="sxs-lookup"><span data-stu-id="6746d-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="6746d-106">Kad piedāvājuma statuss ir atjaunināts uz **Nosūtīts**, to nevar modificēt.</span><span class="sxs-lookup"><span data-stu-id="6746d-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="6746d-107">Tālāk minētajā procedūrā ir aprakstītas opcijas, lai nosūtītu piedāvājumu apstiprināšanai, veiktu atjaunināšanu, pamatojoties uz atsauksmi, un pēc tam nosūtītu piedāvājumu.</span><span class="sxs-lookup"><span data-stu-id="6746d-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="6746d-108">Projekta piedāvājuma apstiprinājuma nosūtīšana</span><span class="sxs-lookup"><span data-stu-id="6746d-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="6746d-109">Varat nosūtīt esošu projekta piedāvājumu apstiprināšanai pa e-pastu vai kā izdrukātu kopiju.</span><span class="sxs-lookup"><span data-stu-id="6746d-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span> 

1. <span data-ttu-id="6746d-110">Dodieties uz **Projektu vadība un uzskaite** > **Piedāvājumi** > **Projekta piedāvājums**.</span><span class="sxs-lookup"><span data-stu-id="6746d-110">Go to **Project management and accounting** > **Quotations** > **Project quotation.**</span></span> 
2. <span data-ttu-id="6746d-111">Lapā **Projekta piedāvājums** atlasiet piedāvājumu, kuru vēlaties nosūtīt apstiprināšanai.</span><span class="sxs-lookup"><span data-stu-id="6746d-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="6746d-112">Cilnes **Sekošana** grupā **Ģenerēt** atlasiet **Apstiprināt**.</span><span class="sxs-lookup"><span data-stu-id="6746d-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span> 
4. <span data-ttu-id="6746d-113">Lapā **Apstiprināt piedāvājumu** iestatiet pieprasītos parametrus.</span><span class="sxs-lookup"><span data-stu-id="6746d-113">On the **Confirm quotation** page, set the required parameters.</span></span> <span data-ttu-id="6746d-114">Piemēram, lai izdrukātu apstiprinājumu, sadaļas **Drukāt** opcijas ieslēdziet opciju **Drukāt apstiprinājumu** un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="6746d-114">For example, to print the confirmation, under **Print** options, turn on **Print confirmation**, and then select **OK**.</span></span>
5. <span data-ttu-id="6746d-115">Lapā **Sūtīt piedāvājumu** atlasiet **Opcijas** un lapā **Drukāt mērķa iestatījumus** atlasiet, vai vēlaties, lai piedāvājums tiktu nosūtīts uz **Ekrānu**, **E-pastu**, **Failu**, **Drukāšanas arhīvu** vai **Printeri**, un pēc tam veiciet visus pielāgojumus, ko vēlaties izmantot apgabalā **Specifikācijas**, lai lai maršrutētu piedāvājumu.</span><span class="sxs-lookup"><span data-stu-id="6746d-115">On the **Send quotation** page, select **Options**, and on the **Print destination settings** page, select whether you want the quotation sent to **Screen**, **E-mail**, **File**, **Print archive**, or **Printer**, and then make any adjustments that you want in the **Specification** area to route the quotation.</span></span>
6. <span data-ttu-id="6746d-116">Lapā **Drukāt mēŗķa iestatījumus** atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="6746d-116">On the **Print destination settings** page, select **OK**.</span></span>  

## <a name="update-a-project-quotation"></a><span data-ttu-id="6746d-117">Projekta piedāvājuma atjaunināšana</span><span class="sxs-lookup"><span data-stu-id="6746d-117">Update a project quotation</span></span>

1. <span data-ttu-id="6746d-120">Dodieties uz **Projektu vadība un uzskaite** > **Piedāvājumi** > **Projekta piedāvājumi**.</span><span class="sxs-lookup"><span data-stu-id="6746d-120">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="6746d-121">Lapā **Projekta piedāvājumi** atlasiet piedāvājumu, kuru vēlaties atjaunināt, un Darbību rūtī atlasiet **Rediģēt**.</span><span class="sxs-lookup"><span data-stu-id="6746d-121">On the **Project quotations** page, select the quotation you want to update and on the Action Pane, select **Edit**.</span></span>
3. <span data-ttu-id="6746d-122">Atjauniniet nepieciešamo lauka informāciju un Darbību rūtī atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="6746d-122">Update the necessary field information, and on the Action Pane, select **Save**.</span></span>  

## <a name="send-a-project-quotation-to-a-customer"></a><span data-ttu-id="6746d-123">Projekta piedāvājuma sūtīšana debitoram</span><span class="sxs-lookup"><span data-stu-id="6746d-123">Send a project quotation to a customer</span></span> 

1. <span data-ttu-id="6746d-124">Dodieties uz **Projekta vadība un uzskaite** > **Piedāvājumi** > **Projekta piedāvājumi** un atlasiet projekta piedāvājumu, kuru vēlaties nosūtīt.</span><span class="sxs-lookup"><span data-stu-id="6746d-124">Go to **Project management and accounting** > **Quotations** > **Project quotations**, and select the project quotation that you want to send.</span></span>

> [!NOTE]
> <span data-ttu-id="6746d-127">Projekta piedāvājumu nevar modificēt pēc tam, kad statuss ir nomainīts uz **Nosūtīts**.</span><span class="sxs-lookup"><span data-stu-id="6746d-127">You can’t modify a project quotation after the status is changed to **Sent**.</span></span>
