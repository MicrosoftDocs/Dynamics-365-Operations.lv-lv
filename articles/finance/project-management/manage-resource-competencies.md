## <a name="set-up-competencies-certificates"></a><span data-ttu-id="9df07-112">Kompetenču iestatīšana: sertifikāti</span><span class="sxs-lookup"><span data-stu-id="9df07-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="9df07-118">Saraksta lapā **Nodarbinātie** pārbaudiet, vai iepriekšējā procedūrā izmantotais nodarbinātais joprojām ir atlasīts.</span><span class="sxs-lookup"><span data-stu-id="9df07-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="9df07-119">Pēc tam darbību rūts cilnē **Nodarbinātais**, grupā **Kompetences** atlasiet **Prasmes**.</span><span class="sxs-lookup"><span data-stu-id="9df07-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="9df07-120">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="9df07-120">Select **New**.</span></span>
3. <span data-ttu-id="9df07-121">Laukā **Prasme** atlasiet **Projektu vadība**.</span><span class="sxs-lookup"><span data-stu-id="9df07-121">In the **Skill** field, select **Project management**.</span></span>

7. <span data-ttu-id="9df07-125">Atlasiet **Saglabāt** un pēc tam aizveriet lapu.</span><span class="sxs-lookup"><span data-stu-id="9df07-125">Select **Save**, and then close the page.</span></span>
