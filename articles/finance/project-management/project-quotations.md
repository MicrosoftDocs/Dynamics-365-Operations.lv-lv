# <a name="project-quotations"></a><span data-ttu-id="ddea1-104">Projekta piedāvājumi</span><span class="sxs-lookup"><span data-stu-id="ddea1-104">Project quotations</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ddea1-105">Šajā rakstā tiek iepazīstināts ar projektu piedāvājumu jēdzienu; projektu piedāvājumus varat izmantot, lai kā pirmo projekta fāzes darbību sniegtu debitoram pievilcīgu piedāvājumu.</span><span class="sxs-lookup"><span data-stu-id="ddea1-105">This article introduces the concept of project quotations, which you can use to make an attractive offer to a customer as the first step of the project phase.</span></span> <span data-ttu-id="ddea1-106">Projekta piedāvājumā var būt ietverti piedāvātie krājumi un pakalpojumi, vispārēja kontaktinformācija, īpaši tirdzniecības līgumi un atlaides, kā arī iespējamie nodokļi un piemaksas.</span><span class="sxs-lookup"><span data-stu-id="ddea1-106">A project quotation might include the items and services that are quoted, basic contact information, special trade agreements and discounts, and possible taxes and surcharges.</span></span> 

<span data-ttu-id="ddea1-107">Spēja pārraudzīt, pārskatīt un kontrolēt projektu piedāvājumu un pasūtījumu konveijeru ir nozīmīga daļa no projektu pārvaldības.</span><span class="sxs-lookup"><span data-stu-id="ddea1-107">The ability to monitor, review, and control the pipeline of project quotations and orders is an important part of project management.</span></span> <span data-ttu-id="ddea1-108">Šo uzdevumu veikšanai var būt noderīgi dažādi rīki.</span><span class="sxs-lookup"><span data-stu-id="ddea1-108">Various tools can help with these tasks.</span></span> <span data-ttu-id="ddea1-109">Piemēram, pareizas atsauces datu definīcijas (piedāvājuma tipi, piedāvājuma izcelsme, prognoze un iespējamība) jums palīdz analizēt konveijeru.</span><span class="sxs-lookup"><span data-stu-id="ddea1-109">For example, correct reference data definitions (quotation types, quotation origin, and prognosis and probability) help you analyze the pipeline.</span></span> <span data-ttu-id="ddea1-110">Šos rīkus varat izmantot, lai iedalītu kategorijās iemeslus, kuru dēļ projekta piedāvājums tika uzvarēts vai zaudēts, kā arī lai noteiktu piedāvājuma iespējamo vērtību.</span><span class="sxs-lookup"><span data-stu-id="ddea1-110">You can use these tools to categorize the reasons why a project quotation was won or lost, and to determine the potential value of a quotation.</span></span> 

<span data-ttu-id="ddea1-111">Projekta piedāvājumā varat ietvert pakalpojumus, vispārīgu kontaktinformāciju, īpašus tirdzniecības līgumus un atlaides, kā arī projektam prognozētos nodokļus un piemaksas.</span><span class="sxs-lookup"><span data-stu-id="ddea1-111">In a project quotation, you enter the services, basic contact information, special trade agreements and discounts, and estimated taxes and surcharges for a project.</span></span> <span data-ttu-id="ddea1-112">Varat arī projektam atlasīt aktivitātes vai uzdevumus, un izveidot uzdevumu un apakšuzdevumu hierarhiju.</span><span class="sxs-lookup"><span data-stu-id="ddea1-112">You can also select the activities or tasks for a project, and create a hierarchy of tasks and subtasks.</span></span> <span data-ttu-id="ddea1-113">Katrai aktivitātei varat ievadīt detalizētu informāciju par šīs aktivitātes hronometrāžu un ilgumu, un par prasmēm un pieredzi, kas darbiniekiem ir nepieciešamas, lai šo aktivitāti varētu veikt.</span><span class="sxs-lookup"><span data-stu-id="ddea1-113">For each activity, you can enter details about the timing and duration of the activity, and about the skills and experience that are required for workers who perform the activity.</span></span> 

<span data-ttu-id="ddea1-114">Projekta piedāvājums ir nesaistošs novērtējums par izpildāmo darbu.</span><span class="sxs-lookup"><span data-stu-id="ddea1-114">The project quotation is a non-binding estimate of the work that must be performed.</span></span> <span data-ttu-id="ddea1-115">Taču, kad piedāvājumā iekļautā informācija tiek kopēta uz projektu, kurš tiek saistīts ar projekta līgumu, šī informācija kļūst par daļu no abu pušu noslēgtā saistošā līguma.</span><span class="sxs-lookup"><span data-stu-id="ddea1-115">However, when the information in the quotation is copied to a project that is associated with a project contract, that information becomes part of a binding agreement between two parties.</span></span> 

<span data-ttu-id="ddea1-116">Ja debitors apstiprina projekta piedāvājumu, šo projekta piedāvājuma informāciju varat kopēt uz projektu.</span><span class="sxs-lookup"><span data-stu-id="ddea1-116">If the customer approves the project quotation, you can copy the information in the project quotation to a project.</span></span> <span data-ttu-id="ddea1-117">Projekta piedāvājuma informāciju varat arī vienlaicīgi kopēt uz projekta prognozi.</span><span class="sxs-lookup"><span data-stu-id="ddea1-117">You can also copy the project quotation information to a project forecast at the same time.</span></span>
