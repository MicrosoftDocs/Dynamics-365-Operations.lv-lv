[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="e86a3-105">Šajā tēmā skaidrots, kā izveidot numuru sērijas nomas ID.</span><span class="sxs-lookup"><span data-stu-id="e86a3-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="e86a3-106">Tajā ir arī paskaidrots, kā izveidot unikālus ID, kas tiek izmantoti indeksa pārvērtēšanas procesā.</span><span class="sxs-lookup"><span data-stu-id="e86a3-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>
