# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="22361-103">Izdevumu pārskata skatīšana no Virsgrāmatas vai darbplūsmas Sagāde un avoti</span><span class="sxs-lookup"><span data-stu-id="22361-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="22361-104">Izpildot vaicājumu, lai skatītu transakcijas Virsgrāmatā vai darbplūsmā Sagāde un avoti, vaicājums var atgriezt pārāk daudz informācijas.</span><span class="sxs-lookup"><span data-stu-id="22361-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="22361-105">Tāpēc ieteicams sašaurināt meklēšanu.</span><span class="sxs-lookup"><span data-stu-id="22361-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="22361-106">Lai iegūtu informāciju, kas jums ir nepieciešama, varat rādīt transakciju detalizēti, lai skatītu izdevumu pārskatu, kurā tā ir iekļauta.</span><span class="sxs-lookup"><span data-stu-id="22361-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="22361-107">Izdevumu pārskatiem varat rādīt transakciju detalizēti no oriģinālā izdevumu pārskata, kas ir saglabāts darbplūsmā Izdevumu pārvaldība.</span><span class="sxs-lookup"><span data-stu-id="22361-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="22361-108">Papildus izdevumu kopsummai, var skatīt detalizētas transakcijas, transakcijas, kas ir lietojamas pievienotās vērtības nodokļa (PVN) atgūšanai un kvītis, kas ir pievienotas izdevumiem.</span><span class="sxs-lookup"><span data-stu-id="22361-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="22361-109">Lai detalizētu skatītu oriģinālo izdevumu pārskatu, kur ir iekļauta transakcija, atlasiet transakciju un pēc tam atlasiet **Oriģinālais dokuments**.</span><span class="sxs-lookup"><span data-stu-id="22361-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="22361-110">Skatot atsevišķas transakcijas, varat skatīt arī visus papildu datus, kas piesaistīti transakcijai.</span><span class="sxs-lookup"><span data-stu-id="22361-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="22361-111">Piemēram, uzskaitītos viesnīcas izdevumus var skatīt detalizētā sarakstā ar izdevumiem, piemēram, interneta piekļuvi, tālruņa zvanus un apkalpošanu viesnīcas numuriņā.</span><span class="sxs-lookup"><span data-stu-id="22361-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
