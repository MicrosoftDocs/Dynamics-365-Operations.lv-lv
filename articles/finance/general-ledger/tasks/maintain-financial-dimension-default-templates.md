# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="f4817-103">Finanšu dimensijas noklusējuma veidņu uzturēšana</span><span class="sxs-lookup"><span data-stu-id="f4817-103">Maintain financial dimension default templates</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f4817-104">Šajā tēmā parādīts, kā uzstādīt finanšu dimensijas noklusējuma veidnes.</span><span class="sxs-lookup"><span data-stu-id="f4817-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="f4817-105">Veidnes var izmantot, lai ātri sadalītu summas no pirmdokumentiem.</span><span class="sxs-lookup"><span data-stu-id="f4817-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="f4817-106">Navigācijas rūtī dodieties uz **Moduļi > Virsgrāmata > Kontu plāns > Dimensijas >Finanšu dimensiju noklusējuma veidnes**.</span><span class="sxs-lookup"><span data-stu-id="f4817-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>
2. <span data-ttu-id="f4817-107">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="f4817-107">Select **New**.</span></span>
3. <span data-ttu-id="f4817-108">Laukā **Veidnes ID** ievadiet veidnes ID.</span><span class="sxs-lookup"><span data-stu-id="f4817-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="f4817-109">Ievadiet procentuālo vērtību pirmās dimensijas vērtībām.</span><span class="sxs-lookup"><span data-stu-id="f4817-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="f4817-110">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="f4817-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="f4817-111">Atlasiet **Pievienot**.</span><span class="sxs-lookup"><span data-stu-id="f4817-111">Select **Add**.</span></span>
7. <span data-ttu-id="f4817-112">Ievadiet citus procentus nākamajām dimensiju vērtībām.</span><span class="sxs-lookup"><span data-stu-id="f4817-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="f4817-113">Šajā piemērā ievadiet procentuālo vērtību, tāpēc jums vēl joprojām ir piešķirti mazāk nekā 100 %.</span><span class="sxs-lookup"><span data-stu-id="f4817-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="f4817-114">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="f4817-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="f4817-115">Atlasiet **Pievienot**.</span><span class="sxs-lookup"><span data-stu-id="f4817-115">Select **Add**.</span></span>
10. <span data-ttu-id="f4817-116">Ievadiet citus procentus, kas padarīs kopējo procentuālo vērtību vienādu ar 100%.</span><span class="sxs-lookup"><span data-stu-id="f4817-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="f4817-117">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="f4817-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="f4817-118">Atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="f4817-118">Select **Save**.</span></span>
