<span data-ttu-id="953a1-104">Šajā procedūrā parādīts, kā izveidot konta aizstājvārdu, kas nodrošina saīsni konta numura ievadīšanai.</span><span class="sxs-lookup"><span data-stu-id="953a1-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="953a1-105">Šajā procedūrā tiek izmantoti demonstrācijas uzņēmuma USMF dati.</span><span class="sxs-lookup"><span data-stu-id="953a1-105">This procedure users demo data company USMF.</span></span>

3. <span data-ttu-id="953a1-108">Laukā Virsgrāmatas konta aizstājvārds ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="953a1-108">In the Ledger account alias field, type a value.</span></span>

9. <span data-ttu-id="953a1-115">Laukā Sākotnējais fokuss atlasiet dimensiju, uz kuru būs vērsts fokuss, kad tiek izmantots šis aizstājvārds.</span><span class="sxs-lookup"><span data-stu-id="953a1-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
