# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="65d2e-103">Virsgrāmatas konta aizstājvārda iestatīšana</span><span class="sxs-lookup"><span data-stu-id="65d2e-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="65d2e-104">Šajā procedūrā parādīts, kā izveidot konta aizstājvārdu, kas nodrošina saīsni konta numura ievadīšanai.</span><span class="sxs-lookup"><span data-stu-id="65d2e-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="65d2e-105">Šajā procedūrā tiek izmantoti demonstrācijas uzņēmuma USMF dati.</span><span class="sxs-lookup"><span data-stu-id="65d2e-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="65d2e-106">Pārejiet uz sadaļu Virsgrāmata > Kontu plāns > Konti > Virsgrāmatas konta aizstājvārds.</span><span class="sxs-lookup"><span data-stu-id="65d2e-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="65d2e-107">Noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="65d2e-107">Click New.</span></span>
3. <span data-ttu-id="65d2e-108">Laukā Virsgrāmatas konta aizstājvārds ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="65d2e-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="65d2e-109">Laukā Konta struktūra atlasiet struktūru, kurai pieder konts un dimensijas.</span><span class="sxs-lookup"><span data-stu-id="65d2e-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="65d2e-110">Laukā Uzņēmums noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.</span><span class="sxs-lookup"><span data-stu-id="65d2e-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="65d2e-111">Sarakstā atrodiet un atlasiet uzņēmumu, uz kuru attiecas aizstājvārds.</span><span class="sxs-lookup"><span data-stu-id="65d2e-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="65d2e-112">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="65d2e-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="65d2e-113">Laukā Virsgrāmatas konta aizstājvārds norādiet kontu un dimensijas.</span><span class="sxs-lookup"><span data-stu-id="65d2e-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="65d2e-114">Konts un dimensijas tiks aizpildītas, izmantojot šo saīsni.</span><span class="sxs-lookup"><span data-stu-id="65d2e-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="65d2e-115">Laukā Sākotnējais fokuss atlasiet dimensiju, uz kuru būs vērsts fokuss, kad tiek izmantots šis aizstājvārds.</span><span class="sxs-lookup"><span data-stu-id="65d2e-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="65d2e-116">Kad ievadīsiet saīsni un konts un dimensijas tiks aizpildītas, lauks Sākotnējais fokuss ir lauks, kur tiks pārvietots kursors vai fokuss.</span><span class="sxs-lookup"><span data-stu-id="65d2e-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
