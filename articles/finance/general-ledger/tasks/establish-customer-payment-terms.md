# <a name="establish-customer-payment-terms"></a><span data-ttu-id="369ce-103">Debitoru maksājumu nosacījumu izveide</span><span class="sxs-lookup"><span data-stu-id="369ce-103">Establish customer payment terms</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="369ce-104">Šī procedūra definē termiņatlaides un apmaksas datuma iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="369ce-104">This procedure defines a cash discount and due date setup.</span></span> <span data-ttu-id="369ce-105">Šis uzdevuma ceļvedis izmanto USMF demonstrācijas uzņēmumu.</span><span class="sxs-lookup"><span data-stu-id="369ce-105">This task guide uses the USMF demo company.</span></span>

1. <span data-ttu-id="369ce-106">Pārejiet uz sadaļu **Navigācijas rūts > Moduļi > Debitori > Maksājumu iestatīšana > Maksājumu dienas**.</span><span class="sxs-lookup"><span data-stu-id="369ce-106">Go to **Navigation pane > Modules > Accounts receivable > Payments setup > Payment days**.</span></span> <span data-ttu-id="369ce-107">**Apmaksas nosacījumu** ir kopīgi **Debitoriem** un **Kreditoriem**.</span><span class="sxs-lookup"><span data-stu-id="369ce-107">The setup for the **Terms of payment** is shared for **Accounts receivable** and **Accounts payable**.</span></span> <span data-ttu-id="369ce-108">Ja tos definējat modulī, tie būs pieejami arī citā modulī.</span><span class="sxs-lookup"><span data-stu-id="369ce-108">If you define it in module, it will be available in the other module also.</span></span> <span data-ttu-id="369ce-109">Šajā uzdevumu ceļvedī visi apmaksas nosacījumi izveidoti sadaļā **Debitori**.</span><span class="sxs-lookup"><span data-stu-id="369ce-109">For this task guide, I set up all the terms of payment under **Accounts receivable**.</span></span>
2. <span data-ttu-id="369ce-110">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="369ce-110">Click **New**.</span></span> <span data-ttu-id="369ce-111">Izveidojiet maksāšanas dienu, ja apmaksas nosacījumos ir prasīta konkrēta nedēļas diena (pirmdiena, otrdiena u.c.) vai konkrēts mēneša datums (5., 10. u.c.).</span><span class="sxs-lookup"><span data-stu-id="369ce-111">Create a payment day if your terms of payment require a specific day of the week (Monday, Tuesday, etc) or a specific date of the month (5th, 10th, etc).</span></span> 
3. <span data-ttu-id="369ce-112">Ievadiet ID laukā **Maksājuma diena**.</span><span class="sxs-lookup"><span data-stu-id="369ce-112">In the **Payment day** field, enter an ID.</span></span>
4. <span data-ttu-id="369ce-113">Laukā **Apraksts** ievadiet maksājuma dienas īsu aprakstu.</span><span class="sxs-lookup"><span data-stu-id="369ce-113">In the **Description** field, enter a description of the payment day.</span></span>
5. <span data-ttu-id="369ce-114">Laukā **Nedēļa/Mēnesis** atlasiet 'Nedēļa' vai 'Mēnesis'.</span><span class="sxs-lookup"><span data-stu-id="369ce-114">In the **Week/Month** field, select either 'Week' or 'Month'.</span></span> <span data-ttu-id="369ce-115">Ja vēlaties ievadīt nedēļas dienu, piemēram, pirmdiena, atlasiet Nedēļa.</span><span class="sxs-lookup"><span data-stu-id="369ce-115">If you want to enter a day of the week, such as Monday, select Week.</span></span> <span data-ttu-id="369ce-116">Ja vēlaties ievadīt mēneša datumu, piemēram, 10., atlasiet Mēnesis.</span><span class="sxs-lookup"><span data-stu-id="369ce-116">If you want to enter a date in the month, such as the 10th, select Month.</span></span> <span data-ttu-id="369ce-117">Šim piemēram atlasiet Mēnesis.</span><span class="sxs-lookup"><span data-stu-id="369ce-117">Select Month for this example.</span></span> 
6. <span data-ttu-id="369ce-118">Ievadiet datumu laukā **Mēneša diena**.</span><span class="sxs-lookup"><span data-stu-id="369ce-118">In the **Day of month** field, enter a date.</span></span> <span data-ttu-id="369ce-119">Datums ir jāievada kā skaitlis, piemēram, "10", nevis "10‑ais".</span><span class="sxs-lookup"><span data-stu-id="369ce-119">The date should be entered as a number, such as '10', and not as '10th'.</span></span> 
7. <span data-ttu-id="369ce-120">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="369ce-120">Click **Save**.</span></span>
8. <span data-ttu-id="369ce-121">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="369ce-121">Close the page.</span></span>
9. <span data-ttu-id="369ce-122">Pārejiet uz sadaļu **Navigācijas rūts > Moduļi > Debitori > Maksājumu iestatīšana > Maksājumu nosacījumi**.</span><span class="sxs-lookup"><span data-stu-id="369ce-122">Go to **Navigation pane > Modules > Accounts receivable > Payments setup > Terms of payment**.</span></span>
10. <span data-ttu-id="369ce-123">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="369ce-123">Click **New**.</span></span> <span data-ttu-id="369ce-124">Apmaksas noteikumi tiek izmantoti, lai definētu, kā tiks aprēķināti apmaksas termiņi.</span><span class="sxs-lookup"><span data-stu-id="369ce-124">Terms of payment is used to define how the due dates will be calculated.</span></span> <span data-ttu-id="369ce-125">Termiņatlaides datuma iestatīšana ir definēta atsevišķā lapā.</span><span class="sxs-lookup"><span data-stu-id="369ce-125">The cash discount date setup is defined in a separate page.</span></span> 
11. <span data-ttu-id="369ce-126">Ievadiet ID laukā **Maksājuma nosacījumi**.</span><span class="sxs-lookup"><span data-stu-id="369ce-126">In the **Terms of payment field**, enter an ID.</span></span>
12. <span data-ttu-id="369ce-127">Ievadiet aprakstu laukā **Apraksts**.</span><span class="sxs-lookup"><span data-stu-id="369ce-127">In the **Description** field, enter a description.</span></span>
13. <span data-ttu-id="369ce-128">Atlasiet **Maksāšanas metodi**, piemēram, SPB, neto, pašreizējais mēnesis utt. Maksāšanas metode tiek izmantota, lai definētu sākuma veidu, kā tiks aprēķināta apmaksājamā summa.</span><span class="sxs-lookup"><span data-stu-id="369ce-128">Select a **Payment method** such as COD, Net, Current month, etc. The Payment method is used to define the start of how the due will be calculated.</span></span> <span data-ttu-id="369ce-129">Piemēram, neto tiek izmantots, ja apmaksas datums vienmēr ir noteikts mēnešu vai dienu skaits pēc rēķina datuma.</span><span class="sxs-lookup"><span data-stu-id="369ce-129">For example, Net is used if the due date is always a set number of months or days after the invoice date.</span></span> <span data-ttu-id="369ce-130">SPB var izmantot, ja maksājums ir nepieciešams, saņemot rēķinu, tāpēc apmaksas datums netiek rēķināts.</span><span class="sxs-lookup"><span data-stu-id="369ce-130">COD can be used to when payment is required upon invoice, so a due date wouldn't be calculated.</span></span> <span data-ttu-id="369ce-131">Šim piemēram atlasiet 'Pašreizējais mēnesis'.</span><span class="sxs-lookup"><span data-stu-id="369ce-131">Select 'Current month' for this task guide.</span></span>  
14. <span data-ttu-id="369ce-132">Ja aprēķinā iekļauta noteikta nedēļas diena vai datums, atlasiet **Maksājuma diena**.</span><span class="sxs-lookup"><span data-stu-id="369ce-132">Select a **Payment day** if a specific day of the  week or date are included in the calculation.</span></span> <span data-ttu-id="369ce-133">Atkarībā no maksāšanas termiņa var ievadīt daudzumu, norādot mēnešu vai dienu skaitu.</span><span class="sxs-lookup"><span data-stu-id="369ce-133">Depending on your terms of payment, you may enter a quantity in Months or Days.</span></span> <span data-ttu-id="369ce-134">Vai var izmantot **Maksājumu grafiku** vai **Maksājumu dienu**, kas jāpievieno maksāšanas metodes beigās.</span><span class="sxs-lookup"><span data-stu-id="369ce-134">Or you can use the **Payment schedule** or **Payment day** to 'add' to the end of the Payment method.</span></span> <span data-ttu-id="369ce-135">Ja apmaksas datums vienmēr ir nākamā mēneša 10. datums, atlasiet **Maksājumu dienu** 10. datumu.</span><span class="sxs-lookup"><span data-stu-id="369ce-135">If the due date will always be the 10th of the next month, select a **Payment day** of the 10th.</span></span> 
15. <span data-ttu-id="369ce-136">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="369ce-136">Click **Save**.</span></span>
16. <span data-ttu-id="369ce-137">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="369ce-137">Close the page.</span></span>
17. <span data-ttu-id="369ce-138">Pārejiet uz sadaļu **Debitori > Maksājumu iestatījumi > Termiņatlaides**.</span><span class="sxs-lookup"><span data-stu-id="369ce-138">Go to **Accounts receivable > Payments setup > Cash discounts**.</span></span>
18. <span data-ttu-id="369ce-139">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="369ce-139">Click **New**.</span></span> <span data-ttu-id="369ce-140">Šajā lapā tiek definēts, kā tiks aprēķināts termiņatlaides datums.</span><span class="sxs-lookup"><span data-stu-id="369ce-140">This page is used to define how the cash discount date will be calculated.</span></span> 
19. <span data-ttu-id="369ce-141">Ievadiet ID laukā **Termiņatlaide**.</span><span class="sxs-lookup"><span data-stu-id="369ce-141">In the **Cash discount** field, enter an ID.</span></span>
20. <span data-ttu-id="369ce-142">Ievadiet aprakstu laukā **Apraksts**.</span><span class="sxs-lookup"><span data-stu-id="369ce-142">In the **Description** field, enter a description.</span></span>
21. <span data-ttu-id="369ce-143">Ja ir pieejama pa pakāpēm sadalīta termiņatlaide, atlasiet **Nākamais atlaižu kods**, kas ir attiecināms pēc šīs jaunās termiņatlaides.</span><span class="sxs-lookup"><span data-stu-id="369ce-143">If a tiered cash discount is available, select the **Next discount code** that is relevant after this new cash discount.</span></span>
22. <span data-ttu-id="369ce-144">Laukā **Dienas** ievadiet dienu skaitu, kas tiek aprēķināts termiņatlaides datumu.</span><span class="sxs-lookup"><span data-stu-id="369ce-144">In the **Days** field, enter the number of days used to calculate the cash dicount date.</span></span> <span data-ttu-id="369ce-145">Ja ir atlasīts princips **Neto**, termiņatlaides datuma aprēķināšanai rēķina datumam tiks pieskaitīts noteikts dienu skaits.</span><span class="sxs-lookup"><span data-stu-id="369ce-145">If **Net** principle is selected, the number of days will be added to the invoice date to calculate the cash discount date.</span></span>  
23. <span data-ttu-id="369ce-146">Laukā **Atlaides procenti** ievadiet termiņatlaidi procentos.</span><span class="sxs-lookup"><span data-stu-id="369ce-146">In the **Discount percentage** field, enter the percentage of the cash discount.</span></span>
24. <span data-ttu-id="369ce-147">**Debitora atlaižu galvenajā kontā** ievadiet galveno kontu, kurā tiks grāmatota termiņatlaide debitora rēķiniem.</span><span class="sxs-lookup"><span data-stu-id="369ce-147">In the **Main account for customer discounts**, enter the main account to which the cash discount will post for customer invoices.</span></span>
25. <span data-ttu-id="369ce-148">Atlasiet opciju laukā **Atlaides korespondējošie konti**.</span><span class="sxs-lookup"><span data-stu-id="369ce-148">In the **Discount offset accounts** field, select an option.</span></span> <span data-ttu-id="369ce-149">Atlasot Konti rēķina rindās, termiņatlaide tiks iegrāmatota tajā pašā līdzekļu/izdevumu galvenajā kontā, kurā kreditora rēķina rindas.</span><span class="sxs-lookup"><span data-stu-id="369ce-149">If you select 'Accounts on the invoice lines', the cash discount will post to the same asset/expense main account on the lines of the vendor invoice.</span></span> <span data-ttu-id="369ce-150">Atlasot Kreditoru rēķiniem izmantot galveno kontu, termiņatlaide tiks iegrāmatota galvenajā kontā, kurš definēts laukā Kreditoru rēķinu galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="369ce-150">If you select 'Use main account for vendor invoices', the cash discount will post to the main account you define in the 'Main account for vendor invoices'.</span></span> <span data-ttu-id="369ce-151">Šim piemēram izvēlieties 'Kreditoru rēķiniem izmantot galveno kontu'.</span><span class="sxs-lookup"><span data-stu-id="369ce-151">For this example, select 'Use main account for vendor invoices'.</span></span> 
26. <span data-ttu-id="369ce-152">Laukā **Kreditora atlaižu galvenais konts** ievadiet galveno kontu, kurā tiks grāmatota termiņatlaide kreditora rēķiniem.</span><span class="sxs-lookup"><span data-stu-id="369ce-152">In the **Main account for vendor discounts** field, enter the main account to which the cash discount will post for vendor invoices.</span></span>
27. <span data-ttu-id="369ce-153">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="369ce-153">Click **Save**.</span></span>
