[!include [banner](../../includes/banner.md)]

11. <span data-ttu-id="d5486-117">Laukā **Konta veids** ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="d5486-117">In the **Account type** field, type a value.</span></span>
12. <span data-ttu-id="d5486-118">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="d5486-118">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="d5486-119">Laukā **Debets** ievadiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="d5486-119">In the **Debit** field, type a value.</span></span>

14. <span data-ttu-id="d5486-121">Laukā **Konts** norādiet vēlamās vērtības.</span><span class="sxs-lookup"><span data-stu-id="d5486-121">In the **Account** field, specify the desired values.</span></span>

    - <span data-ttu-id="d5486-129">Procenti: summas vaučerī tiek pārveidotas par procentu koeficientiem, kas ļauj piemērot jebkuru summu, kad atlasīta vaučera veidne.</span><span class="sxs-lookup"><span data-stu-id="d5486-129">Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>
    - <span data-ttu-id="d5486-130">Summa: tiks saglabātas un piemērotas faktiskās summas.</span><span class="sxs-lookup"><span data-stu-id="d5486-130">Amount: The actual amounts will be stored and applied.</span></span>

22. <span data-ttu-id="d5486-132">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="d5486-132">Click **New**.</span></span>
23. <span data-ttu-id="d5486-133">Laukā **Nosaukums** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu pārlūku.</span><span class="sxs-lookup"><span data-stu-id="d5486-133">In the **Name** field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="d5486-134">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="d5486-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="d5486-135">Noklikšķiniet uz **Rindas**.</span><span class="sxs-lookup"><span data-stu-id="d5486-135">Click **Lines**.</span></span>
26. <span data-ttu-id="d5486-136">Noklikšķiniet uz **Funkcijas**.</span><span class="sxs-lookup"><span data-stu-id="d5486-136">Click **Functions**.</span></span>
