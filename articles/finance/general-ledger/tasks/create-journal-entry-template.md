[!include [task guide banner](../../includes/task-guide-banner.md)]

11. <span data-ttu-id="dbea0-117">Laukā **Konta veids** ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="dbea0-117">In the **Account type** field, type a value.</span></span>
12. <span data-ttu-id="dbea0-118">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="dbea0-118">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="dbea0-119">Laukā **Debets** ievadiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="dbea0-119">In the **Debit** field, type a value.</span></span>

    - <span data-ttu-id="dbea0-129">Procenti: summas vaučerī tiek pārveidotas par procentu koeficientiem, kas ļauj piemērot jebkuru summu, kad atlasīta vaučera veidne.</span><span class="sxs-lookup"><span data-stu-id="dbea0-129">Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>
