# <a name="create-a-main-account"></a><span data-ttu-id="4e3cb-103">Galvenā konta izveide</span><span class="sxs-lookup"><span data-stu-id="4e3cb-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="4e3cb-104">Šajā uzdevuma ceļvedī aprakstīts, kā pievienot galveno kontu esošajam kontu plānam.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="4e3cb-105">Šajā ierakstā tiek izmantots USMF demonstrācijas uzņēmums.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="4e3cb-106">Dodieties uz **Navigācijas rūts > Moduļi > Virsgrāmata > Kontu plāns > Konti > Galvenie konti**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="4e3cb-107">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-107">Click **New**.</span></span>
3. <span data-ttu-id="4e3cb-108">Laukā **Galvenais konts** ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="4e3cb-109">Laukā **Nosaukums** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="4e3cb-110">Laukā **Galvenā konta veids** atlasiet tipu, kas vislabāk atspoguļo kontu bilanci un atrašanās vietu finanšu pārskatos.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="4e3cb-111">Sarakstā atlasiet konta kategoriju, kurai pieder galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="4e3cb-112">Konta kategorija tiek izmantota noklusējuma finanšu pārskatiem un Power BI paneļa saturam.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="4e3cb-113">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="4e3cb-114">Mainiet noklusējuma debeta vai kredīta bilanci.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="4e3cb-115">Laukā **Noklusējuma valūta** atlasiet vērtību no valūtu saraksta.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="4e3cb-116">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="4e3cb-117">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="4e3cb-118">Pārslēdziet sadaļas **Juridiskas personas prioritātes** paplašinājumu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="4e3cb-119">Noklikšķiniet uz **Pievienot**, lai atlasītu juridisku personu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="4e3cb-120">Sarakstā atlasiet Juridiska persona.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="4e3cb-121">Noklikšķiniet uz **Pievienot**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-121">Click **Add**.</span></span>
15. <span data-ttu-id="4e3cb-122">Sarakstā atzīmējiet atlasīto rindu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="4e3cb-123">Atzīmējiet vai notīriet izvēles rūtiņu **Pārtraukts**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="4e3cb-124">Izvērsiet sadaļu **Finanšu pārskatu veidošana**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="4e3cb-125">Laukā **Maiņas kursa tips** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="4e3cb-126">Sarakstā atlasiet **Maiņas kursa tips kontam**.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="4e3cb-127">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="4e3cb-128">Laukā **Valūtas pārrēķināšanas tips** atlasiet maiņas kursu aprēķināšanas metodi kontam.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="4e3cb-129">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="4e3cb-129">Close the page.</span></span>
