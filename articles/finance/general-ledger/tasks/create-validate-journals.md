# <a name="create-and-validate-journals"></a><span data-ttu-id="7c071-103">Žurnālu izveide un validēšana</span><span class="sxs-lookup"><span data-stu-id="7c071-103">Create and validate journals</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7c071-104">Šī procedūra izveido un apstiprina žurnālus un žurnāla rindas.</span><span class="sxs-lookup"><span data-stu-id="7c071-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="7c071-105">Varat izmēģināt šo procedūru, izmantojot demonstrācijas datu uzņēmumu USMF.</span><span class="sxs-lookup"><span data-stu-id="7c071-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="7c071-106">Ejiet uz **Virsgrāmata > Žurnāla ieraksti > Vispārējie žurnāli**.</span><span class="sxs-lookup"><span data-stu-id="7c071-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="7c071-107">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="7c071-107">Click **New**.</span></span>
3. <span data-ttu-id="7c071-108">Laukā **Nosaukums** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu pārlūku.</span><span class="sxs-lookup"><span data-stu-id="7c071-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="7c071-109">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="7c071-109">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="7c071-110">Noklikšķiniet uz **Rindas**.</span><span class="sxs-lookup"><span data-stu-id="7c071-110">Click **Lines**.</span></span>
6. <span data-ttu-id="7c071-111">Laukā **Konts** ievadiet atbilstošu kontu atbilstoši Konta veidam.</span><span class="sxs-lookup"><span data-stu-id="7c071-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>
7. <span data-ttu-id="7c071-112">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="7c071-112">In the **Description** field, type a value.</span></span>
8. <span data-ttu-id="7c071-113">Ievadiet konta summu vai nu kā **Debets**, vai **Kredīts**.</span><span class="sxs-lookup"><span data-stu-id="7c071-113">Enter an amount for the account as either a **Debit** or **Credit**.</span></span> 
9. <span data-ttu-id="7c071-114">Laukā **Korespondējošais konts** ievadiet atbilstošu kontu atbilstoši Korespondējoša konta veidam.</span><span class="sxs-lookup"><span data-stu-id="7c071-114">In the **Offset account** field, enter an appropriate account based on the Offset account type.</span></span>
10. <span data-ttu-id="7c071-115">Noklikšķiniet uz **Validēt**.</span><span class="sxs-lookup"><span data-stu-id="7c071-115">Click **Validate**.</span></span>
11. <span data-ttu-id="7c071-116">Noklikšķiniet uz **Validēt**.</span><span class="sxs-lookup"><span data-stu-id="7c071-116">Click **Validate**.</span></span>
12. <span data-ttu-id="7c071-117">Noklikšķiniet uz **Grāmatot**.</span><span class="sxs-lookup"><span data-stu-id="7c071-117">Click **Post**.</span></span>
13. <span data-ttu-id="7c071-118">Noklikšķiniet **Dokuments**.</span><span class="sxs-lookup"><span data-stu-id="7c071-118">Click **Voucher**.</span></span>
