# <a name="set-up-sales-tax-reporting-codes"></a><span data-ttu-id="20850-103">Iestatīt PVN pārskatu kodus</span><span class="sxs-lookup"><span data-stu-id="20850-103">Set up sales tax reporting codes</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="20850-104">PVN pārskatu kodi attiecas uz lauka numuru PVN pārskatā.</span><span class="sxs-lookup"><span data-stu-id="20850-104">The Sales tax reporting codes refer to a field number on a sales tax report.</span></span> <span data-ttu-id="20850-105">Tos izmanto valstij atbilstošu pārskatu izkārtojumos un PVN maksājumu pārskatā pēc kodiem, lai drukātu PVN summas apmaksas periodā, summējot pēc pārskata koda.</span><span class="sxs-lookup"><span data-stu-id="20850-105">They are used on country specific report layouts and the Sales tax payment by code report to print sales tax amounts for a settlement period summarized per reporting code.</span></span> <span data-ttu-id="20850-106">Kad esat izveidojis PVN pārskata kodus, varat uz tiem atsaukties Pārskatu iestatīšanas kopsavilkuma cilnē PVN koda lapā.</span><span class="sxs-lookup"><span data-stu-id="20850-106">After you create Sales tax reporting codes, you can refer to them on the Report setup FastTabs in the Sales tax code page.</span></span> 

<span data-ttu-id="20850-107">Šis paraugs izmanto DEMF demonstrācijas uzņēmumu.</span><span class="sxs-lookup"><span data-stu-id="20850-107">This recording uses the DEMF demo company.</span></span>

1. <span data-ttu-id="20850-108">Sadaļā **Navigācijas rūts**, pārejiet uz **Nodoklis > Iestatīšana > PVN > PVN pārskatu kodi**.</span><span class="sxs-lookup"><span data-stu-id="20850-108">In the **Navigation pane**, go to **Tax > Setup > Sales tax > Sales tax reporting codes**.</span></span>
2. <span data-ttu-id="20850-109">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="20850-109">Click **New**.</span></span>
3. <span data-ttu-id="20850-110">Atlasiet pārskata izkārtojumu, kuram atbilst pārskata kods.</span><span class="sxs-lookup"><span data-stu-id="20850-110">Select the report layout that the reporting code belongs to.</span></span> <span data-ttu-id="20850-111">Šis izkārtojums tiek izmantots, lai filtrētu pieejamos pārskata kodus PVN kodam.</span><span class="sxs-lookup"><span data-stu-id="20850-111">This layout is used to filter the available reporting codes for a Sales tax code.</span></span> <span data-ttu-id="20850-112">Katrs PVN kods atbilst maksājumu periodam, kurš saistīts ar PVN iestādi, kas izmanto pārskata izkārtojumu.</span><span class="sxs-lookup"><span data-stu-id="20850-112">Each Sales tax code belongs to a settlement period which belongs to a Sales tax authority which uses a Report layout.</span></span>  
4. <span data-ttu-id="20850-113">Laukā **Pārskata kods** ievadiet skaitli.</span><span class="sxs-lookup"><span data-stu-id="20850-113">In the **Reporting code** field, enter a number.</span></span>
5. <span data-ttu-id="20850-114">Laukā **Pārskata teksts** ievadiet aprakstu, kuru parādīt pārskatos.</span><span class="sxs-lookup"><span data-stu-id="20850-114">In the **Report text** field, enter a description to display on reports.</span></span>
6. <span data-ttu-id="20850-115">Laukā **Īss apraksts** ievadiet aprakstu iekšējai lietošanai.</span><span class="sxs-lookup"><span data-stu-id="20850-115">In the **Brief description** field, enter a description for internal purposes.</span></span>
7. <span data-ttu-id="20850-116">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="20850-116">Click **Save**.</span></span>
