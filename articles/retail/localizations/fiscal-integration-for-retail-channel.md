<span data-ttu-id="cb81f-145">Finanšu integrācijas struktūra nodrošina tālāk norādītās opcijas kļūmju apstrādei finanšu reģistrācijas laikā.</span><span class="sxs-lookup"><span data-stu-id="cb81f-145">The fiscal integration framework provides the following options to handle failures during fiscal registration:</span></span>
