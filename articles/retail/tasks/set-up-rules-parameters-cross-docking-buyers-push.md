# <a name="set-up-rules-and-parameters-for-cross-docking-and-buyers-push"></a> Kārtulu un parametru iestatīšana pārkraušanai sadales centrā un sagādes sadalē

[!include[task guide banner](../includes/task-guide-banner.md)]

Šī procedūra parāda, kā izveidot papildināšanas kārtulas. Papildināšanas kārtulas var lietot, lai kontrolētu to, kā preces tiek sadalītas veikalos, izmantojot pārkraušanu sadales centrā un sagādes sadali. Papildināšanas kārtulas var iestatīt veikaliem vai veikalu grupām. Katrā kārtulas rindā definētā svara vērtība kontrolēs to, kā preču daudzumi tiks sadalīti starp veikaliem, ja pārkraušanai sadales centrā vai sagādes sadalēm kā sadales metode tiks izmantotas papildināšanas kārtulas. Procedūrā tiek izmantoti demonstrācijas uzņēmuma “USRT” dati.

1. Dodieties uz cilni Papildināšanas kārtulas.
2. Noklikšķiniet uz Jauns.
3. Laukā Papildināšanas kārtula ierakstiet vērtību.
4. Apraksta laukā ierakstiet vērtību.
5. Noklikšķiniet uz Saglabāt.
6. Noklikšķiniet uz Pievienot.
7. Sarakstā atzīmējiet atlasīto rindu.
    * Varat izvēlēties veidu Papildināšanas hierarhiju vai Kanāls. Vērtība kontrolē, vai atlase laukā Nosaukums būs kanālu vai noteikta kanāla hierarhija.  Šajā piemērā atstājiet to iestatītu kā Papildināšanas hierarhija.  
8. Laukā Nosaukums atlasiet vērtību.
    * Noklusējuma svara vērtība tiek aizpildīta, izmantojot noliktavā definēto svaru.  Šo svara vērtību var izmantot kārtulai Papildināšana, vai arī laukā Svars var ievadīt jaunu svara vērtību.  
9. Laukā Svars ievadiet skaitli.
10. Noklikšķiniet uz Pievienot.
11. Sarakstā atzīmējiet atlasīto rindu.
12. Laukā Veids atlasiet Kanāls.
13. Laukā Nosaukums ievadiet vai atlasiet kādu vērtību.
14. Laukā Svars ievadiet skaitli.
15. Noklikšķiniet uz Saglabāt.
