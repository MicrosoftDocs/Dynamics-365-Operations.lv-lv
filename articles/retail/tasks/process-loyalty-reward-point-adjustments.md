<span data-ttu-id="742ff-104">Šajā procedūra parādīts, kā atrast informāciju par lojalitātes programmas karti un koriģēt lojalitātes programmas atlīdzības punktus.</span><span class="sxs-lookup"><span data-stu-id="742ff-104">This procedure demonstrates how to look up loyalty card information and adjust loyalty reward points.</span></span> <span data-ttu-id="742ff-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo uzdevumu, ir USRT.</span><span class="sxs-lookup"><span data-stu-id="742ff-105">The demo data company used to create this task is USRT.</span></span> <span data-ttu-id="742ff-106">Šis uzdevums ir paredzēts lomai Mazumtirdzniecības operāciju pārvaldnieks vai Klientu apkalpošanas vadītājs.</span><span class="sxs-lookup"><span data-stu-id="742ff-106">This task is intended for the Retail operations manager role or a Customer service manager role.</span></span>

2. <span data-ttu-id="742ff-108">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="742ff-108">In the list, find and select the desired record.</span></span>

13. <span data-ttu-id="742ff-121">Noklikšķiniet uz Jā.</span><span class="sxs-lookup"><span data-stu-id="742ff-121">Click Yes.</span></span>
14. <span data-ttu-id="742ff-122">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="742ff-122">Close the page.</span></span>
