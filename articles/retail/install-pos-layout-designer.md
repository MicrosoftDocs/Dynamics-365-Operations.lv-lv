2. <span data-ttu-id="b1f1c-111">Atlasiet jebkuru izkārtojumu, kura programmas tips ir **Modern POS operētājsistēmai Windows** vai **Mākoņa POS**, un pēc tam noklikšķiniet uz **Izkārtojuma veidotājs**.</span><span class="sxs-lookup"><span data-stu-id="b1f1c-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
