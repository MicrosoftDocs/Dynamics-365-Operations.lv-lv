# <a name="create-an-onboarding-template"></a>Pievienošanās veidnes izveide

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard nodrošina dažādas veidnes, kas var palīdzēt iespējami ātri izveidot pievienošanas ceļvedi. Varat izmantot vienu vai vairākas no šīm veidnēm, vai varat izveidot savas veidnes. Onboard nodrošina teksta paraugu, ko varat izmantot, veidojot savas veidnes. Tāpēc process ir viegls, pat ja sākat no nulles.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Pievienošanas veidnes izveide no esošas veidnes

1. Kreisās puses izvēlnē atlasiet **Veidnes**.
2. Sadaļā **Populāras veidnes no galerijas** vai **Manas veidnes** atlasiet veidni. Lai skatītu vairāk veidņu atlasiet **Vairāk veidņu galerijā**.
3. Izpildiet kādu no norādītajām transakcijām.

    - Ja veidne ir no galerijas, atlasiet **Saglabāt kā manu veidni**, ievadiet jauno veidnes nosaukumu un atlasiet **Saglabāt**.
    - Ja veidne ir no **Manas veidnes**, atlasiet **Saglabāt kā veidni**, ievadiet jauno veidnes nosaukumu un atlasiet **Saglabāt**.

    [![Veidnes izveide no esošas veidnes](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Jaunas pievienošanas veidnes izveide

1. Kreisās puses izvēlnē atlasiet **Veidnes**.
2. Sadaļā **Manas veidnes** atlasiet elementu **Pievienot** (pluszīme \[**+**\]).

    [![Jaunas veidnes izveide](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. Dialoglodziņā **Izveidot ceļveža veidni** ievadiet nosaukumu šai veidnei un pēc tam atlasiet **Saglabāt**.
