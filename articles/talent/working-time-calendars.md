# <a name="working-time-calendars"></a><span data-ttu-id="feacf-103">Darba laika kalendāri</span><span class="sxs-lookup"><span data-stu-id="feacf-103">Working time calendars</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="feacf-104">Izmantojot darba laika kalendāru, var izveidot kalendāru ar darba stundām un dienām, kuras darbinieki strādā jūsu organizācijā.</span><span class="sxs-lookup"><span data-stu-id="feacf-104">The working time calendar enables you to create a calendar with the hours and days that employees work in your organization.</span></span> <span data-ttu-id="feacf-105">Kalendāri ļauj racionalizēt brīvā laika pieprasīšanas procesu, pēc noklusējuma izmantojot stundas vai dienas.</span><span class="sxs-lookup"><span data-stu-id="feacf-105">Calendars streamline the time off request process by default in the hours or days.</span></span> <span data-ttu-id="feacf-106">Ja darbinieks iesniedz brīvā laika pieprasījumu, viņam nav jāuztraucas par brīvdienām un slēgšanas laiku, kas tiek aprēķināts, izmantojot darba laika kalendāru.</span><span class="sxs-lookup"><span data-stu-id="feacf-106">When an employee submits a time off request, they don't have to worry about holidays and closures, which is handled for them through the working time calendar.</span></span>

## <a name="setting-up-a-working-time-calendar"></a><span data-ttu-id="feacf-107">Darba laika kalendāra iestatīšana</span><span class="sxs-lookup"><span data-stu-id="feacf-107">Setting up a working time calendar</span></span>

<span data-ttu-id="feacf-108">Kalendāri ietvert ģenerēšanas datus, iekļaujamās dienas un stundas, kalendāra dienas, darba laiku šajās dienās, kā arī reģistrētos darbiniekus.</span><span class="sxs-lookup"><span data-stu-id="feacf-108">Calendars include generation details, the days and hours you want to included, the days of the calendar, working times for those days as well as enrolled employees.</span></span> 

<span data-ttu-id="feacf-109">Lai iestatītu kalendāru, rīkojieties šādi:</span><span class="sxs-lookup"><span data-stu-id="feacf-109">To set up a calendar, follow these steps:</span></span>

1. <span data-ttu-id="feacf-110">Lapā **Organizācijas administrēšana** noklikšķiniet uz **Kalendāri**.</span><span class="sxs-lookup"><span data-stu-id="feacf-110">On the **Organization Administration** page, click **Calendars**.</span></span>

2. <span data-ttu-id="feacf-111">Darbību rūtī atlasiet **Jauns** un ievadiet nosaukumu un aprakstu.</span><span class="sxs-lookup"><span data-stu-id="feacf-111">On the Action Pane, select **New** and enter a name and description.</span></span>

3. <span data-ttu-id="feacf-112">Izvēlieties savas organizācijas darba dienas un ievadiet darba laiku.</span><span class="sxs-lookup"><span data-stu-id="feacf-112">Choose the work days for your organization and enter work time.</span></span>

4. <span data-ttu-id="feacf-113">Pievienojiet brīvdienas un slēgšanas laikus, izmantojot pogu **Pievienot**.</span><span class="sxs-lookup"><span data-stu-id="feacf-113">Add Holidays and closures by using the **Add** button.</span></span>

5. <span data-ttu-id="feacf-114">Ievadiet brīvdienu un slēgšanas laiku nosaukumu un aprakstu, piemēram, ASV brīvdienas un svētku dienas.</span><span class="sxs-lookup"><span data-stu-id="feacf-114">Enter the name and description for the holidays and closures, such as US holidays or Bank holidays.</span></span> <span data-ttu-id="feacf-115">Ievadiet brīvdienu un slēgšanas laika datumus.</span><span class="sxs-lookup"><span data-stu-id="feacf-115">Enter the dates for the holidays and closures.</span></span> <span data-ttu-id="feacf-116">Saglabājiet svētku dienu un slēgšanas laika sarakstu un aizveriet lapu.</span><span class="sxs-lookup"><span data-stu-id="feacf-116">Save the Holidays and closures list and close the page.</span></span>

6. <span data-ttu-id="feacf-117">Atlasiet nolaižamajā izvēlnē atlasiet brīvdienu un slēgšanas laika sarakstu.</span><span class="sxs-lookup"><span data-stu-id="feacf-117">Select the holidays and closures list from the drop-down menu.</span></span>

7. <span data-ttu-id="feacf-118">Ja uz darbiniekiem attiecas ārpusdarba laiks, piemēram, pusdienu vai citi pārtraukumi, pievienojiet arī to.</span><span class="sxs-lookup"><span data-stu-id="feacf-118">If your employees have non-work time, such as lunches or breaks, add those as well.</span></span> <span data-ttu-id="feacf-119">Atlasiet **Ārpusdarba laiks** un ievadiet nosaukumu un laika diapazonu.</span><span class="sxs-lookup"><span data-stu-id="feacf-119">Select **Non-work time** and enter the name and time range.</span></span> <span data-ttu-id="feacf-120">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="feacf-120">Close the page.</span></span> 

8. <span data-ttu-id="feacf-121">Noklikšķiniet uz **Pievienot**, lai pievienotu kalendārā ārpusdarba laiku.</span><span class="sxs-lookup"><span data-stu-id="feacf-121">Click **Add** to add the non-work time to your calendar.</span></span>

9. <span data-ttu-id="feacf-122">Cilnē **Diena** atlasiet **Ģenerēt**, lai ģenerētu dienas kalendārā.</span><span class="sxs-lookup"><span data-stu-id="feacf-122">On the **Days** tab, select **Generate** to generate the days in your calendar.</span></span> <span data-ttu-id="feacf-123">Ievadiet kalendāra datumu diapazonu.</span><span class="sxs-lookup"><span data-stu-id="feacf-123">Enter the date range for the calendar.</span></span> <span data-ttu-id="feacf-124">Dienu un darba laika vērtības tiek ģenerētas, ņemot vērā darba dienu skaitu un laiku, kas norādīts ģenerēšanas opcijās saistībā ar atlasītajiem datumiem.</span><span class="sxs-lookup"><span data-stu-id="feacf-124">The days and working times are generated based on the work days and times defined under the generation options in conjunction with the dates selected.</span></span>

10. <span data-ttu-id="feacf-125">Lai piešķirtu kalendāru darbiniekam, darbību rūtī atlasiet **Piešķirt darbiniekiem**.</span><span class="sxs-lookup"><span data-stu-id="feacf-125">To assign a calendar to employees, select **Assign to employees** in the Action Pane.</span></span> <span data-ttu-id="feacf-126">Atlasiet darbiniekus, kurus vēlaties piešķirt šim kalendāram, un pēc tam noklikšķiniet uz **Piešķirt**.</span><span class="sxs-lookup"><span data-stu-id="feacf-126">Select the that employees you would like to assign this calendar to, and then click **Assign**.</span></span>

<span data-ttu-id="feacf-127">Darbiniekiem nav obligāti nepieciešama pieeja piešķirtajam kalendāram.</span><span class="sxs-lookup"><span data-stu-id="feacf-127">Employees aren't required to have calendars assigned.</span></span> <span data-ttu-id="feacf-128">Ja ir definēts darba laika kalendārs, brīvās dienas automātiski tiek izslēgtas no pieprasījuma.</span><span class="sxs-lookup"><span data-stu-id="feacf-128">If there is a working time calendar defined, off days are automatically excluded from the request.</span></span> <span data-ttu-id="feacf-129">Pēc noklusējuma darba laiks kalendārā tiek noteikts ar stundu vai dienu apjomu.</span><span class="sxs-lookup"><span data-stu-id="feacf-129">The amount, in hours or days, defaults to the working times defined in the calendar.</span></span> <span data-ttu-id="feacf-130">Ja darbiniekam nav piešķirts kalendārs, visas dienas ir pieejamas brīvā laika pieprasīšanai, bet brīvā laika apjoms pēc noklusējuma neattiecas uz pieprasījumu.</span><span class="sxs-lookup"><span data-stu-id="feacf-130">If an employee doesn't have a calendar assigned, all days are available for time off and the amount of time off is not the default for the request.</span></span>
