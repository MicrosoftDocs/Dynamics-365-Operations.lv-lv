# <a name="whats-new-or-changed-in-dynamics-365-talent---core-hr-august-2018"></a>Jaunumi un izmaiņas programmā Dynamics 365 Talent — Core HR (2018. gada augusts)

**8.1.104 būvējums**

Šajā sadaļā ir aprakstīti līdzekļi, kas ir vai nu jauni, vai kas ir mainīti programmā Dynamics 365 Talent: Core HR.

## <a name="view-expiring-records-in-manager-self-service"></a>Skatiet vadītāju patstāvīgi lietojamā pakalpojuma ierakstus, kam beidzas derīgums

Tagad varat skatīt vadītāju patstāvīgi lietojamā pakalpojuma ierakstus, kam beidzas derīgums Jaunas opcijas ļauj jums konfigurēt, kāda informācija būs pieejami vadītājiem skatīšanai. Šīs opcijas ietver:

-   Sertifikāti

-   Identifikācijas numuri

-   Probācijas periodi

-   Izvērtēšana

-   Testi

Šis līdzeklis arī sniedz iespēju norādīt diapazonu dienās, lai meklētu ierakstus, kam beidzas derīgums.

## <a name="configurable-transfer-actions"></a>Konfigurējamas pārsūtīšanas darbības

Var konfigurēt pēc lomas opcijas, kas būs pieejama pārsūtīšanas pieprasījuma izveides laikā. Šis līdzeklis nodrošina papildu elastību lomām organizācijā.

Piemēram, vadītājiem, kuri pieprasa darbinieku pārsūtīšanu, nedrīkst būt piekļuve atlīdzības summas ieteikšanai vai uzdevumu sarakstu atlasīšanai, kas tiks saistīti ar pārsūtīšanas pieprasījumu. Vadītāji var izveidot un iesniegt pārsūtīšanas pieprasījumus, bet nevar ievadīt atlīdzības vai uzdevumu sarakstu piešķires. Šajā pašā konfigurācijā HR var piešķirt jaunās atlīdzības vērtības un piešķirt jebkurus papildu kontrolsarakstus, kuri jāaizpilda pārsūtīšanas veikšanas rezultātā.

Pēc noklusējuma jaunās konfigurācijas opcijas ir iestatītas, lai iespējas nevarētu mainīt pirms šī atjauninājuma.

## <a name="leave-and-absence"></a>Atvaļinājumi un kavējumi

Tagad ir pieejami papildu datumu lauki atvaļinājumiem un kavējumiem.

Ar šo līdzekli varat iestatīt uzkrājuma perioda principu plāna līmenī, lai lietotu noteiktus darbinieku datumus. Atvaļinājuma uzkrāšanas procesa laikā var lietot datumus, kuri nav plāna sākuma datums. Darbiniekam specifisku datumu opcijas ietver tālāk norādītās vērtības.

-   Pielāgots (pieejama pirms šī atjauninājuma)

-   Gadadienas datums

-   Sākotnējais darbā pieņemšanas datums

-   Darba stāža datums

-   Nodarbinātā koriģētais pieņemšanas datums

-   Nodarbinātā pieņemšanas datums

Ja konfigurēts, lai izmantotu vienu no darbiniekam specifiskiem datumiem, reģistrācijas process izmanto norādīto datumu, lai aprēķinātu uzkrāšanas periodus. Uzkrāšanas perioda princips tiek parādīts arī darbinieka plāna reģistrācijā, lai palīdzētu jums saprast, kas tiek izmantots uzkrāšanas procesa laikā.

## <a name="microsoft-word-integration"></a>Microsoft Word integrēšana

Dokumentu veidnes ir iespējotas visā Core HR programmatūrā. Šis līdzeklis ļauj izveidot vēstules un pārskatus uz jūsu izveidotās Word veidnes bāzes.

Papildinformācija par šo līdzekli ir pieejama vietnē [Office integrāciju apmācība](https://docs.microsoft.com/dynamics365/unified-operations/dev-itpro/office-integration/office-integration-tutorial?toc=/dynamics365/unified-operations/talent/toc.json).
