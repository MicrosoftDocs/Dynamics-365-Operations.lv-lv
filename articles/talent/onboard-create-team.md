# <a name="create-a-hiring-team"></a>Darbā pieņemšanas grupas izveide

[!include [banner](includes/banner.md)]

Programmā Microsoft Dynamics 365 Talent: Onboard varat izveidot darbā pieņemšanas grupas. Pēc tam katrai grupai varat piešķirt pievienošanas ceļvežus un veidnes.

## <a name="create-a-hiring-team"></a>Darbā pieņemšanas grupas izveide

1. Kreisās puses izvēlnē atlasiet vienumu **Grupas**.
2. Sadaļā **Grupas** atlasiet elementu **Pievienot** (pluszīme \[**+**\]).
3. Dialoglodziņa **Izveidot jaunu grupu** sadaļā **Grupas nosaukums** ievadiet darbā pieņemšanas grupas nosaukumu.

    ![[Jaunas komandas izveide pakalpojumā Onboard](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. Sadaļā **Izvēlēties grupas dalībniekus** ievadiet katra grupas dalībnieka vārdu vai e-pasta adresi.

    Lai noņemtu grupas dalībniekus, atlasiet **X**, kas tiek parādīts blakus viņu vārdam zem lodziņa.

5. Lai nosūtītu e-pasta ziņojumu jauniem grupas dalībniekiem, ieslēdziet opciju **Sūtīt e-pasta ziņojumu jaunajiem dalībniekiem**.
6. Atlasiet **Izveidot**.
7. Ja esat norādījis, ka vēlaties nosūtīt e-pasta ziņojumu jaunajiem grupas dalībniekiem, rediģējiet e-pasta ziņojumu nākamajā dialoglodziņā un pēc tam atlasiet **Gatavs**.

## <a name="assign-onboarding-guides-to-a-hiring-team"></a>Pievienošanas ceļvežu piešķiršana darbā pieņemšanas grupai

1. Kreisās puses izvēlnē atlasiet vienumu **Grupas**.
2. Atlasiet grupu.
3. Cilnē **Ceļveži** atlasiet **Pievienot ceļvežus**.

    ![[Pievienošanas ceļvežu pievienošana grupai](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. Atlasiet izvēles rūtiņu katram pievienošanas ceļvedim, kuru vēlaties piešķirt grupai, un pēc tam atlasiet **Pievienot**.

    ![[Pievienošanas ceļvežu atlase pievienošanai grupai](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)

## <a name="assign-onboarding-templates-to-a-hiring-team"></a>Pievienošanas veidņu piešķiršana darbā pieņemšanas grupai

1. Kreisās puses izvēlnē atlasiet vienumu **Grupas**.
2. Atlasiet grupu.
3. Cilnē **Veidnes** atlasiet **Pievienot veidnes**.

    ![[Veidņu pievienošana grupai](./media/onboard-add-templates-to-team.png)](./media/onboard-add-templates-to-team.png)

4. Atlasiet izvēles rūtiņu katrai veidnei, kuru vēlaties piešķirt grupai, un pēc tam atlasiet **Pievienot**.

    ![[Ceļvežu atlase pievienošanai grupai](./media/onboard-select-templates.png)](./media/onboard-select-templates.png)
