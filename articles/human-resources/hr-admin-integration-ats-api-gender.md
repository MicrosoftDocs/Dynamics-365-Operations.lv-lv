# <a name="gender"></a>Dzimums

Šajā tēmā aprakstīta dzimuma opcija, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums: mshr_hcmpersongender

Šis uzskaitījums sniedz vērtību dzimumu kopu izvērtēšanas kandidātam. Tas ir pieejams mshr_hcmpersongender opciju kopā.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | None | Dzimums nav precizēts. |
| 200000001 | Vīrietis | Vīrietis. |
| 200000002 | Sieviete | Sieviete. |
| 200000003 | Nenoteikts | Izvēle nav dzimumam specifiska. |
