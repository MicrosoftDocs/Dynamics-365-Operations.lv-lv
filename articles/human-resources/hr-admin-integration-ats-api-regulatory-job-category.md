# <a name="regulatory-job-category"></a>Normatīvā darba kategorija

Fiziskais nosaukums: mshr_hcmregulatoryjobcategory

Šis uzskaitījums norāda opciju kopu normatīvā darba kategorijas opciju kopas vērtībām. Tas ir nodrošināts esošajā mshr_hcmregulatoryjobcategory opciju kopā.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | None | Nav. |
| 200000001 | Vadošais darbinieks | Vadītāju/vecāko līmeņu amatpersonas un vadītāji. |
| 200000002 | Pārvaldnieks | Pirmā/vidējā līmeņa amatpersonas un vadītāji. |
| 200000003 | Profesionālas | Speciālisti. |
| 200000004 | Tehniķis | Tehniķi. |
| 200000005 | SalesWorker | Pārdošanas darbinieki. |
| 200000006 | Administratīvs | Administratīvā atbalsta darbinieki. |
| 200000007 | CraftWorker | Amatnieki. |
| 200000008 | Operatīvie darbinieki | Operatīvie darbinieki. |
| 200000009 | Strādnieks | Strādnieki/palīgi. |
| 200000010 | ServiceWorker | Apkopes darbinieki. |
