# <a name="skill-level-type"></a>Prasmju līmeņa veids

Šajā tēmā aprakstīta opcija Prasmju līmeņa veids, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums: mshr_hrmskillleveltype

Šī uzskaitījuma nodrošina opciju vērtību kopu, kas kategoriē līmeni, kāds piešķirts kandidāta prasmei.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | Faktiskais | Faktiskais prasmju līmenis. |
| 200000001 | Adresāts | Prasmju līmeņa mērķa vērtība. |
