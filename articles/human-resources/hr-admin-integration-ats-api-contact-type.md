# <a name="contact-type"></a>Kontaktpersonas tips

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā tēmā aprakstīta opcija Kontaktpersonas veids, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums: mshr_logisticselectronicaddressmethodtype

Šis uzskaitījums sniedz statusa kandidātu kontaktpersonas veidu vērtību opciju kopu. 

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | None | Veids nav atlasīts. |
| 200000001 | Tālruņa numurs | Tālruņa numurs. |
| 200000002 | E-pasta adrese | E-pasta adrese. |
| 200000003 | Vietrādis URL | Tīmekļa vietnes URL. |
| 200000004 | Telekss | Teleksa numurs. |
| 200000005 | Fakss | Faksa numurs. |
| 200000006 | Facebook | Facebook konts. Identificēts ar lietotāja ID. |
| 200000007 | Twitter | Twitter konts. Identificēts ar @username. |
| 200000008 | LinkedIn | LinkedIn konts. Identificēts ar lietotājvārdu. |
