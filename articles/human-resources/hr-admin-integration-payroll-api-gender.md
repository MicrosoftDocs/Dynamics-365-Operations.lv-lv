Šajā rakstā ir aprakstītas opciju Dzimums kopas Dynamics 365 Human Resources.

Šis uzskaitījums sniedz vērtību dzimumu kopu izvērtēšanas darbiniekam. Tas ir pieejams mshr_hcmpersongender opciju kopā.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | None | Dzimums nav precizēts. |
| 200000001 | Vīrietis | Vīrietis. |
| 200000002 | Sieviete | Sieviete. |
| 200000003 | Nenoteikts | Izvēle nav dzimumam specifiska. |
