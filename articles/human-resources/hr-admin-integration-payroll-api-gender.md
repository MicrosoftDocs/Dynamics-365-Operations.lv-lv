| Vērtība | Iezīme | Apraksts |

| 200000000 | None | Dzimums nav precizēts. |
| 200000001 | Vīrietis | Vīrietis. |
| 200000002 | Sieviete | Sieviete. |
| 200000003 | Nenoteikts | Izvēle nav dzimumam specifiska. |
