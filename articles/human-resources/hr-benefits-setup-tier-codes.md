# <a name="set-up-tier-codes"></a>Iestatīt pakāpju kodus

Dynamics 365 Human Resources izmanto pakāpju kodus ar atvieglojumu likmēm, ko maksā nodarbinātie un darba devējs. Varat definēt neierobežotu pakāpju kodu skaitu un iestatīt likmes aprēķinus, lai apmierinātu sava uzņēmuma vajadzības.

1. Darbvietā **Atvieglojumu pārvaldība**, sadaļā **Iestatījumi** atlasiet **Pakāpju kodi**.

2. Atlasiet **Jauns**.

3. Norādiet vērtības tālāk minētajos laukos.

   | Lauks | Apraksts |
   | --- | --- |
   | **Pakāpes kods** | Unikāls atvieglojumu pakāpes kods vai numurs, lai identificētu pakāpes kodu. |
   | **Apraksts** | Atvieglojuma pakāpes koda apraksts. |

4. Atlasiet **Saglabāt**.
