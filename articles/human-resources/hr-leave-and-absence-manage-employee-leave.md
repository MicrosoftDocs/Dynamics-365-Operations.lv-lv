# <a name="manage-employee-leave"></a>Darbinieka atvaļinājuma pārvaldība

[!include [banner](includes/preview-feature.md)]

Darbinieka atvaļinājumu var pārvaldīt pēc atvaļinājuma veida. Tas ietver atvaļinājumu reģistrācijas beigu datumu un atvaļinājumu veida bilanču koriģēšanu. 

## <a name="adjust-leave-balances"></a>Koriģēt atvaļinājumu bilances

1. Darbinieka ierakstā atlasiet **Atvaļinājums**.

2. Atlasiet **Atvaļinājumu un kavējumu iestatīšana**.

3. Atlasiet **Koriģēt bilanci**.

4. Atlasiet **Atvaļinājuma veids**.

5. Ievadīt **Korekcijas summa**. 

6. Varat arī atlasīt **Datums**. 

Koriģējot darbinieka atvaļinājuma bilanci, jūs varat iekļaut iemesla kodu un komentāru.
