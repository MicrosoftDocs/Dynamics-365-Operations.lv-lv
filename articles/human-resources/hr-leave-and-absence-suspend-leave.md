# <a name="suspend-leave"></a><span data-ttu-id="f08d1-103">Pārtraukt atvaļinājumu</span><span class="sxs-lookup"><span data-stu-id="f08d1-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="f08d1-104">Jūs varat aizturēt atvaļinājumu darbiniekam, lai pārtrauktu atvaļinājumu uzkrājumu apstrādi atsevišķiem atvaļinājumu veidiem.</span><span class="sxs-lookup"><span data-stu-id="f08d1-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="f08d1-105">Atvaļinājuma un prombūtnes aizturēšana nodarbinātajam</span><span class="sxs-lookup"><span data-stu-id="f08d1-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="f08d1-106">Darbinieka ierakstā atlasiet **Atvaļinājums**.</span><span class="sxs-lookup"><span data-stu-id="f08d1-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="f08d1-107">Atlasiet **Aizturēt atvaļinājumu**.</span><span class="sxs-lookup"><span data-stu-id="f08d1-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="f08d1-108">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="f08d1-108">Select **New**.</span></span>

4. <span data-ttu-id="f08d1-109">Dialoglodziņā **Aizturēo atvaļinājumu uzkrājumu** atlasiet **Atvaļinājuma veidu** kopā ar atlikšanas **Sākuma datumu** un **Beigu datumu**.</span><span class="sxs-lookup"><span data-stu-id="f08d1-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="f08d1-110">Pēc izvēles varat pievienot **Komentāru** atlikšanai.</span><span class="sxs-lookup"><span data-stu-id="f08d1-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="f08d1-111">Ja uzkrājumi tiek apstrādāti, kamēr darbinieka atvaļinājums tiek atlikts, uzkrājums netiks veikts aizturēto atvaļinājumu veidiem.</span><span class="sxs-lookup"><span data-stu-id="f08d1-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
