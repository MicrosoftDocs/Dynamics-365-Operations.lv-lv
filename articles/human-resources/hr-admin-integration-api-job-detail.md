## <a name="properties"></a>Rekvizīti

| Rekvizīts<br>**Fiziskais nosaukums**<br>**_tips_** | Izmantot | Apraksts |
| --- | --- | --- |
| **Darba ID**<br>mshr_jobid<br>*Virkne* | Tikai lasāms<br>Obligāts | Unikāls darba ID. |
| **Tirgus cenas zemais slieksnis**<br>mshr_marketpricelowthreshold<br>*Decimāldaļskaitlis* | | Sistēmas ģenerēta GUID vērtība, lai unikāli identificētu pozīciju.  |
