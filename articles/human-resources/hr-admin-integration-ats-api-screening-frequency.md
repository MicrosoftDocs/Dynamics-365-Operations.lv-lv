# <a name="screening-frequency"></a>Izvērtēšanas biežums


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā tēmā aprakstīta opcija Izvērtēšanas biežums, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums: mshr_hcmfrequencyunit

Šis uzskaitījums sniedz vērtību opciju kopu izvērtēšanas biežumam. 

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 Tikai vienu reizi | Izvērtēšana ir nepieciešama tikai vienu reizi. |
| 200000001 katru dienu | Izvērtēšanas biežums tiek aprēķināts dienās. |
| 200000002 reizi nedēļā | Izvērtēšanas biežums tiek aprēķināts nedēļās. |
| 200000003 reizi mēnesī | Izvērtēšanas biežums tiek aprēķināts mēnešos. |
| 200000004 reizi gadā | Izvērtēšanas biežums tiek aprēķināts gados. |
