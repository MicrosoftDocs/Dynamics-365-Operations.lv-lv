# <a name="applicant-integration-result"></a>Kandidātu integrācijas rezultāts

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā tēmā aprakstīta opcijas kopa Pieteikuma integrēcacijas ruzultāts, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums: mshr_hcmapplicantintegrationresult

Šis uzskaitījums nodrošina statusu kandidāta ierakstam.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | Nav apstrādāts | Kandidāts joprojām tiek apsvērts. |
| 200000001 | Pieņemts darbā | Kandidāts ir pieņemts darbā. |
| 200000002 | Nav pieņemts darbā | Netika pieņemts lēmums pieņemt darbā kandidātu. |
| 200000003 | Noraidīts | Apsvērumā kandidāts tika noraidīts. |
