# <a name="marital-status"></a>Ģimenes stāvoklis

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā tēmā aprakstīta opcija Ģimenes stāvokā statuss, kas iestatīta programmai Dynamics 365 Human Resources .

Fiziskais nosaukums:mshr_dirpersonmaritalstatus

Šis uzskaitījums sniedz statusa vērtību personas ierakstu ǵimenes stāvokļa opciju kopaai.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | None | Nav atlasīta vērtība.
| 200000001 | Viens | Vientuļš. |
| 200000002 | Precējies (-usies) | Precējies. |
| 200000003 | Šķīries (-usies) | Šķīries (-usies). |
| 200000004 | Atraitnis | Atraitnība. |
