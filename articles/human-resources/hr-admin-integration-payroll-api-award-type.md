[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā rakstā aprakstītas piemaksas tipa opcijas, kas iestatītas Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Opciju kopa: Piemaksas tips

Fiziskais nosaukums: mshr_HrmCompVarAwardEmplType

Šī uzskaitījuma nodrošina opciju, kas iestatīta mainīgās atlīdzības piemaksas tipam Human Resources virtuālajos entītijās.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 | Vispārēja piemaksa | Vispārēja piemaksa. |
| 200000001 | Gabalsummas fiksētā samaksa | Gabalsummas fiksētā samaksa. |
