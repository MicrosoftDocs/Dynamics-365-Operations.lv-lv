# <a name="screening-frequency-generate-from"></a>Izvērtēšanas biežums ǵenerāts no


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šajā rakstā ir aprakstīts izvērtēšanas biežuma ģenerēšana no opcijas, kas iestatīta Dynamics 365 Human Resources.

Fiziskais nosaukums: mshr_hcmfrequencygeneratefrom

Šis uzskaitījums nodrošina opciju vērtību kopas aprēķina sākuma datuma noteikšanai nākamajai nepieciešamai izvērtēšanai.

| Vērtība | Iezīme | Apraksts |
| --- | --- | --- |
| 200000000 Nav atlasīts | Nav atlasīta vērtība. |
| 200000001 Pabeigšanas datums | Aprēķins ir veikts no pēdējā izvērtēšanas datuma. |
| 200000002 Jānorāda datums | Aprēķins ir veikts no pēdējā nepieciešamā datuma. |
