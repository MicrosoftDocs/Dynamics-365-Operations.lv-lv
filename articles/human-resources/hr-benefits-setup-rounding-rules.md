# <a name="set-up-rounding-rules"></a>Iestatīt noapaļošanas noteikumus

[!include [banner](includes/preview-feature.md)]

Varat iestatīt noapaļošanas kārtulas pakalpojumā Dynamics 365 Human Resources lai apmierinātu savas organizācijas vajadzības.

1. Darbvietā **Atvieglojumu pārvaldība**, sadaļā **Iestatījumi** atlasiet **Noapaļošanas kārtulas**.

2. Atlasiet **Jauns**.

3. Norādiet vērtības **Noapaļošanai līdz tuvākajam**, **Scenārijam**un **Noapaļošanas veidlapai**.

4. Atlasiet **Saglabāt**.
