<span data-ttu-id="10467-104">Šajā tēmā paskaidrots, kā izveidot rezultātu pārskatu, un tajā aprakstīts katras pārskata nodaļas nolūks.</span><span class="sxs-lookup"><span data-stu-id="10467-104">This topic explains how to create a performance review and describes the purpose for each section of the review.</span></span> <span data-ttu-id="10467-105">Šī procedūra tika izveidota, izmantojot demonstrācijas uzņēmuma USMF datus.</span><span class="sxs-lookup"><span data-stu-id="10467-105">This procedure was created using the USMF demo data company.</span></span>

    1. <span data-ttu-id="10467-116">Atlasiet **Jā** vai **Nē** laukā **Rādīt pielikumus**, lai rādītu vai slēptu pielikumu cilni.</span><span class="sxs-lookup"><span data-stu-id="10467-116">Select **Yes** or **No** in the **Show attachments** field to show or hide the attachments tab.</span></span>
    1. <span data-ttu-id="10467-117">Atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="10467-117">Select **Save**.</span></span>

17. <span data-ttu-id="10467-128">Atlasiet cilni **Mērījumi**.</span><span class="sxs-lookup"><span data-stu-id="10467-128">Select the **Measurements** tab.</span></span>
18. <span data-ttu-id="10467-129">Atlasiet **Pievienot mērījumu**, lai atvērtu dialoglodziņa izvēlni.</span><span class="sxs-lookup"><span data-stu-id="10467-129">Select **Add measurement** to open the dialog menu.</span></span>
19. <span data-ttu-id="10467-130">Ievadiet vai atlasiet vērtību laukā **Mērījums**.</span><span class="sxs-lookup"><span data-stu-id="10467-130">In the **Measurement** field, enter or select a value.</span></span>
26. <span data-ttu-id="10467-131">Laukā **Mērķa summa** ievadiet skaitli.</span><span class="sxs-lookup"><span data-stu-id="10467-131">In the **Target amount** field, enter a number.</span></span>

21. <span data-ttu-id="10467-133">Atlasiet cilni **Darbības**.</span><span class="sxs-lookup"><span data-stu-id="10467-133">Select the **Activities** tab.</span></span>
22. <span data-ttu-id="10467-134">Atlasiet **Pievienot**.</span><span class="sxs-lookup"><span data-stu-id="10467-134">Select **Add**.</span></span>

    - <span data-ttu-id="10467-150">Veiktspējas periodam izveido noklusējuma sākuma un beigu datumus.</span><span class="sxs-lookup"><span data-stu-id="10467-150">The performance period creates the default start and end dates.</span></span> <span data-ttu-id="10467-151">Šie datumi ir rediģējami.</span><span class="sxs-lookup"><span data-stu-id="10467-151">Those dates are editable.</span></span>

34. <span data-ttu-id="10467-160">Atlasiet cilni **Pārskats**.
