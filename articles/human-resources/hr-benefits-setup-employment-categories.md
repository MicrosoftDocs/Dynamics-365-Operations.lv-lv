# <a name="create-employment-categories"></a>Nodarbinātības kategoriju izveide

[!include [banner](includes/preview-feature.md)]

Iestatiet nodarbinātības kategorijas, Dynamics 365 Human Resources, lai tās saskaņotu ar saviem nodarbinātajiem.

1. Darbvietā **Atvieglojumu pārvaldība**, sadaļā **Iestatījumi** atlasiet **Nodarbinātības kategorijas**.

2. Atlasiet **Jauns**.

3. Norādiet vērtības **Nodarbinātības veids**, **Nodarbinātības kategorija**un **Apraksts**.

4. Atlasiet **Saglabāt**.
