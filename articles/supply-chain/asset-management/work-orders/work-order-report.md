<span data-ttu-id="702dc-104">Jūs varat ģenerēt darba pasūtījumu pārskatu, kurā ir parādīta detalizēta informācija vienu vai vairākiem jūsu izvēlētiem darba pasūtījumiem.</span><span class="sxs-lookup"><span data-stu-id="702dc-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="702dc-105">Atlasiet **Pamatlīdzekļu pārvaldība** > **Pārskati** > **Darba pasūtījumi** > **Darba uzdevumu pārskats**.</span><span class="sxs-lookup"><span data-stu-id="702dc-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="702dc-106">Lai noteiktu informāciju, kas jāiekļauj pārskatā, pie pārslēgšanas pogas atlasiet “Jā”/“Nē” un aizpildiet laukus, kā tas ir pieprasīts.</span><span class="sxs-lookup"><span data-stu-id="702dc-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>

4. <span data-ttu-id="702dc-109">Kopsavilkuma cilnē **Palaist fonā** vajadzības gadījumā varat iestatīt darba pasūtījuma pārskatu veidošanu kā pakešuzdevumu.</span><span class="sxs-lookup"><span data-stu-id="702dc-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>
