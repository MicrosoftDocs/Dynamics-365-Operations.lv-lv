# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="dfb02-103">Funkcionālā novietojuma dzīves cikla stāvokļi</span><span class="sxs-lookup"><span data-stu-id="dfb02-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="dfb02-104">Šajā tēmā ir aprakstīts, kā iestatīt funkcionālā novietojuma dzīves cikla stāvokļus un dzīves cikla modeļus Līdzekļu pārvaldībā.</span><span class="sxs-lookup"><span data-stu-id="dfb02-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="dfb02-105">Funkcionālā novietojuma dzīves cikla stāvokļi definē stāvokļus, kādi var būt funkcionālajam novietojumam, piemēram, izveidots, aktīvs un beidzies.</span><span class="sxs-lookup"><span data-stu-id="dfb02-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="dfb02-106">Varat skatīt visus funkcionālos novietojumus neatkarīgi no to dzīves cikla stāvokļa saraksta lapā **Visas funkcionālie novietojumi**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="dfb02-107">Funkcionālā novietojuma stāvokli var mainīt, atlasot to saraksta lapā **Visi funkcionālie novietojumi** un atlasot vienumu **Atjaunināt funkcionālā novietojuma stāvokli**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="dfb02-108">Funkcionālā novietojuma dzīves cikla stāvokļu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="dfb02-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="dfb02-109">Atlasiet **Līdzekļu pārvaldība** > **Iestatīšana** > **Funkcionālie novietojumi** > **Dzīves cikla stāvokļi**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="dfb02-110">Atlasiet **Jauns**, lai izveidotu jaunu funkcionālā novietojuma stāvokli.</span><span class="sxs-lookup"><span data-stu-id="dfb02-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="dfb02-111">Ievietojiet stāvokļa ID laukā **Dzīves cikla stāvoklis** un funkcionālā novietojuma stāvokļa nosaukumu laukā **Nosaukums**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="dfb02-112">Laukā **Dzīves cikla modeļi** varat redzēt to funkcionālo novietojumu dzīves cikla modeļu skaitu, kuri izmanto funkcionālā novietojuma stāvokli.</span><span class="sxs-lookup"><span data-stu-id="dfb02-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="dfb02-113">Kopsavilkuma cilnē **Vispārīgi** atlasiet "Jā" uz pārslēgšanas pogas **Aktīvs**, ja funkcionālajam novietojumam vajadzētu būt aktīvam šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="dfb02-114">Atlasiet "Jā" uz pārslēgšanas pogas **Izveidot līdzekļus**, ja vajadzētu būt iespējamam automātiski izveidot līdzekli ar tādu pašu nosaukumu kā funkcionālajam novietojumam un uzstādīt to funkcionālajā novietojumā šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="dfb02-115">Šī pārslēgšanas poga attiecas uz lauku **Līdzekļa veids** kopsavilkuma cilnē **Vispārīgi** veidlapā **Funkcionālo novietojumu veidi** (**Līdzekļu pārvaldība** > **Iestatīšana** > **Funkcionālie novietojumi** > **Funkcionālo novietojumu veidi**).</span><span class="sxs-lookup"><span data-stu-id="dfb02-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="dfb02-116">Atlasiet "Jā" uz pārslēgšanas pogas **Pārdēvēt atrašanās vietu**, ja vajadzētu būt iespējamam mainīt funkcionālā novietojuma nosaukumu šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="dfb02-117">Atlasiet "Jā" uz pārslēgšanas pogas **Jaunas apakšvietas**, ja vajadzētu būt iespējamam pievienot jaunas apakšvietas funkcionālajam novietojumam šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="dfb02-118">Atlasiet "Jā" uz pārslēgšanas pogas **Uzstādīt līdzekļus**, ja vajadzētu būt iespējamam uzstādīt līdzekļus funkcionālajā novietojumā šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="dfb02-119">Atlasiet "Jā" uz pārslēgšanas pogas **Dzēst funkcionālo novietojumu**, ja vajadzētu būt iespējamam dzēst funkcionālo novietojumu šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="dfb02-120">Atlasiet līdzekļa stāvokli laukā **Dzīves cikla stāvoklis**, ja vēlaties, lai līdzekļa dzīves cikla stāvoklis visiem līdzekļiem, kas uzstādīti funkcionālajā novietojumā, tiktu automātiski atjaunināts šajā stāvoklī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="dfb02-121">Piemērs: ja aizverat funkcionālo novietojumu un iestatāt funkcionālā novietojuma dzīves cikla stāvokli uz "Pabeigts", iespējams, vēlēsities automātiski mainīt šajā funkcionālajā novietojumā uzstādīto līdzekļu dzīves cikla stāvokli uz "Netiek lietots".</span><span class="sxs-lookup"><span data-stu-id="dfb02-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="dfb02-122">Funkcionālā novietojuma dzīves cikla stāvokļi, dzīves cikla modeļi un veidi ir saistīti un tiek izmantoti tādā pašā veidā kā darba pasūtījuma dzīves cikla stāvokļi, darba pasūtījuma dzīves cikla modeļi un darba pasūtījumu veidi.</span><span class="sxs-lookup"><span data-stu-id="dfb02-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="dfb02-123">Funkcionālā novietojuma dzīves cikla modeļu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="dfb02-123">Set up functional location lifecycle models</span></span>

<span data-ttu-id="dfb02-124">Kad esat izveidojis dzīves cikla stāvokļus, kas nepieciešami jūsu funkcionālajam novietojumam, tos var iedalīt grupās.</span><span class="sxs-lookup"><span data-stu-id="dfb02-124">When you have created the lifecycle states required for your functional locations, they can be divided into groups.</span></span> <span data-ttu-id="dfb02-125">Tas tiek darīts, lai izveidotu dzīves cikla modeļu plūsmu, ko var izmantot dažādiem funkcionālo novietojumu veidiem.</span><span class="sxs-lookup"><span data-stu-id="dfb02-125">This is done to create the lifecycle model flow that may be used for different types of functional locations.</span></span> <span data-ttu-id="dfb02-126">Jāizveido vismaz viens standarta funkcionālā novietojuma dzīves cikla modelis.</span><span class="sxs-lookup"><span data-stu-id="dfb02-126">As a minimum, one standard functional location lifecycle model should be created.</span></span>

1. <span data-ttu-id="dfb02-127">Atlasiet **Līdzekļu pārvaldība** > **Iestatīšana** > **Funkcionālie novietojumi** > **Dzīves cikla modeļi**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-127">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle models**.</span></span>
2. <span data-ttu-id="dfb02-128">Atlasiet **Jauns**, lai izveidotu dzīves cikla modeli.</span><span class="sxs-lookup"><span data-stu-id="dfb02-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="dfb02-129">Ievietojiet dzīves cikla modeļa ID laukā **Dzīves cikla modelis** un dzīves cikla modeļa nosaukumu laukā **Nosaukums**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="dfb02-130">Laukos **Funkcionālo novietojumu veidi** un **Dzīves cikla stāvokļi** varat redzēt to funkcionālo novietojumu veidu skaitu, kas izmanto dzīves cikla modeli, kā arī dzīves cikla modelī atlasīto stāvokļu skaitu.</span><span class="sxs-lookup"><span data-stu-id="dfb02-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="dfb02-131">Kopsavilkuma cilnē **Dzīves cikla stāvokļi** atlasiet stāvokļus, kuri būtu jāiekļauj modelī.</span><span class="sxs-lookup"><span data-stu-id="dfb02-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="dfb02-132">Tas tiek darīts, noklikšķinot uzstāvokļa sadaļā **Atlikušie dzīves cikla stāvokļi** un noklikšķinot uz pogas ![bultiņa uz priekšu](media/02-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="dfb02-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="dfb02-133">Ja vēlaties atlasīt visus modelim pieejamos stāvokļus, noklikšķiniet uz pogas ![atlasīt visus pieejamos posmus](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="dfb02-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="dfb02-134">Visi stāvokļi tiek nosūtīti uz sadaļu **Atlasītie dzīves cikla stāvokļi**.</span><span class="sxs-lookup"><span data-stu-id="dfb02-134">All states are transferred to the **Lifecycle states selected** section.</span></span>
6. <span data-ttu-id="dfb02-135">Ja vēlaties modelima noņemt atlasīto stāvokli, atlasiet stāvokli sadaļā **Atlasītie dzīves cikla stāvokļi** un pēc tam atlasiet pogu ![bultiņa atpakaļ](media/04-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="dfb02-135">If you want to remove a selected state from the model, select the state in the **Lifecycle states selected** section and then select the ![back arrow](media/04-setup-for-functional-locations.png) button.</span></span>
7. <span data-ttu-id="dfb02-136">Atlasiet **Dzīves cikla stāvokļa atjauninājumi**, lai definētu, kuri dzīves cikla stāvokļi var sekot atlasītajam stāvoklim.</span><span class="sxs-lookup"><span data-stu-id="dfb02-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
