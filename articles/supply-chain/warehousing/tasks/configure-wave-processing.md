# <a name="configure-wave-processing"></a><span data-ttu-id="909bc-103">Kopuma apstrādes konfigurēšana</span><span class="sxs-lookup"><span data-stu-id="909bc-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="909bc-104">Šajā ceļvedī aprakstīts, kā iestatīt kritērijus, kas nosaka, vai kopumi tiek apstrādāti manuāli vai automātiski, un darbam, kas tiek ģenerēts noliktavai, apstrādājot kopumu, kā arī vai kopumi tiek apstrādāti automātiski vai manuāli.</span><span class="sxs-lookup"><span data-stu-id="909bc-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="909bc-105">Norādiet kritērijus, izveidojot kopuma veidnes un vaicājumus, kas sakrīt ar kopumu izsniegtajām rindām pārdošanas pasūtījumos, ražošanas pasūtījumos un kanban.</span><span class="sxs-lookup"><span data-stu-id="909bc-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="909bc-106">Kopuma apstrādes tiek izmantotas noliktavās, kuras izmanto noliktavas vadības moduli, un ne tās, kas izmanto funkcionalitāti moduļa krājuma vadībā.</span><span class="sxs-lookup"><span data-stu-id="909bc-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="909bc-107">Šo procedūru varat palaist demonstrācijas datu uzņēmumā USMF.</span><span class="sxs-lookup"><span data-stu-id="909bc-107">You can run this procedure in demo data company USMF.</span></span>

1. <span data-ttu-id="909bc-108">Dodieties uz **Navigācijas rūts > Moduļi > Noliktavas pārvaldība > Uzstādīšana > Kopumi > Kopumu veidnes**.</span><span class="sxs-lookup"><span data-stu-id="909bc-108">Go to **Navigation pane > Modules > Warehouse management > Setup > Waves > Wave templates**.</span></span>
2. <span data-ttu-id="909bc-109">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="909bc-109">Click **New**.</span></span>
3. <span data-ttu-id="909bc-110">Ierakstiet vērtību laukā **Kopuma veidnes nosaukums**.</span><span class="sxs-lookup"><span data-stu-id="909bc-110">In the **Wave template name** field, type a value.</span></span> <span data-ttu-id="909bc-111">Iestatot kopuma veidni, norādiet secību, kādā veidnes tiks saskaņotas ar izlaistajām rindām pārdošanas pasūtījumos, ražošanas pasūtījumos vai kanban.</span><span class="sxs-lookup"><span data-stu-id="909bc-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="909bc-112">Kad rindas tiek nodotas uz noliktavu vai uz ražošanu, tiek izmantota pirmā kopuma veidne, kas atbilst kritērijiem.</span><span class="sxs-lookup"><span data-stu-id="909bc-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="909bc-113">Mēs iesakām šī saraksta sākumā novietot veidnes ar visspecifiskākajiem kritērijiem.</span><span class="sxs-lookup"><span data-stu-id="909bc-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="909bc-114">Jo plašāki kritēriji, jo vairāk iespējams, ka rinda atbildīs kritērijiem, un rezultātā rindas var tikt piešķirtas nepareizajam kopumam.</span><span class="sxs-lookup"><span data-stu-id="909bc-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>  
4. <span data-ttu-id="909bc-115">Laukā **Kopuma veidnes apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="909bc-115">In the **Wave template description** field, type a value.</span></span>
5. <span data-ttu-id="909bc-116">Laukā **Vieta** ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="909bc-116">In the **Site** field, enter or select a value.</span></span> <span data-ttu-id="909bc-117">Ja izmantojat USMF, varat atlasīt "Vieta 2".</span><span class="sxs-lookup"><span data-stu-id="909bc-117">If you’re using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="909bc-118">Laukā **Noliktava** ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="909bc-118">In the **Warehouse** field, enter or select a value.</span></span> <span data-ttu-id="909bc-119">Ja izmantojat krājumu USMF, varat atlasīt 24. noliktavu.</span><span class="sxs-lookup"><span data-stu-id="909bc-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="909bc-120">Iestatiet lauku **Automatizēt kopuma izveidi** uz **Jā**.</span><span class="sxs-lookup"><span data-stu-id="909bc-120">Set the **Automate wave creation** field to **Yes**.</span></span> <span data-ttu-id="909bc-121">Atlasiet šo opciju, lai automātiski izveidotu kopumu, kad pārdošanas pasūtījums, ražošanas pasūtījums vai Kanban tiek nodotas izpildei noliktavā.</span><span class="sxs-lookup"><span data-stu-id="909bc-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>  
8. <span data-ttu-id="909bc-122">Iestatiet opciju **Apstrādāt kopumu, to pārvietojot uz noliktavu** uz **Jā**.</span><span class="sxs-lookup"><span data-stu-id="909bc-122">Set the **Process wave at release to warehouse** option to **Yes**.</span></span> <span data-ttu-id="909bc-123">Atlasiet šo opciju, lai automātiski apstrādātu kopumu un izveidotu darbu, kad rinda tiek nodotas izpildei noliktavā.</span><span class="sxs-lookup"><span data-stu-id="909bc-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>  
9. <span data-ttu-id="909bc-124">Iestatiet opciju **Automatizēt kopuma izlaidi** uz **Jā**.</span><span class="sxs-lookup"><span data-stu-id="909bc-124">Set the **Automate wave release option** to **Yes**.</span></span> <span data-ttu-id="909bc-125">Atlasiet šo opciju, lai automātiski izlaistu kopumu.</span><span class="sxs-lookup"><span data-stu-id="909bc-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="909bc-126">Izdošanas darbs ir izveidots un pieejams mobilajās ierīcēs.</span><span class="sxs-lookup"><span data-stu-id="909bc-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="909bc-127">Iestatiet opciju **Piešķirt atvērtiem kopumiem** uz **Jā**.</span><span class="sxs-lookup"><span data-stu-id="909bc-127">Set the **Assign to open waves option** to **Yes**.</span></span> <span data-ttu-id="909bc-128">Rindas tiek piešķirtas kopumiem, pamatojoties uz kopuma veidnes vaicājuma filtru.</span><span class="sxs-lookup"><span data-stu-id="909bc-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="909bc-129">Iestatiet opciju **Apstrādāt kopumu automātiski pie sliekšņvērtības** uz **Jā**.</span><span class="sxs-lookup"><span data-stu-id="909bc-129">Set the **Process wave automatically at threshold option** to **Yes**.</span></span> <span data-ttu-id="909bc-130">Atlasiet šo opciju, lai automātiski apstrādātu kopumu, kad tā vērtības sasniedz svara, sūtījuma un lauku grupā Kopuma robežvērtības norādīto rindu robežvērtības.</span><span class="sxs-lookup"><span data-stu-id="909bc-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="909bc-131">Šī opcija ir pieejama tikai tad, ja laukā Kopuma veidnes tips ir atlasīta opcija Nosūtīšana.</span><span class="sxs-lookup"><span data-stu-id="909bc-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>

13. <span data-ttu-id="909bc-135">Izvērsiet sadaļu **Metodes**.</span><span class="sxs-lookup"><span data-stu-id="909bc-135">Expand the **Methods** section.</span></span>

    - <span data-ttu-id="909bc-136">Kopuma veidnes metodes ļauj kontrolēt darbību secību, kurām katrs kopums iet cauri, kad tas tiek apstrādāts.</span><span class="sxs-lookup"><span data-stu-id="909bc-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it’s processed.</span></span> <span data-ttu-id="909bc-137">Piemēram, jums var būt kopuma papildināšanas metode.</span><span class="sxs-lookup"><span data-stu-id="909bc-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="909bc-138">Pievienojot metodi, tā automātiski tiek uzskaitīta atbilstošajā atrašanās vietā darbību secībā.</span><span class="sxs-lookup"><span data-stu-id="909bc-138">When you add a method, it’s automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="909bc-139">Ja esat iestatījis Automatizēt papildināšanas darba izpildes opciju kā Jā, jums ir jāpievieno papildināšanas metode šeit.</span><span class="sxs-lookup"><span data-stu-id="909bc-139">If you’ve set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>
