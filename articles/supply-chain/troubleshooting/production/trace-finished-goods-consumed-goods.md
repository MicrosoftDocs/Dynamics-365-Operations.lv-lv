# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Nevar izsekot gatavo preču sērijas numurus pret patērēto preču sērijas numuriem

## <a name="symptoms"></a>Simptomi

Nevar izsekot gatavo preču sērijas numurus ar materiālu sērijas numuriem, ko ražošanas pasūtījums patērē, lai izveidotu šīs pabeigtās preces.

## <a name="workaround"></a>Kļūdas apiešanas risinājums

Risinājums ir izveidot ražošanas pasūtījumus 1 daudzumam.
