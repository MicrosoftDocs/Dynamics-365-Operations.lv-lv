# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Resursa nosaukums netiek atjaunināts pēc darbinieka nosaukuma maiņas adrešu grāmatā

## <a name="symptoms"></a>Simptomi

Ja maināt darbinieka nosaukumu globālajā adrešu grāmatā, salīdzināšanas resursa nosaukums netiek atjaunināts.

## <a name="resolution"></a>Novēršana

Atbilstošā resursa nosaukums šobrīd netiek atjaunināts automātiski. Lai labotu problēmu, ir manuāli jāatjaunina resursa nosaukums.
