# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>Plānoto pasūtījumu apstiprināšanas laikā nav norādīts kreditors

Kļūdas kods: SYS23532

## <a name="symptoms"></a>Simptomi

Mēģinot apstiprināt plānotos pasūtījumus saņemsit šādu kļūdas ziņojumu:

> Nav norādīts kreditors

## <a name="resolution"></a>Novēršana

Lai norādītu kreditoru, veiciet tālāk aprakstītās darbības.

1. Atveriet plānoto pasūtījumu, kurā trūkst kreditora.
1. Kopsavilkuma cilnē **Plānotā piegāde** laukā **Kreditors** atlasiet kreditoru.
