# <a name="inventory-aging-report-discrepancies"></a>Krājumu vecumstruktūru pārskata neatbilstība

## <a name="symptoms"></a>Simptomi

**Krājumu vecumstruktūru pārskats** rāda dažādas vērtības, ja tās tiek apskatītas dažādās noliktavas dimensijās (piemēram, vietnē vai noliktavā).

## <a name="resolution"></a>Novēršana

Lai iegūtu papildu informāciju par pārskata loģiku, skatiet [Krājumu vecumstruktūru pārskata piemēri un loģika](../../cost-management/inventory-aging-report.md).
