# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Nederīga numura zīme, veicot ID skenēšanu mobilajā programmā

## <a name="symptoms"></a>Simptomi

Skenējot numura zīmes ID Warehouse Management mobilajā programmā, iespējams, saņemsit šādu kļūdas ziņojumu:

> Numura zīme nav derīga.

## <a name="resolution"></a>Novēršana

Pārliecinieties, ka numura zīmes ID pastāv numura zīmju tabulā un ka numura zīmes krājumi un daudzumi ir pieejami (respektīvi, tie nav bloķēti).
