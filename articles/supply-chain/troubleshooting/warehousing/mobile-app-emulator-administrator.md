# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Piekļūt mobilās programmas emulatoram var tikai ar administratora lomu

## <a name="symptoms"></a>Simptomi

Iestatot noliktavas pakalpojumā Supply Chain Management, piekļūt mobilās programmas emulatoram var tikai ar administratora lomu.

## <a name="resolution"></a>Novēršana

Mobilās ierīces programmas emulators ir iestatīts darbam tikai ar administratora kontu. Visiem pārbaudes un klātienes apstrādes nolūkiem mēs iesakām izmantot pašu Warehouse Management mobile programmu.
