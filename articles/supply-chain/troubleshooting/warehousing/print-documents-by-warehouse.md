# <a name="print-a-delivery-note-or-packing-content-by-warehouse-or-site"></a>Noliktavas vai vietas piegādes ziņojuma vai iepakojuma satura drukāšana

## <a name="symptoms"></a>Simptomi

Jūs vēlaties drukāt piegādes pavadzīmi vai iepakojuma saturu pēc noliktavas vai vietnes lapā **Darba audita veidnes rindas atjaunināšana**.

## <a name="resolution"></a>Novēršana

Drukājot dokumentu, izmantojot Drukāšanas pārvaldības iestatījumus, ierobežojiet tvērumu (vieta/noliktava), izmantojot Drukāšanas pārvaldību, nevis lapā **Darba audita veidnes rindas atjauninājums** atlasot **Rediģēt drukas iestatījumus**.
