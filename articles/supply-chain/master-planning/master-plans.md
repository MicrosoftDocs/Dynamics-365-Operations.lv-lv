# <a name="master-plans-overview"></a><span data-ttu-id="730ec-103">Vispārējo plānu pārskats</span><span class="sxs-lookup"><span data-stu-id="730ec-103">Master plans overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="730ec-104">Izmantojiet dažādus vispārējos plānus, lai atbalstītu uzņēmuma ikdienas darba operācijas, simulētu dažādas plānošanas stratēģijas, ko vēlaties pārraudzīt, un īstenotu uzņēmuma politiku, piemēram, attiecībā uz iekšējo veiktspēju vai debitoru apmierinātību.</span><span class="sxs-lookup"><span data-stu-id="730ec-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="730ec-105">Vispārējos plānus varat konfigurēt lapā **Vispārējie plāni**.</span><span class="sxs-lookup"><span data-stu-id="730ec-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="730ec-106">Ir divi plānu tipi:</span><span class="sxs-lookup"><span data-stu-id="730ec-106">There are two types of plans:</span></span>
-   <span data-ttu-id="730ec-107">**Statiskais plāns** — vispārējās plānošanas aprēķinā tiek izmantoti pašreizējie dati, lai izveidotu neto vajadzību plānu.</span><span class="sxs-lookup"><span data-stu-id="730ec-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="730ec-108">Šis plāns netiek mainīts līdz nākamajai reize, kad veicat vispārējo plānošanu.</span><span class="sxs-lookup"><span data-stu-id="730ec-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="730ec-109">Tas ir darbības plāns, ko var izmantot dažādi uzņēmuma darbinieki, piemēram, sagādnieks vai ražošanas plānotājs, lai pieņemtu pamatotus lēmumus un veiktu ikdienas uzdevumus un aktivitātes.</span><span class="sxs-lookup"><span data-stu-id="730ec-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="730ec-110">**Dinamiskais plāns** — sākumā šis plāns neatšķiras no neto vajadzību plāna, kas tiek izveidots, veicot vispārējo plānošanu.</span><span class="sxs-lookup"><span data-stu-id="730ec-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="730ec-111">Taču dinamisko plānu var atjaunināt ikreiz, kad mainās pamatdati.</span><span class="sxs-lookup"><span data-stu-id="730ec-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="730ec-112">Tas var notikt, piemēram, izveidojot jaunu pārdošanas pasūtījumu.</span><span class="sxs-lookup"><span data-stu-id="730ec-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="730ec-113">Tas ļauj pārraudzīt mainīgo pasūtījumu tīklu un krājumu pieejamību, neizjaucot statisko plānu, ko citi izmanto saviem darba procesiem.</span><span class="sxs-lookup"><span data-stu-id="730ec-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="730ec-114">Uzņēmums var izvēlēties strādāt tikai ar dinamisko plānu vai izmantot gan statisko, gan dinamisko plānu.</span><span class="sxs-lookup"><span data-stu-id="730ec-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="730ec-115">Turklāt var konfigurēt jebkuru vispārējo plānu, lai parādītu noteiktu stratēģiju vai pievērstos kādam jautājumam.</span><span class="sxs-lookup"><span data-stu-id="730ec-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="730ec-116">Tālāk ir sniegti piemēri.</span><span class="sxs-lookup"><span data-stu-id="730ec-116">Examples are as follows:</span></span>
-   <span data-ttu-id="730ec-117">Iestatiet austāku krājumu līmeni, lai garantētu nodrošinājumu pret iztrūkumiem.</span><span class="sxs-lookup"><span data-stu-id="730ec-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="730ec-118">Iestatiet ilgākas drošības rezerves, lai nodrošinātu aizsardzību pret neuzticamiem kreditoriem.</span><span class="sxs-lookup"><span data-stu-id="730ec-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="730ec-119">Varat arī iestatīt sākotnējo dinamisko plānu tā, lai tas tiktu atjaunots ar jauno vajadzību plānu ikreiz, kad veicat vispārējo plānošanu.</span><span class="sxs-lookup"><span data-stu-id="730ec-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="730ec-120">Šos iestatījumus varat norādīt lapā **Vispārējās plānošanas parametri**.</span><span class="sxs-lookup"><span data-stu-id="730ec-120">You can specify these settings on the **Master planning parameters** page.</span></span>
