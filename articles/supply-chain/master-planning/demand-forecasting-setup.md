<span data-ttu-id="14aba-118">Pēc noklusējuma, ja starpuzņēmumu plānošanas grupas dalībniekiem nav piešķirts neviens krājumu sadalījuma princips, pieprasījuma apjoma prognoze tiek aprēķināta visiem krājumiem, kas ir piešķirti visiem krājumu sadalījuma principiem visos Dynamics 365 for Finance and Operations uzņēmumos.</span><span class="sxs-lookup"><span data-stu-id="14aba-118">By default, if no item allocation keys are assigned to intercompany planning group members, a demand forecast is calculated for all items that are assigned to all item allocation keys from all Finance and Operations companies.</span></span> <span data-ttu-id="14aba-119">Lapā **Statistiskās bāzlīnijas prognozes ģenerēšana** ir pieejamas papildu uzņēmumu un krājumu sadalījuma principu filtrēšanas opcijas.</span><span class="sxs-lookup"><span data-stu-id="14aba-119">Additional filtering options for companies and item allocation keys are available on the **Generate statistical baseline forecast** page.</span></span> 

<span data-ttu-id="14aba-120">Pārskatiet prognozēto krājumu skaitu.</span><span class="sxs-lookup"><span data-stu-id="14aba-120">Review the number of items that are forecasted.</span></span> <span data-ttu-id="14aba-121">Nevajadzīgie krājumi var radīt izmaksu palielināšanos, ja tiek izmantots pakalpojums Microsoft Azure Machine Learning.</span><span class="sxs-lookup"><span data-stu-id="14aba-121">Unnecessary items might cause increased costs when you use Microsoft Azure Machine Learning.</span></span>

<span data-ttu-id="14aba-123">Lai iestatītu pieprasījuma prognozēšanas parametrus, dodieties uz **Vispārējā plānošana** &gt; **Iestatīšana** &gt; **Pieprasījuma prognozēšanas parametri**.</span><span class="sxs-lookup"><span data-stu-id="14aba-123">To set up demand forecasting parameters, go to **Master planning** &gt; **Setup** &gt; **Demand forecasting parameters**.</span></span> <span data-ttu-id="14aba-124">Tā kā pieprasījuma prognozēšana ietver starpuzņēmumus, iestatījumiem ir globāla nozīme.</span><span class="sxs-lookup"><span data-stu-id="14aba-124">Because demand forecasting runs cross-company, the setup is global.</span></span> <span data-ttu-id="14aba-125">Tas nozīmē, ka iestatījumi attiecas uz visiem uzņēmumiem.</span><span class="sxs-lookup"><span data-stu-id="14aba-125">In other words, the setup applies to all companies.</span></span>
