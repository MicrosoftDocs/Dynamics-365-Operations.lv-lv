# <a name="set-up-a-transportation-tender"></a>Transportēšanas norēķinu iestatīšana

[!include [banner](../../includes/banner.md)]

Šajā procedūrā parādīts, kā iestatīt transportēšanas norēķinus. To parasti veic transportēšanas koordinators. Šo procedūru varat lietot ar demonstrācijas datu uzņēmumu USMF.


## <a name="select-a-route"></a>Atlasīt maršrutu
1. Dodieties uz Transportēšanas pārvaldība > Plānošana > Kravu plānošanas rīks.
2. Noņemiet atzīmi izvēles rūtiņai Slēpt nosūtītos un saņemtos.
3. Atlasiet rindu ar Kravas ID 00006.
4. Noklikšķiniet uz Novērtēšana un maršrutēšana.
5. Noklikšķiniet uz Maršruti.

## <a name="create-the-transportation-tender"></a>Izveidot transportēšanas norēķinu
1. Noklikšķiniet uz Transportēšanas norēķini.
2. Noklikšķiniet uz Jauns.
3. Izvērsiet sadaļu Vispārīgi.
4. Laukā Pieprasītās likmes, ievadiet skaitli.
5. Noklikšķiniet uz Saglabāt.
6. Noklikšķiniet uz Atjaunināt statusu.
7. Klikšķiniet Iesniegt.
8. Atlasīt maršrutu.
