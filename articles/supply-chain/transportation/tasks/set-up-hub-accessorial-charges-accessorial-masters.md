# <a name="set-up-hub-accessorial-charges-and-accessorial-masters"></a>Pārkraušanas punkta papildobjekta maksas un papildobjekta šablonu iestatīšana

[!include [banner](../../includes/banner.md)]

Šajā procedūrā parādīts, kā izveidot papildobjekta šablonu pārkraušanas punktam un izmantot šo šablonu, lai izveidotu pārkraušanas punkta papildobjekta maksu. Procedūrā tiek izmantota USMF datu kopa. Šo iestatīšanu parasti veic transportēšanas koordinators.


## <a name="set-up-a-hub-master"></a>Pārkraušanas punkta šablona iestatīšana
1. Dodieties uz Transportēšanas pārvaldība > Iestatīšana > Vērtējums > Papildobjekta šabloni.
2. Noklikšķiniet uz Jauns.
3. Laukā Papildobjekta šablons ierakstiet vērtību.
4. Laukā Nosaukums ierakstiet kādu vērtību.
5. Laukā Papildobjekta tips atlasiet "Pārkraušanas punkts".
6. Noklikšķiniet uz Saglabāt.
7. Aizvērt lapu.

## <a name="set-up-a-hub-accessorial-charge"></a>Pārkraušanas punkta papildobjekta maksas iestatīšana
1. Dodieties uz Transportēšanas pārvaldība > Iestatīšana > Vērtējums > Pārkraušanas punkta papildobjekta maksas.
2. Noklikšķiniet uz Jauns.
3. Laukā Pārkraušanas punkta papildobjekta ID ierakstiet vērtību.
4. Laukā Pārkraušanas punkts noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.
5. Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.
6. Laukā Pārkraušanas punkta pozīcija atlasiet opciju.
    * Varat izveidot maksu kā saņemšanas vai nodošanas maksu. Atkarībā no veiktās atlases maksa tiks piemērota atbilstošam transportēšanas segmentam jūsu maršrutā.  
7. Laukā Papildobjekta šablons noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.
8. Sarakstā noklikšķiniet uz saites atlasītajā rindā.
    * Atlasiet tikko izveidoto šablonu.  
9. Noklikšķiniet uz Saglabāt.
10. Aizvērt lapu.
