# <a name="set-up-an-appointment-for-a-load"></a>Kravas norīkojuma iestatīšana

[!include [banner](../../includes/banner.md)]

Šajā procedūrā parādīts kā iestatīt un plānot doka norīkojumu kravai. To parasti veic transportēšanas koordinators. Šo procedūru varat lietot ar demonstrācijas datu uzņēmumu USMF.


## <a name="select-the-load"></a>Atlasīt kravu
1. Dodieties uz Transportēšanas pārvaldība > Plānošana > Kravu plānošanas rīks.
2. Noņemiet atzīmi izvēles rūtiņai Slēpt nosūtītos un saņemtos.
3. Sarakstā atlasiet kravu, kuras statuss ir Nosūtīts.
4. Noklikšķiniet uz Transportēšana.
5. Noklikšķiniet uz Norīkojumu plānošana.

## <a name="create-an-appointment"></a>Izveidot norīkojumu
1. Noklikšķiniet uz Jauns.
2. Laukā Norīkojuma kārtula ievadiet vai atlasiet kādu vērtību.
3. Noklikšķiniet uz Saglabāt.
4. Noklikšķiniet uz Atjaunināt statusu.
5. Noklikšķiniet uz Apstiprināt.
6. Noklikšķiniet uz Saglabāt.
7. Aizvērt lapu.
