# <a name="create-time-windows"></a><span data-ttu-id="7242a-103">Laika loga izveide</span><span class="sxs-lookup"><span data-stu-id="7242a-103">Create time windows</span></span>   

[!include [banner](../includes/banner.md)]

<span data-ttu-id="7242a-104">Laika logus varat izmantot, lai optimizētu pakalpojuma pasūtījumu rindu plānošanas.</span><span class="sxs-lookup"><span data-stu-id="7242a-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="7242a-105">Noklikšķiniet uz **Pakalpojumu pārvaldība** \> **Iestatījumi** \> **Pakalpojumu līgumi** \> **Laika logs**.</span><span class="sxs-lookup"><span data-stu-id="7242a-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>
2. <span data-ttu-id="7242a-106">Nospiediet taustiņu kombināciju **Alt+N**, lai izveidotu jaunu laika logu.</span><span class="sxs-lookup"><span data-stu-id="7242a-106">Press **Alt+N** to create a new time window.</span></span>
3. <span data-ttu-id="7242a-107">Laukā **Laika logs** ievadiet laika loga nosaukumu.</span><span class="sxs-lookup"><span data-stu-id="7242a-107">In the **Time window** field, enter a name for the time window.</span></span>
