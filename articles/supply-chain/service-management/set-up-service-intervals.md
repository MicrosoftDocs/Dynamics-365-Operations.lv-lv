# <a name="set-up-service-intervals"></a><span data-ttu-id="8bd3c-103">Pakalpojumu intervālu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="8bd3c-103">Set up service intervals</span></span>  

[!include[banner](../includes/banner.md)]

<span data-ttu-id="8bd3c-104">Pakalpojumu intervāls norāda biežumu, ar kādu pakalpojuma pasūtījuma rindas tiek izveidotas pakalpojumu līguma rindām, kad veidojat pakalpojumu pasūtījumus.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="8bd3c-105">Noklikšķiniet uz **Pakalpojumu pārvaldība** \> **Iestatījumi** \> **Pakalpojumu līgumi** \> **Pakalpojumu intervāli**.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="8bd3c-106">Izveidojiet jaunu pakalpojumu intervālu.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-106">Create a new service interval.</span></span>
3. <span data-ttu-id="8bd3c-107">Ievadiet pakalpojumu intervāla ID un aprakstu.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="8bd3c-108">Lauka **Diapazons** atlasiet diapazonu.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="8bd3c-109">Laukā **Biežums** ierakstiet biežuma veidu.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="8bd3c-110">Biežums ir koeficents, ar kuru jāreizina vērtības, lai iegūtu pakalpojumu līguma intervālu.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="8bd3c-111">Lai saglabātu pakalpojumu intervālu, nospiediet taustiņu kombināciju**Alt+S**.</span><span class="sxs-lookup"><span data-stu-id="8bd3c-111">Press **Alt+S** to save the service interval.</span></span>
