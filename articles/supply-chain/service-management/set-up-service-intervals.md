# <a name="set-up-service-intervals"></a><span data-ttu-id="a65e7-103">Pakalpojumu intervālu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="a65e7-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a65e7-104">Pakalpojumu intervāls norāda biežumu, ar kādu pakalpojuma pasūtījuma rindas tiek izveidotas pakalpojumu līguma rindām, kad veidojat pakalpojumu pasūtījumus.</span><span class="sxs-lookup"><span data-stu-id="a65e7-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="a65e7-105">Noklikšķiniet uz **Pakalpojumu pārvaldība** \> **Iestatījumi** \> **Pakalpojumu līgumi** \> **Pakalpojumu intervāli**.</span><span class="sxs-lookup"><span data-stu-id="a65e7-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="a65e7-106">Izveidojiet jaunu pakalpojumu intervālu.</span><span class="sxs-lookup"><span data-stu-id="a65e7-106">Create a new service interval.</span></span>
3. <span data-ttu-id="a65e7-107">Ievadiet pakalpojumu intervāla ID un aprakstu.</span><span class="sxs-lookup"><span data-stu-id="a65e7-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="a65e7-108">Lauka **Diapazons** atlasiet diapazonu.</span><span class="sxs-lookup"><span data-stu-id="a65e7-108">In the **Range** field, select the range.</span></span>

6. <span data-ttu-id="a65e7-111">Lai saglabātu pakalpojumu intervālu, nospiediet taustiņu kombināciju **Alt+S**.</span><span class="sxs-lookup"><span data-stu-id="a65e7-111">Press **Alt+S** to save the service interval.</span></span>
