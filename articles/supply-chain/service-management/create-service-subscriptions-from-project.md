# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="43c99-103">Pakalpojuma abonementu izveide no projekta</span><span class="sxs-lookup"><span data-stu-id="43c99-103">Create service subscriptions from a project</span></span>    

<span data-ttu-id="43c99-104">[!iekļaut [reklāmkarogu](../includes/banner.md)</span><span class="sxs-lookup"><span data-stu-id="43c99-104">[!include [banner](../includes/banner.md)</span></span>


<span data-ttu-id="43c99-105">Jūs varat izveidot pakalpojuma abonementu no projekta sadaļā **Projektu pārvaldība un uzskaite**.</span><span class="sxs-lookup"><span data-stu-id="43c99-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="43c99-106">Noklikšķiniet uz **Projektu pārvaldība un uzskaite** \> **Vispārīgi** \> **Projekti** \> **Visi projekti**.</span><span class="sxs-lookup"><span data-stu-id="43c99-106">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="43c99-107">Atveriet projektu, kuram vēlaties izveidot abonementu.</span><span class="sxs-lookup"><span data-stu-id="43c99-107">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="43c99-108">Veidlapas **Projekti** **Darbību rūts** cilnē **Pārvaldība** noklikšķiniet uz **Pakalpojums** \> **Pakalpojuma abonements**.</span><span class="sxs-lookup"><span data-stu-id="43c99-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="43c99-109">Veidlapas **Abonements** laukā **Pakalpojuma abonements** ievadiet unikālu identifikācijas numuru.</span><span class="sxs-lookup"><span data-stu-id="43c99-109">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="43c99-110">Atlasītajam projektam abonementā tiek automātiski ievadīts projekta numurs.</span><span class="sxs-lookup"><span data-stu-id="43c99-110">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="43c99-111">Ievadiet pārējo informāciju formā.</span><span class="sxs-lookup"><span data-stu-id="43c99-111">Enter the remaining information in the form.</span></span>

<span data-ttu-id="43c99-112">Tagad varat izveidot abonementa darbības.</span><span class="sxs-lookup"><span data-stu-id="43c99-112">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="43c99-113">Ja abonementu grupai esat iestatījis numuru sēriju, norādot abonementu grupu, abonementam tiek automātiski ģenerēts abonementa ID.</span><span class="sxs-lookup"><span data-stu-id="43c99-113">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
