# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="855c4-103">Pakalpojuma abonementu izveide no projekta</span><span class="sxs-lookup"><span data-stu-id="855c4-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="855c4-104">Jūs varat izveidot pakalpojuma abonementu no projekta sadaļā **Projektu pārvaldība un uzskaite**.</span><span class="sxs-lookup"><span data-stu-id="855c4-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="855c4-105">Noklikšķiniet uz **Projektu pārvaldība un uzskaite** \> **Vispārīgi** \> **Projekti** \> **Visi projekti**.</span><span class="sxs-lookup"><span data-stu-id="855c4-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="855c4-106">Atveriet projektu, kuram vēlaties izveidot abonementu.</span><span class="sxs-lookup"><span data-stu-id="855c4-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="855c4-107">Veidlapas **Projekti** **Darbību rūts** cilnē **Pārvaldība** noklikšķiniet uz **Pakalpojums** \> **Pakalpojuma abonements**.</span><span class="sxs-lookup"><span data-stu-id="855c4-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="855c4-108">Veidlapas **Abonements** laukā **Pakalpojuma abonements** ievadiet unikālu identifikācijas numuru.</span><span class="sxs-lookup"><span data-stu-id="855c4-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="855c4-109">Atlasītajam projektam abonementā tiek automātiski ievadīts projekta numurs.</span><span class="sxs-lookup"><span data-stu-id="855c4-109">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="855c4-110">Ievadiet pārējo informāciju formā.</span><span class="sxs-lookup"><span data-stu-id="855c4-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="855c4-111">Tagad varat izveidot abonementa darbības.</span><span class="sxs-lookup"><span data-stu-id="855c4-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="855c4-112">Ja abonementu grupai esat iestatījis numuru sēriju, norādot abonementu grupu, abonementam tiek automātiski ģenerēts abonementa ID.</span><span class="sxs-lookup"><span data-stu-id="855c4-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
