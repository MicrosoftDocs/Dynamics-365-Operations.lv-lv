# <a name="set-up-subscription-groups"></a><span data-ttu-id="23e6e-103">Abonementu grupu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="23e6e-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="23e6e-104">Noklikšķiniet uz **Pakalpojumu pārvaldība** \> **Iestatīšana** \> **Pakalpojumu abonementi** \> **Abonementu grupas**.</span><span class="sxs-lookup"><span data-stu-id="23e6e-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="23e6e-105">Nospiediet taustiņu kombināciju Ctrl + N vai noklikšķiniet uz **Jauns**, lai izveidotu jaunu abonementu grupu.</span><span class="sxs-lookup"><span data-stu-id="23e6e-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="23e6e-106">Laukā **Abonementu grupa** ievadiet jaunās abonementu grupas nosaukumu.</span><span class="sxs-lookup"><span data-stu-id="23e6e-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="23e6e-107">Laukā **Rēķinu izrakstīšana** atlasiet šim abonementam rēķinu izrakstīšanas biežumu.</span><span class="sxs-lookup"><span data-stu-id="23e6e-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="23e6e-108">Rēķinu izrakstīšanas biežums tiek norādīts ar perioda kodu, kas ir iestatīts formā **Periodu veidi**.</span><span class="sxs-lookup"><span data-stu-id="23e6e-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="23e6e-109">Lai uzkrātu ieņēmumus par abonementu transakcijām, atzīmējiet izvēles rūtiņu **Uzkrāt ieņēmumus** un pēc tam laukā **Uzkrājums** atlasiet biežumu.</span><span class="sxs-lookup"><span data-stu-id="23e6e-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="23e6e-110">Uzkrāšanas biežums ir balstīts uz perioda kodu, kas ir iestatīts formā **Periodi**.</span><span class="sxs-lookup"><span data-stu-id="23e6e-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
