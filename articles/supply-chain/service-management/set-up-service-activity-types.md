# <a name="set-up-service-activity-types"></a>Pakalpojumu aktivitāšu tipu iestatīšana 

[!include [banner](../includes/banner.md)]


1.  Noklikšķiniet uz **Pārdošana un mārketings** \> **Iestatīšana** \> **Aktivitātes** \> **Aktivitāšu veidi**.
    
    –vai–
    
    Noklikšķiniet uz **Organizācijas administrēšana** \> **Iestatīšana** \> **Aktivitātes** \> **Aktivitāšu veidi**.

2.  Formā **Aktivitāšu veidi** nospiediet taustiņu kombināciju CTRL + N, lai pievienotu jaunu aktivitāšu veida rindu.

3.  Laukā **Veids** ievadiet aktivitātes veida nosaukumu.

4.  Laukā **Apraksts** ievadiet tekstu, kas apraksta šo aktivitāšu veidu.

5.  Aizveriet formu, lai saglabātu izmaiņas.
