[!include [banner](../includes/banner.md)]


<span data-ttu-id="d1fb0-104">Lietojiet veidlapu **Atgriešanas pasūtījuma dzēšana**, lai dzēstu vairākus atgriešanas pasūtījumus.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="d1fb0-105">Noklikšķiniet uz **Pārdošana un mārketings** \> **Periodiskās darbības** \> **Iztīrīt** \> **Dzēst atgriešanas pasūtījumus**.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

2.  <span data-ttu-id="d1fb0-106">Noklikšķiniet uz **Atlasīt**, lai atvērtu veidlapu **Pārdošanas datu atjaunināšana**.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-106">Click **Select** to open the **Sales update** form.</span></span>

3.  <span data-ttu-id="d1fb0-107">Norādiet kritērijus un noklikšķiniet uz **Labi**, lai atgrieztos veidlapā **Atgriešanas pasūtījuma dzēšana**.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-107">Specify the criteria, and then click **OK** to return to the **Delete return order** form.</span></span> <span data-ttu-id="d1fb0-108">Atgriešanas pasūtījumiem, kas izveidoti juridiskajai personai, tiek parādīti, balstoties uz norādītajiem kritērijiem.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-108">Return orders that have been created in the legal entity are displayed, based on the specified criteria.</span></span>

5.  <span data-ttu-id="d1fb0-110">Noklikšķiniet uz **Jā**, lai dzēstu preču atgriešanas pasūtījumus.</span><span class="sxs-lookup"><span data-stu-id="d1fb0-110">Click **Yes** to delete the return orders.</span></span>
