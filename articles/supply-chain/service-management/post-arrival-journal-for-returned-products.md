# <a name="post-arrival-journal-for-returned-products"></a><span data-ttu-id="42e5b-103">Grāmatot atgrieztu preču saņemšanas žurnālu</span><span class="sxs-lookup"><span data-stu-id="42e5b-103">Post arrival journal for returned products</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="42e5b-104">Lai apstrādātu atgriešanu, vispirms pārbaudiet atgriezto daudzumu un atjauniniet daudzuma lauku krājumu saņemšanas žurnālā.</span><span class="sxs-lookup"><span data-stu-id="42e5b-104">To process a return, first validate the return quantity, update the quantity field in the item arrival journal.</span></span> <span data-ttu-id="42e5b-105">Pēc tam atlasiet atgriešanas metodes kodu vai norādiet, ka atgrieztās preces ir jāpārbauda.</span><span class="sxs-lookup"><span data-stu-id="42e5b-105">Then select a disposition code or indicate that the returned items have to be inspected.</span></span> <span data-ttu-id="42e5b-106">Pēc šo darbību veikšanas jūs varat iegrāmatot krājumu saņemšanas žurnālā atgriešanas pasūtījumu.</span><span class="sxs-lookup"><span data-stu-id="42e5b-106">After completing these steps, you can post the item arrival journal for the return order.</span></span>

1.  <span data-ttu-id="42e5b-107">Noklikšķiniet uz **Krājumu pārvaldība** \> **Periodiskās darbības** \> **Saņemšanas pārskats**.</span><span class="sxs-lookup"><span data-stu-id="42e5b-107">Click **Inventory management** \> **Periodic** \> **Arrival overview**.</span></span>

2.  <span data-ttu-id="42e5b-108">Filtrā **Iestatījumu nosaukums** atlasiet vienumu **Atgriešanas pasūtījums**.</span><span class="sxs-lookup"><span data-stu-id="42e5b-108">In the **Setup name** filter, select **Return order**.</span></span>

3.  <span data-ttu-id="42e5b-109">Ja saņemšanas saraksts ir garš, izmantojiet zonas **Diapazons** laukus, lai sarakstu saīsinātu.</span><span class="sxs-lookup"><span data-stu-id="42e5b-109">If the list of receipts is long, use the fields in the **Range** area to narrow the list.</span></span>

4.  <span data-ttu-id="42e5b-110">Atrodiet atgriešanas pasūtījuma rindu, ko vēlaties grāmatot, atlasiet rūtiņu **Atlasīt saņemšanai** un noklikšķiniet uz **Sākt saņemšanu**.</span><span class="sxs-lookup"><span data-stu-id="42e5b-110">Locate the line of the return order that you want to post, select its **Select for arrival** box, and then click **Start arrival**.</span></span>

5.  <span data-ttu-id="42e5b-111">Noklikšķiniet uz **Žurnāli** \> **Rādīt saņemšanas no ieejas plūsmām**, lai atvērtu veidlapu **Novietojumu žurnāls**.</span><span class="sxs-lookup"><span data-stu-id="42e5b-111">Click **Journals** \> **Show arrivals from receipts** to open the **Location journal** form.</span></span>
    

    > [!TIP]
    > <P><span data-ttu-id="42e5b-112">Lai apskatītu detalizētu informāciju, atlasiet žurnālu un pēc tam noklikšķiniet uz <STRONG>Rindas</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="42e5b-112">To view detailed information, select a journal, and then click <STRONG>Lines</STRONG>.</span></span></P>


6.  <span data-ttu-id="42e5b-113">Veiciet nepieciešamos atjauninājumus un pēc tam noklikšķiniet uz **Grāmatot**.</span><span class="sxs-lookup"><span data-stu-id="42e5b-113">Make any necessary updates, and then click **Post**.</span></span>

<span data-ttu-id="42e5b-114">Kad žurnāls ir iegrāmatots, atgrieztie krājumi tiek reģistrēti krājumos un veidlapa **Atgriešanas pasūtījumi** norāda, ka krājumi ir nonākuši noliktavā.</span><span class="sxs-lookup"><span data-stu-id="42e5b-114">After the journal is posted, the returned items are registered in inventory, and the **Return orders** form indicates that the items have arrived at the warehouse.</span></span>
