## <a name="credit-subscription-transactions"></a><span data-ttu-id="db420-104">Abonementa transakciju kreditēšana</span><span class="sxs-lookup"><span data-stu-id="db420-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="db420-105">Noklikšķiniet uz **Pakalpojumu pārvaldība** \> **Vispārīgi** \> **Pakalpojumu abonementi** \> **Visi pakalpojumu abonementi**.</span><span class="sxs-lookup"><span data-stu-id="db420-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="db420-106">Atlasiet abonementa darījumam pievienoto abonementu, kuram vēlaties izveidot kredīta notu.</span><span class="sxs-lookup"><span data-stu-id="db420-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="db420-107">Atlasiet cilni **Analizēšana** un pēc tam noklikšķiniet uz pogas **Apmaksas transakcijas** Darbību rūtī.</span><span class="sxs-lookup"><span data-stu-id="db420-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="db420-108">Veidlapā **Apmaksas transakcijas** atlasiet transakciju, kurai vēlaties izveidot kredīta notu.</span><span class="sxs-lookup"><span data-stu-id="db420-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="db420-109">Noklikšķiniet uz **Funkcijas** \> **Atlasīt kredīta notai**.</span><span class="sxs-lookup"><span data-stu-id="db420-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="db420-110">Veidlapā **Atlasīt kredīta notai** atlasiet transakciju, kuru vēlaties kreditēt, un pēc tam noklikšķiniet uz **Labi**.</span><span class="sxs-lookup"><span data-stu-id="db420-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="db420-111">Kad tiek veidota kredīta nota, pārliecinieties, ka ir atlasīts vienums <STRONG>Kredīta notas</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="db420-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="db420-112">Tas ir atrodams dialoglodziņa <STRONG>Rēķina izveide</STRONG> sarakstā <STRONG>Rēķina metode</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="db420-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="db420-113">Ja lauks **Kreditēšanas uzkrājumu atgriešana** veidlapā **Pakalpojumu pārvaldības parametri** ir iestatīts uz **Manuāli**, katra uzkrāto ieņēmumu transakcija ir jāatgriež atsevišķi pirms transakcijai tiek izveidots kredīta notas piedāvājums.</span><span class="sxs-lookup"><span data-stu-id="db420-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
