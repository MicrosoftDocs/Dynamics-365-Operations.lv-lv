[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="96ab0-104">Šajā procedūrā ir parādīts, kā skatīt izmaksu ierakstus izmaksu objektam.</span><span class="sxs-lookup"><span data-stu-id="96ab0-104">This procedure shows how to view cost entries for a cost object.</span></span> <span data-ttu-id="96ab0-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="96ab0-105">The demo data company used to create this procedure is USMF.</span></span> <span data-ttu-id="96ab0-106">Šī procedūra ir paredzēta izmaksu kontrolierim.</span><span class="sxs-lookup"><span data-stu-id="96ab0-106">This procedure is intended for the cost controller.</span></span>

1. <span data-ttu-id="96ab0-107">Noklikšķiniet uz Izmaksu administrēšana.</span><span class="sxs-lookup"><span data-stu-id="96ab0-107">Click Cost administration.</span></span>

3. <span data-ttu-id="96ab0-109">Izmantojiet līdzekli Ātrais filtrs, lai atrastu ierakstus.</span><span class="sxs-lookup"><span data-stu-id="96ab0-109">Use the Quick Filter to find records.</span></span> <span data-ttu-id="96ab0-110">Piemēram, filtrējiet pēc lauka Krājuma numurs, izmantojot vērtību "m0004".</span><span class="sxs-lookup"><span data-stu-id="96ab0-110">For example, filter on the Item number field with a value of 'm0004'.</span></span>

5. <span data-ttu-id="96ab0-112">Noklikšķiniet uz Izmaksu objekti.</span><span class="sxs-lookup"><span data-stu-id="96ab0-112">Click Cost objects.</span></span>
6. <span data-ttu-id="96ab0-113">Noklikšķiniet uz Izmaksu ieraksti.</span><span class="sxs-lookup"><span data-stu-id="96ab0-113">Click Cost entries.</span></span>

    * <span data-ttu-id="96ab0-115">Ja izmaksu ieraksti ir tukši, vienumu No datuma iestatiet uz 2012. gada 31. janvāri un vienumu Līdz datumam iestatiet uz 2012. gada 31. decembri.</span><span class="sxs-lookup"><span data-stu-id="96ab0-115">If cost entries are blank, set From date to January 31, 2012 and To date to December 31, 2012.</span></span>
