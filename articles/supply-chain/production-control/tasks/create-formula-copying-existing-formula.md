# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Formulas izveide, kopējot no esošas formulas

[!include [banner](../../includes/banner.md)]

Ar šo procedūru tiek parādīts, kā ātri izveidot jaunu formulu, kopējot no esošas formulas. Lai izveidotu šo procedūru, varat izmantot demonstrācijas datu uzņēmumus USMF, USP2 vai USPI.


## <a name="create-a-new-formula"></a>Jaunas formulas izveide
1. Dodieties uz Formulas.
2. Noklikšķiniet uz Jauns.
3. Laukā Formula ierakstiet vērtību.
4. Laukā Nosaukums ierakstiet kādu vērtību.
5. Laukā Vieta ievadiet vai atlasiet kādu vērtību.
6. Laukā Krājumu grupa ievadiet vai atlasiet kādu vērtību.
7. Noklikšķiniet uz Saglabāt.

## <a name="copy-from-an-existing-formula"></a>Esošas formulas kopēšana
1. Noklikšķiniet uz Kopēt.
2. Laukā Krājuma kods ievadiet vai atlasiet kādu vērtību.
3. Laukā Formulas versija ievadiet vai atlasiet kādu vērtību.
4. Noklikšķiniet uz OK.

## <a name="revise-the-copied-formula"></a>Kopētās formulas pārskatīšana
1. Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.
2. Noklikšķiniet uz Dzēst.
3. Noklikšķiniet uz Jā.

## <a name="approve-formula"></a>Apstiprināt formulu
1. Noklikšķiniet uz Apstiprināt formulu.
2. Laukā Apstiprināja ievadiet vai atlasiet kādu vērtību.
3. Noklikšķiniet uz OK.
4. Aizvērt lapu.
