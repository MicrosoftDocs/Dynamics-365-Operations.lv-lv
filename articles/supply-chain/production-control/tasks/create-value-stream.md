# <a name="create-a-value-stream"></a>Vērtību plūsmas izveide

[!include [banner](../../includes/banner.md)]

Vērtību plūsmas ir pārvaldības struktūrvienības, kas sagrupētas vienā vai vairākās ražošanas plūsmās. Lai definētu ražošanas plūsmu, vispirms ir jādefinē vērtību plūsma.


## <a name="open-the-form"></a>Atvērt veidlapu
1. Pārejiet uz sadaļu Ražošanas kontrole > Iestatījumi > Lean ražošanas plūsma > Vērtību plūsmas.

## <a name="create-a-new-value-stream"></a>Jaunas vērtību plūsmas izveide
1. Noklikšķiniet uz Jauns, lai atvērtu nolaižamo dialoglodziņu.
2. Laukā Nosaukums ierakstiet kādu vērtību.

## <a name="select-a-value-stream-manager"></a>Vērtību plūsmas pārvaldnieka atlase
1. Laukā Vadītājs noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu uzmeklēšanu.
    * Vērtību plūsmas pārvaldnieks ir definēts, lai dokumentētu organizāciju.  
2. Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.
3. Sarakstā noklikšķiniet uz saites atlasītajā rindā.

## <a name="save-and-close"></a>Saglabāt un aizvērt
1. Noklikšķiniet uz Saglabāt.
2. Aizvērt lapu.
3. Atsvaidziniet lapu.
