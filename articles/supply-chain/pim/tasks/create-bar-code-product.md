# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="3f9ec-103">Preces svītrkoda izveide</span><span class="sxs-lookup"><span data-stu-id="3f9ec-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="3f9ec-104">Šajā tēmā parādīts, kā manuāli izveidot svītrkodu, kā piemēru izmantojot krājuma kodu M0001.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="3f9ec-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="3f9ec-106">Sākumlapā atlasiet **Izlaisto preču uzturēšana**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="3f9ec-107">Dodieties uz **Preces > Izlaistās preces** sadaļā **Saites**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="3f9ec-108">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="3f9ec-109">Šim piemēram atlasiet krājuma kodu **M0001**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="3f9ec-110">Darbību rūtī atlasiet **Pārvaldīt krājumus**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="3f9ec-111">Atlasiet **Svītrkodi**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="3f9ec-112">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-112">Select **New**.</span></span>
7. <span data-ttu-id="3f9ec-113">Atzīmējiet atlasīto rindu, kas izveidota tālāk dotajā sarakstā.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="3f9ec-114">Laukā **Svītrkodu iestatījumi** ievadiet vai atlasiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="3f9ec-115">Laukā **Svītrkods** ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="3f9ec-116">Laukā **Svītrkods** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="3f9ec-117">Laukā **Daudzums** ierakstiet kādu skaitli.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="3f9ec-118">Atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-118">Select **Save**.</span></span>
13. <span data-ttu-id="3f9ec-119">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="3f9ec-119">Close the page.</span></span>
