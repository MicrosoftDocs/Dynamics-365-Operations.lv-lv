[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7bfb8-104">Šajā tēmā parādīts, kā manuāli izveidot svītrkodu, kā piemēru izmantojot krājuma kodu M0001.</span><span class="sxs-lookup"><span data-stu-id="7bfb8-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="7bfb8-105">USMF ir paraugdatu uzņēmums, kas tiek izmantots šīs procedūras izveidei.</span><span class="sxs-lookup"><span data-stu-id="7bfb8-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="7bfb8-106">Pārejiet uz sadaļu **Preču informācijas pārvaldība \> Preces \> Izlaistās preces**.</span><span class="sxs-lookup"><span data-stu-id="7bfb8-106">Go to **Product information management \> Products \> Released products**.</span></span>

1. <span data-ttu-id="7bfb8-118">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="7bfb8-118">Close the page.</span></span>
