# <a name="nonconformance-management"></a><span data-ttu-id="98bc1-104">Neatbilstības pārvaldība</span><span class="sxs-lookup"><span data-stu-id="98bc1-104">Nonconformance management</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="98bc1-105">Šajā rakstā ir aprakstīts pamata iestatīšana, kas ir nepieciešama, lai lietotu neatbilstības.</span><span class="sxs-lookup"><span data-stu-id="98bc1-105">This article describes the basic setup that is required in order to use nonconformances.</span></span> <span data-ttu-id="98bc1-106">Papildu iestatīšana ir nepieciešama, ja vēlaties izmantot kvalitātes pārbaudes pasūtījumus.</span><span class="sxs-lookup"><span data-stu-id="98bc1-106">Additional setup is required if you want to use quality orders.</span></span>

<span data-ttu-id="98bc1-107">Lai iespējotu neatbilstības pārvaldību, izpildiet tālāk aprakstītās darbības.</span><span class="sxs-lookup"><span data-stu-id="98bc1-107">To enable nonconformance management, follow these steps:</span></span>

1.  <span data-ttu-id="98bc1-108">Definējiet krājuma un noliktavas pārvaldības parametrus, kas ir saistīti ar neatbilstībām.</span><span class="sxs-lookup"><span data-stu-id="98bc1-108">Define the Inventory and warehouse management parameters that are related to nonconformances:</span></span>
    -   <span data-ttu-id="98bc1-109">Opcijai **Izmantot kvalitātes pārvaldību** iestatiet **Jā**.</span><span class="sxs-lookup"><span data-stu-id="98bc1-109">Set the **Use quality management** option to **Yes**.</span></span>
    -   <span data-ttu-id="98bc1-110">Laukā **Stundas likme** ievadiet darbaspēka stundas likmi vietējā valūtā.</span><span class="sxs-lookup"><span data-stu-id="98bc1-110">In the **Hourly rate** field, enter an hourly labor rate in the local currency.</span></span> <span data-ttu-id="98bc1-111">Stundas likme tiek lietota, lai aprēķinātu izmaksas operācijām, kas ir saistītas ar neatbilstību.</span><span class="sxs-lookup"><span data-stu-id="98bc1-111">The hourly rate is used to calculate costs for operations that are related to a nonconformance.</span></span> <span data-ttu-id="98bc1-112">Stundas likme un aprēķinātās izmaksas nodrošina atsauces informāciju par neatbilstību.</span><span class="sxs-lookup"><span data-stu-id="98bc1-112">The hourly rate and calculated costs provide reference information for a nonconformance.</span></span> <span data-ttu-id="98bc1-113">Šie vienumi nemijiedarbojas ar citām funkcijām.</span><span class="sxs-lookup"><span data-stu-id="98bc1-113">They don't interact with other functionality.</span></span>
    -   <span data-ttu-id="98bc1-114">Lai definētu drukājamā dokumenta tipu, izmantojiet cilni **Kvalitātes pārvaldība** lapā **Pārskatu iestatīšana**.</span><span class="sxs-lookup"><span data-stu-id="98bc1-114">Use the **Quality management** tab on the **Report setup** page to define the type of document to print.</span></span> <span data-ttu-id="98bc1-115">Varat drukāt neatbilstības pārskatu, neatbilstības atzīmi vai labojumu pārskatu.</span><span class="sxs-lookup"><span data-stu-id="98bc1-115">You can print a nonconformance report, a nonconformance tag, or a correction report.</span></span> <span data-ttu-id="98bc1-116">Dažādu dokumenta tipu iedrukāšanai pārskatā var definēta vairāk nekā vienu ierakstu, vai iekšēju vai ārēju paziņojumu drukāšanai.</span><span class="sxs-lookup"><span data-stu-id="98bc1-116">You can define more than one record to print different document types on a report, or to print internal and external notes.</span></span> <span data-ttu-id="98bc1-117">Var būt lietderīgi izmantot lapu **Dokumenta veids**, lai neatbilstībām un labojumiem definētu unikālus dokumentu veidus.</span><span class="sxs-lookup"><span data-stu-id="98bc1-117">You might find it helpful to use the **Document type** page to define a unique document type for nonconformances and a unique document type for corrections.</span></span> <span data-ttu-id="98bc1-118">Piemēram, ievadiet piezīmes par neatbilstību, izmantojot unikālu neatbilstību dokumenta veidu.</span><span class="sxs-lookup"><span data-stu-id="98bc1-118">For example, want to enter notes about a nonconformance by using the unique document type for nonconformances.</span></span> <span data-ttu-id="98bc1-119">Šajā gadījumā unikālo dokumenta veidu norādiet pārskata opcijās.</span><span class="sxs-lookup"><span data-stu-id="98bc1-119">In this case, identify the unique document type in the report options.</span></span>
    -   <span data-ttu-id="98bc1-120">Iespējojiet neatbilstības un labošanas atsauču numuru sērijas.</span><span class="sxs-lookup"><span data-stu-id="98bc1-120">Enable number sequences for nonconformance and correction references.</span></span>

2.  <span data-ttu-id="98bc1-121">Iespējojiet neatbilstībām lietotāja apstiprināšanu.</span><span class="sxs-lookup"><span data-stu-id="98bc1-121">Enable user approval of nonconformances.</span></span> <span data-ttu-id="98bc1-122">Izmantojiet lapas **Lietotāji** lauku **Nosaukums**, lai piešķirtu darbinieku katram lietotājam, kuram ir jāapstiprina neatbilstība.</span><span class="sxs-lookup"><span data-stu-id="98bc1-122">Use the **Name** field on the **Users** page to assign an employee to each user who must approve a nonconformance.</span></span> <span data-ttu-id="98bc1-123">Sistēma izmanto darbiniekus, kuri maina neatbilstības statusu, lai izsekotu neatbilstības vēsturi.</span><span class="sxs-lookup"><span data-stu-id="98bc1-123">The system uses the employees who change the status of a noncomformance to track the nonconformance history.</span></span> <span data-ttu-id="98bc1-124">Lietotāji var apstiprināt neatbilstību tikai tad, ja viņiem ir piešķirts darbinieka identifikators.</span><span class="sxs-lookup"><span data-stu-id="98bc1-124">Users can't approve a nonconformance unless they have been assigned an employee identifier.</span></span>
3.  <span data-ttu-id="98bc1-125">Definējiet problēmu veidus, kas tiks piešķirti neatbilstībām.</span><span class="sxs-lookup"><span data-stu-id="98bc1-125">Define the problem types that will be assigned to nonconformances.</span></span> <span data-ttu-id="98bc1-126">Izmantojiet lapu **Problēmtipi**, lai definētu to kvalitātes problēmu klasifikāciju, kas rodas dažādiem neatbilstības veidiem.</span><span class="sxs-lookup"><span data-stu-id="98bc1-126">Use the **Problem types** page to define a classification of quality problems that are encountered for the various nonconformance types.</span></span> <span data-ttu-id="98bc1-127">Varar iestatīt šādus neatbilstības veidus: **Iekšējais**, **Debitora**, **Kreditora**, **Pakalpojuma pieprasījuma**, **Ražošanas** un **Līdzprodukta ražošanas**.</span><span class="sxs-lookup"><span data-stu-id="98bc1-127">You can set up the following nonconformance types: **Internal**, **Customer**, **Vendor**, **Service request**, **Production**, and **Co-product production**.</span></span> <span data-ttu-id="98bc1-128">Izmantojiet lapu **Neatbilstības tipi**, lai autorizētu problēmas veidu, kas jāizmanto vienam vai vairākiem neatbilstības veidiem.</span><span class="sxs-lookup"><span data-stu-id="98bc1-128">Use the **Non conformance types** page to authorize a problem type to be used in one or more nonconformance types.</span></span> <span data-ttu-id="98bc1-129">Piemēram, problēmas veids, kas attiecas uz defekta kodu, var attiekties uz visiem neatbilstības veidiem, turpretī problēmas veids, kas attiecas uz debitora sūdzībām, var attiekties tikai uz neatbilstības veidu **Debitors** un **Pakalpojuma pieprasījums**.</span><span class="sxs-lookup"><span data-stu-id="98bc1-129">For example, a problem type that is related to a defect code might apply to all nonconformance types, whereas a problem type that is related to customer complaints might apply only to the **Customer** and **Service request** nonconformance types.</span></span>
4.  <span data-ttu-id="98bc1-130">Definējiet karantīnas zonas, lai nodrošinātu norādījumus par rīcību ar defektīviem materiāliem.</span><span class="sxs-lookup"><span data-stu-id="98bc1-130">Define quarantine zones to provide guidance about defective material should be handled.</span></span> <span data-ttu-id="98bc1-131">Izmantojiet lapu **Karantīnas zonas**, lai definētu zonas, ko var piešķirt neatbilstībai.</span><span class="sxs-lookup"><span data-stu-id="98bc1-131">Use the **Quarantine zones** page to define zones that can be assigned to a nonconformance.</span></span> <span data-ttu-id="98bc1-132">Izdrukāta neatbilstības atzīme parādīs piešķirto karantīnas zonu un informāciju par lietojumu, lai sniegtu norādījumus par defektīvu materiālu apstrādi.</span><span class="sxs-lookup"><span data-stu-id="98bc1-132">The printed nonconformance tag will show the assigned quarantine zone and information about usage, to provide guidance about how to handle defective material.</span></span> <span data-ttu-id="98bc1-133">Zonas var atbilst krājumu vietām vai operāciju resursiem.</span><span class="sxs-lookup"><span data-stu-id="98bc1-133">The zones might correspond to inventory locations or operations resources.</span></span>
5.  <span data-ttu-id="98bc1-134">Definējiet diagnostikas veidus, kas tiks piešķirti labojumiem.</span><span class="sxs-lookup"><span data-stu-id="98bc1-134">Define the diagnostic types that will be assigned to corrections.</span></span> <span data-ttu-id="98bc1-135">Izmantojiet lapu **Diagnostikas tipi**, lai definētu diagnostikas darbību klasifikāciju.</span><span class="sxs-lookup"><span data-stu-id="98bc1-135">Use the **Diagnostic types** page to define a classification of diagnostic actions.</span></span> <span data-ttu-id="98bc1-136">Labojumā norāda, kāda veida diagnostikas darbība jāveic attiecībā uz apstiprināto neatbilstību un kam tā ir jāveic.</span><span class="sxs-lookup"><span data-stu-id="98bc1-136">A correction specifies what type of diagnostic action should be taken for an approved nonconformance, and who should take that action.</span></span> <span data-ttu-id="98bc1-137">Tajā norāda arī pieprasīto un plānotais pabeigšanas datumu.</span><span class="sxs-lookup"><span data-stu-id="98bc1-137">It also specifies the requested completion date and the planned completion date.</span></span>
6.  <span data-ttu-id="98bc1-138">Definējiet saistītās operācijas, kas tiks piešķirtas neatbilstībām.</span><span class="sxs-lookup"><span data-stu-id="98bc1-138">Define the related operations that will be assigned to nonconformances.</span></span> <span data-ttu-id="98bc1-139">Izmantojiet lapu **Operācijas**, lai definētu klasifikāciju darbam, ko var veikt apstiprinātajai neatbilstībai.</span><span class="sxs-lookup"><span data-stu-id="98bc1-139">Use the **Operations** page to define a classification of the work that can be performed for an approved nonconformance.</span></span> <span data-ttu-id="98bc1-140">Ja neatbilstībai piešķirat saistīto operāciju, varat nodrošināt detalizētu informāciju, piemēram, par saistīto materiālu, darba stundām un dažādām papildmaksām, kas nepieciešamas operācijas izpildē.</span><span class="sxs-lookup"><span data-stu-id="98bc1-140">When you assign a related operation to a nonconformance, you can provide detailed information, such as information about the associated material, labor hours, and miscellaneous charges that are required in order to perform the operation.</span></span> <span data-ttu-id="98bc1-141">Šī informācija tiek izmantota, lai operācijai aprēķinātu novērtētās izmaksas.</span><span class="sxs-lookup"><span data-stu-id="98bc1-141">This information is used to calculate an estimated cost for the operation.</span></span> <span data-ttu-id="98bc1-142">Detalizētā informācija un novērtētās izmaksas tiek nodrošinātas tikai atsauces nolūkā.</span><span class="sxs-lookup"><span data-stu-id="98bc1-142">The detailed information and estimated costs are for reference.</span></span> <span data-ttu-id="98bc1-143">Kvalitātes saistītās operācijas ir atšķirīgas no operācijām, ko var definēt ražošanas maršrutā.</span><span class="sxs-lookup"><span data-stu-id="98bc1-143">The related operations for quality differ from the operations that can be defined for a production route.</span></span>
