11. <span data-ttu-id="253a4-117">Klikšķiniet Iesniegt.</span><span class="sxs-lookup"><span data-stu-id="253a4-117">Click Submit.</span></span>
12. <span data-ttu-id="253a4-118">Laukā Komentārs ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="253a4-118">In the Comment field, type a value.</span></span>
13. <span data-ttu-id="253a4-119">Klikšķiniet Iesniegt.</span><span class="sxs-lookup"><span data-stu-id="253a4-119">Click Submit.</span></span>
