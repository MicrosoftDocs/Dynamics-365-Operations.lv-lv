<span data-ttu-id="46830-106">Šajā uzdevumā tiek izmantota SSI datu kopa.</span><span class="sxs-lookup"><span data-stu-id="46830-106">This task uses the SSI data set.</span></span>

11. <span data-ttu-id="46830-117">Klikšķiniet Iesniegt.</span><span class="sxs-lookup"><span data-stu-id="46830-117">Click Submit.</span></span>
12. <span data-ttu-id="46830-118">Laukā Komentārs ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="46830-118">In the Comment field, type a value.</span></span>
13. <span data-ttu-id="46830-119">Klikšķiniet Iesniegt.</span><span class="sxs-lookup"><span data-stu-id="46830-119">Click Submit.</span></span>
