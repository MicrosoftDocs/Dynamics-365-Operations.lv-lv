[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8ce93-104">Šajā procedūrā ir parādīts, kā izveidot projekta pirkšanas pasūtījumu.</span><span class="sxs-lookup"><span data-stu-id="8ce93-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="8ce93-105">Šajā uzdevumā tiek izmantota USSI datu kopa.</span><span class="sxs-lookup"><span data-stu-id="8ce93-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="8ce93-106">Dodieties uz Projektu vadība un uzskaite > Projekti > Visi projekti.</span><span class="sxs-lookup"><span data-stu-id="8ce93-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="8ce93-107">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="8ce93-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="8ce93-108">Darbību rūtī noklikšķiniet uz Pārvaldīt.</span><span class="sxs-lookup"><span data-stu-id="8ce93-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="8ce93-109">Noklikšķiniet uz Krājumu uzdevums.</span><span class="sxs-lookup"><span data-stu-id="8ce93-109">Click Item task.</span></span>
5. <span data-ttu-id="8ce93-110">Noklikšķiniet uz Pirkšanas pasūtījums.</span><span class="sxs-lookup"><span data-stu-id="8ce93-110">Click Purchase order.</span></span>
6. <span data-ttu-id="8ce93-111">Ievadiet vai atlasiet vērtību laukā kreditora konts.</span><span class="sxs-lookup"><span data-stu-id="8ce93-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="8ce93-112">Laukā Vieta ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="8ce93-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="8ce93-113">Šīs darbības nav obligātas, bet tās vienkāršo pirkšanas pasūtījumu, pirkšanas pasūtījuma rindām iestatot noklusējuma vietu un noliktavu.</span><span class="sxs-lookup"><span data-stu-id="8ce93-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="8ce93-114">Laukā Noliktava ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="8ce93-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="8ce93-115">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="8ce93-115">Click OK.</span></span>
10. <span data-ttu-id="8ce93-116">Sarakstā atzīmējiet atlasīto rindu.</span><span class="sxs-lookup"><span data-stu-id="8ce93-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="8ce93-117">Laukā Krājuma kods ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="8ce93-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="8ce93-118">Tas var būt krājuma kods vai sagādes kategorija.</span><span class="sxs-lookup"><span data-stu-id="8ce93-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="8ce93-119">Izvērsiet sadaļu Detalizēta informācija par rindu.</span><span class="sxs-lookup"><span data-stu-id="8ce93-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="8ce93-120">Noklikšķiniet uz cilnes Projekts.</span><span class="sxs-lookup"><span data-stu-id="8ce93-120">Click the Project tab.</span></span>
    * <span data-ttu-id="8ce93-121">Pārbaudiet, vai ir pieejamas pārdošanas un izmaksu cenas.</span><span class="sxs-lookup"><span data-stu-id="8ce93-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="8ce93-122">Ja tās nav pieejamas, bet ir nepieciešamas, ievadiet informāciju.</span><span class="sxs-lookup"><span data-stu-id="8ce93-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="8ce93-123">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="8ce93-123">Click Save.</span></span>
