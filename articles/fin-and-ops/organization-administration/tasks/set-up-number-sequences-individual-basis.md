<span data-ttu-id="a4d5e-104">Šajā tēmā ir paskaidrots, kā iestatīt atsevišķu numuru sēriju.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-104">This topic explains how to set up number sequences on an individual basis.</span></span> <span data-ttu-id="a4d5e-105">Numuru sērijas tiek izmantotas, lai ģenerētu lasāmus, unikālus pamatdatu ierakstu identifikatorus un darījumu ierakstus, kam tie ir nepieciešami.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-105">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require them.</span></span> <span data-ttu-id="a4d5e-106">Pamatdati vai darījumu ieraksts, kam nepieciešams identifikators, tiek saukts par atsauci.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-106">A master data or transaction record that requires an identifier is referred to as a reference.</span></span> <span data-ttu-id="a4d5e-107">Pirms atsaucei varat izveidot jaunus ierakstus, ir jāiestata numuru sērija un tā jāsaista ar atsauci.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-107">Before you can create new records for a reference, you must set up a number sequence and associate it with the reference.</span></span> <span data-ttu-id="a4d5e-108">Varat iestatīt visas nepieciešamās numuru sērijas vienlaicīgi, izmantojot vedni **Numuru sēriju iestatīšana** vai arī varat izveidot vai modificēt atsevišķas numuru sērijas, izmantojot lapu **Numuru sērijas**.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-108">You can set up all required number sequences at the same time by using the **Set up number sequences** wizard, or you can create or modify individual number sequences by using the **Number sequences** page.</span></span>

2. <span data-ttu-id="a4d5e-110">Atlasiet **Numuru sērija**.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-110">Select **Number sequence**.</span></span>

6. <span data-ttu-id="a4d5e-118">Izvērsiet sadaļu **Segmenti**.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-118">Expand the **Segments** section.</span></span>
    - <span data-ttu-id="a4d5e-119">Definējiet formātu numuru sērijai, pievienojot, noņemot un pārkārtojot segmentus.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-119">Define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    - <span data-ttu-id="a4d5e-120">Visu tvērumu numuru sērijas var saturēt *Konstantus segmentus* un *Burtciparu segmentus*.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-120">Number sequences of all scopes can contain *Constant segments* and *Alphanumeric segments*.</span></span> <span data-ttu-id="a4d5e-121">Konstanti segmenti satur burtciparu rakstzīmju kopu, kas nemainās.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-121">Constant segments contain a set of alphanumeric characters that do not change.</span></span> <span data-ttu-id="a4d5e-122">Izmantojiet šo segmenta tipu, lai pievienotu defisi vai citus atdalītājus starp numuru sērijas segmentiem.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-122">Use this segment type to add a hyphen or other separators between number sequence segments.</span></span> <span data-ttu-id="a4d5e-123">Burtciparu segmenti satur numura zīmju (#) un zīmju ampersand (&) data-ttu-id="a4d5e-124">Šīs rakstzīmes apzīmē burtus un ciparus, kas pieaug katru reizi, kad tiek izmantots skaitlis no šīs sērijas.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-124">These ciparu segments.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-127">At least one alphanumeric segment must be present.</span></span> <span data-ttu-id="a4d5e-128">Tvērumu segmenti, piemēram, 9. <span data-ttu-id="a4d5e-143">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="a4d5e-143">Click **Save**.</span></span>
