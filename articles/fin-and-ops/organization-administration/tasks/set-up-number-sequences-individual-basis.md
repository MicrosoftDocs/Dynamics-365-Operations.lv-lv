# <a name="set-up-number-sequences-on-an-individual-basis"></a><span data-ttu-id="98e01-103">Iestatīt atsevišķas numuru sērijas</span><span class="sxs-lookup"><span data-stu-id="98e01-103">Set up number sequences on an individual basis</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="98e01-104">Numuru sērijas tiek izmantotas, lai ģenerētu lasāmus, unikālus pamatdatu ierakstu identifikatorus un darījumu ierakstus, kam tie ir nepieciešami.</span><span class="sxs-lookup"><span data-stu-id="98e01-104">Number sequences are used to generate readable, unique identifiers for master data records and transaction records that require them.</span></span> <span data-ttu-id="98e01-105">Pamatdati vai darījumu ieraksts, kam nepieciešams identifikators, tiek saukts par atsauci.</span><span class="sxs-lookup"><span data-stu-id="98e01-105">A master data or transaction record that requires an identifier is referred to as a reference.</span></span> <span data-ttu-id="98e01-106">Pirms atsaucei varat izveidot jaunus ierakstus, ir jāiestata numuru sērija un tā jāsaista ar atsauci.</span><span class="sxs-lookup"><span data-stu-id="98e01-106">Before you can create new records for a reference, you must set up a number sequence and associate it with the reference.</span></span> <span data-ttu-id="98e01-107">Varat iestatīt visas nepieciešamās numuru sērijas vienlaicīgi, izmantojot vedni Numuru sēriju iestatīšana, vai arī varat izveidot vai modificēt atsevišķas numuru sērijas, izmantojot lapu Numuru sērijas.</span><span class="sxs-lookup"><span data-stu-id="98e01-107">You can set up all required number sequences at the same time by using the Set up number sequences wizard, or you can create or modify individual number sequences by using the Number sequences page.</span></span>

1. <span data-ttu-id="98e01-108">Pārejiet uz sadaļu Organizācijas administrēšana > Numuru secība > Numuru secība.</span><span class="sxs-lookup"><span data-stu-id="98e01-108">Go to Organization administration > Number sequences > Number sequences.</span></span>
2. <span data-ttu-id="98e01-109">Noklikšķiniet uz Numuru sērija.</span><span class="sxs-lookup"><span data-stu-id="98e01-109">Click Number sequence.</span></span>
3. <span data-ttu-id="98e01-110">Laukā Numuru sērijas kods, ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="98e01-110">In the Number sequence code field, type a value.</span></span>
4. <span data-ttu-id="98e01-111">Laukā Nosaukums ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="98e01-111">In the Name field, type a value.</span></span>
5. <span data-ttu-id="98e01-112">Izvērsiet sadaļu Tvēruma parametri.</span><span class="sxs-lookup"><span data-stu-id="98e01-112">Expand the Scope parameters section.</span></span>
    * <span data-ttu-id="98e01-113">Kopsavilkuma cilnē Apjoma parametri, atlasiet numuru sērijas tvēruma diapazonu un atlasiet tvēruma vērtības.</span><span class="sxs-lookup"><span data-stu-id="98e01-113">On the Scope parameters FastTab, select a scope for the number sequence and select scope values.</span></span>     <span data-ttu-id="98e01-114">Tvērums definē, kuras organizācijas izmanto šo numuru sēriju.</span><span class="sxs-lookup"><span data-stu-id="98e01-114">The scope defines which organizations use the number sequence.</span></span> <span data-ttu-id="98e01-115">Turklāt numuru sērijām, kuru tvērums nav Koplietots, var būt segmenti, kas atbilst to tvērumam.</span><span class="sxs-lookup"><span data-stu-id="98e01-115">In addition, number sequences that have a scope other than Shared can have segments that correspond to their scope.</span></span> <span data-ttu-id="98e01-116">Piemēram, numuru sērija ar tvērumu Juridiska persona var ietvert juridiskas personas segmentu.</span><span class="sxs-lookup"><span data-stu-id="98e01-116">For example, a number sequence with a scope of Legal entity can have a legal entity segment.</span></span> <span data-ttu-id="98e01-117">Papildinformāciju par tvērumiem skatiet palīdzības tēmā “Numuru sēriju apskats”.</span><span class="sxs-lookup"><span data-stu-id="98e01-117">For more information about scopes, see the "Number sequence overview" help topic.</span></span>  
6. <span data-ttu-id="98e01-118">Izvērsiet sadaļu Segmenti.</span><span class="sxs-lookup"><span data-stu-id="98e01-118">Expand the Segments section.</span></span>
    * <span data-ttu-id="98e01-119">Kopsavilkuma cilnē Segmenti, definējiet formātu numuru sērijai, pievienojot, noņemot un pārkārtojot segmentus.</span><span class="sxs-lookup"><span data-stu-id="98e01-119">On the Segments FastTab, define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    * <span data-ttu-id="98e01-120">Visu tvērumu numuru sērijas var saturēt konstantus segmentus un burtciparu segmentus.</span><span class="sxs-lookup"><span data-stu-id="98e01-120">Number sequences of all scopes can contain Constant segments and Alphanumeric segments.</span></span> <span data-ttu-id="98e01-121">Konstanti segmenti satur burtciparu rakstzīmju kopu, kas nemainās.</span><span class="sxs-lookup"><span data-stu-id="98e01-121">Constant segments contain a set of alphanumeric characters that do not change.</span></span> <span data-ttu-id="98e01-122">Izmantojiet šo segmenta tipu, lai pievienotu defisi vai citus atdalītājus starp numuru sērijas segmentiem.</span><span class="sxs-lookup"><span data-stu-id="98e01-122">Use this segment type to add a hyphen or other separators between number sequence segments.</span></span> <span data-ttu-id="98e01-123">Burtciparu segmenti satur numura zīmju (#) un zīmju ampersand (&) kombināciju.</span><span class="sxs-lookup"><span data-stu-id="98e01-123">Alphanumeric segments contain a combination of number signs (#) and ampersands (&).</span></span> <span data-ttu-id="98e01-124">Šīs rakstzīmes apzīmē burtus un ciparus, kas pieaug katru reizi, kad tiek izmantots skaitlis no šīs sērijas.</span><span class="sxs-lookup"><span data-stu-id="98e01-124">These characters represent letters and numbers that increment every time that a number from the sequence is used.</span></span> <span data-ttu-id="98e01-125">Norādiet numura zīmi (#), lai norādītu pieaugošus numurus, un zīmi &, lai norādītu pieaugošus burtus.</span><span class="sxs-lookup"><span data-stu-id="98e01-125">Use a number sign (#) to indicate incrementing numbers and an ampersand (&) to indicate incrementing letters.</span></span> <span data-ttu-id="98e01-126">Piemēram, formāts #####_2014 izveido sēriju 00001_2014, 00002_2014 utt.</span><span class="sxs-lookup"><span data-stu-id="98e01-126">For example, the format #####_2014 creates the sequence 00001_2014, 00002_2014, and so on.</span></span>     <span data-ttu-id="98e01-127">Nepieciešams vismaz viens burtu un ciparu segments.</span><span class="sxs-lookup"><span data-stu-id="98e01-127">At least one alphanumeric segment must be present.</span></span> <span data-ttu-id="98e01-128">Tvērumu segmenti, piemēram, uzņēmums vai juridiskā persona, nav obligāti.</span><span class="sxs-lookup"><span data-stu-id="98e01-128">Scope segments, such as company or legal entity, are not required.</span></span> <span data-ttu-id="98e01-129">Tomēr, ja formātā neiekļaujat tvēruma segmentus, katrai atlasītajai atsaucei tik un tā tiek ģenerēti numuri katram tvērumam.</span><span class="sxs-lookup"><span data-stu-id="98e01-129">However, if you do not include scope segments in the format, numbers for the selected reference are still generated per scope.</span></span>  
7. <span data-ttu-id="98e01-130">Izvērsiet sadaļu Atsauces.</span><span class="sxs-lookup"><span data-stu-id="98e01-130">Expand the References section.</span></span>
    * <span data-ttu-id="98e01-131">Kopsavilkuma cilnē Atsauces, atlasiet dokumentu tipu vai pārskatu, kam piešķirt šo numuru sēriju.</span><span class="sxs-lookup"><span data-stu-id="98e01-131">On the References FastTab, select the document type or record to assign this number sequence to.</span></span>     <span data-ttu-id="98e01-132">Šis solis nav obligāts sērijām, kas definētas īpašiem lietojumprogrammas izmantošanas modeļiem.</span><span class="sxs-lookup"><span data-stu-id="98e01-132">This step is optional for sequences that are defined for special application usage patterns.</span></span> <span data-ttu-id="98e01-133">Šajos gadījumos jauns numurs tiek ģenerēts, izmantojot numuru sērijas koda vai ID vērtību, neizmantojot atsauci.</span><span class="sxs-lookup"><span data-stu-id="98e01-133">In these scenarios, a new number is generated by using the value of a number sequence code or ID, without using a reference.</span></span> <span data-ttu-id="98e01-134">Īpašo programmu lietošanas modeļa piemērs ir dokumentu sērija, ko izmanto konkrētiem žurnālu nosaukumiem.</span><span class="sxs-lookup"><span data-stu-id="98e01-134">An example of a special application usage pattern is a voucher series that is used for specific journal names.</span></span> <span data-ttu-id="98e01-135">Tomēr nav ieteicams izmantot šādus modeļus.</span><span class="sxs-lookup"><span data-stu-id="98e01-135">However, we do not recommend that you use such patterns.</span></span>  
8. <span data-ttu-id="98e01-136">Izvērsiet sadaļu Vispārīgi.</span><span class="sxs-lookup"><span data-stu-id="98e01-136">Expand the General section.</span></span>
    * <span data-ttu-id="98e01-137">Kopsavilkuma cilnē Vispārīgi, norādiet, vai numuru sērija ir manuāla, un nepārtraukta vai ar pārtraukumiem.</span><span class="sxs-lookup"><span data-stu-id="98e01-137">On the General FastTab, specify whether the number sequence is manual, and continuous or non-continuous.</span></span> <span data-ttu-id="98e01-138">Turklāt, ievadiet mazāko un lielāko numuru, ko var izmantot šajā numuru sērijā.</span><span class="sxs-lookup"><span data-stu-id="98e01-138">In addition, enter the lowest and highest numbers that can be used in the number sequence.</span></span>     <span data-ttu-id="98e01-139">Nav ieteicams numuru sēriju ar pārtraukumiem mainīt pret nepārtrauktu numuru sēriju.</span><span class="sxs-lookup"><span data-stu-id="98e01-139">We do not recommend changing a non-continuous number sequence to a continuous number sequence.</span></span> <span data-ttu-id="98e01-140">Šāda numuru sērija nevar būt patiesi nepārtraukta.</span><span class="sxs-lookup"><span data-stu-id="98e01-140">The number sequence will not be truly continuous.</span></span> <span data-ttu-id="98e01-141">Šī izmaiņa var izraisīt arī atslēgu dublikātu pārkāpumus datu bāzē.</span><span class="sxs-lookup"><span data-stu-id="98e01-141">This change may also cause duplicate key violations in the database.</span></span> <span data-ttu-id="98e01-142">Turklāt nepārtrauktām numuru sērijām ir lielāka ietekme uz veiktspēju.</span><span class="sxs-lookup"><span data-stu-id="98e01-142">In addition, continuous number sequences have a larger effect on performance.</span></span>   
9. <span data-ttu-id="98e01-143">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="98e01-143">Click Save.</span></span>
