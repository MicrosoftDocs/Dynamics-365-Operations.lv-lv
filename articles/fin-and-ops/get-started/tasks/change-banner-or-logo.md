# <a name="change-the-banners-or-logo-images-for-legal-entities"></a><span data-ttu-id="a21e3-103">Juridiskām personām paredzētu reklāmkarogu vai logotipa attēlu mainīšana</span><span class="sxs-lookup"><span data-stu-id="a21e3-103">Change the banners or logo images for legal entities</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="a21e3-104">Šajā procedūrā uzskaitītas darbības, kuras sistēmas administratori var izmantot, lai atjauninātu reklāmkarogu vai logotipa attēlu, kas tiek rādīts juridiskai personai.</span><span class="sxs-lookup"><span data-stu-id="a21e3-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="a21e3-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="a21e3-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="a21e3-106">Pārejiet uz sadaļu Organizācijas administrēšana > Organizācijas > Juridiskās personas.</span><span class="sxs-lookup"><span data-stu-id="a21e3-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="a21e3-107">Kreisajā pusē esošajā sarakstā atlasiet juridisko personu, kurai vēlaties augšupielādēt reklāmkarogu vai logotipa attēlu.</span><span class="sxs-lookup"><span data-stu-id="a21e3-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="a21e3-108">Ja tā ir jau atlasīta, pārejiet uz nākamo darbību.</span><span class="sxs-lookup"><span data-stu-id="a21e3-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="a21e3-109">Izvērsiet sadaļu Attēli.</span><span class="sxs-lookup"><span data-stu-id="a21e3-109">Expand the Images section.</span></span> <span data-ttu-id="a21e3-110">Ja tā ir jau izvērsta, pārejiet uz nākamo darbību.</span><span class="sxs-lookup"><span data-stu-id="a21e3-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="a21e3-111">Noklikšķiniet uz Rediģēt.</span><span class="sxs-lookup"><span data-stu-id="a21e3-111">Click Edit.</span></span>
5. <span data-ttu-id="a21e3-112">Laukā Informācijas paneļa uzņēmuma attēla veids atlasiet Reklāmkarogs.</span><span class="sxs-lookup"><span data-stu-id="a21e3-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="a21e3-113">Noklikšķiniet uz Mainīt.</span><span class="sxs-lookup"><span data-stu-id="a21e3-113">Click Change.</span></span>
    * <span data-ttu-id="a21e3-114">Padoms: ideālā reklāmkaroga attēla izšķirtspēja ir 1920 x 281 pikseļi.</span><span class="sxs-lookup"><span data-stu-id="a21e3-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="a21e3-115">Ideālais logotipa attēla platums ir 350 pikseļi.</span><span class="sxs-lookup"><span data-stu-id="a21e3-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="a21e3-116">Kad augšupielāde tiek pabeigta, jaunajam reklāmkarogam vai logotipa attēlam būtu jābūt redzamam sadaļā Informācijas paneļa attēls.</span><span class="sxs-lookup"><span data-stu-id="a21e3-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="a21e3-117">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="a21e3-117">Click Save.</span></span>
8. <span data-ttu-id="a21e3-118">Dodieties uz Vispārīgi > Vispārīgi > Noklusējuma informācijas panelis.</span><span class="sxs-lookup"><span data-stu-id="a21e3-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="a21e3-119">Jaunajam reklāmkarogam vai logotipa attēlam būtu jābūt redzamam informācijas panelī.</span><span class="sxs-lookup"><span data-stu-id="a21e3-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="a21e3-120">Var būt nepieciešams mainīt savu uzņēmumu uz to, kuram augšupielādējāt reklāmkarogu, izmantojot uzņēmuma atlasītāju.</span><span class="sxs-lookup"><span data-stu-id="a21e3-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
