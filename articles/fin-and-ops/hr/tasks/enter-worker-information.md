[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="548bc-104">Kad personāla atlases darbinieks ir izveidojis masveida darbā pieņemšanas projektu un ģenerējis amatus, nākamais solis ir ievadīt darbinieka informāciju par cilvēkiem, kas pieņemti darbā.</span><span class="sxs-lookup"><span data-stu-id="548bc-104">After a recruiter has created a mass hire project and generated the positions, the next step is to enter the worker information for the people being hired.</span></span> <span data-ttu-id="548bc-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="548bc-105">The demo data company used to create this procedure is USMF.</span></span> <span data-ttu-id="548bc-106">Demonstrācijas dati ietver atvērtu masveida darbā pieņemšanas projektu.</span><span class="sxs-lookup"><span data-stu-id="548bc-106">The demo data contains an open mass hire project.</span></span> <span data-ttu-id="548bc-107">Lai sāktu šo procedūru, pārejiet uz sadaļu Personāla vadība > Personāla atlase > Masveida darbā pieņemšanas projekti.</span><span class="sxs-lookup"><span data-stu-id="548bc-107">To begin this procedure, go to Human resources > Recruitment > Mass hire projects.</span></span>

1. <span data-ttu-id="548bc-108">Sarakstā atzīmējiet atlasīto rindu.</span><span class="sxs-lookup"><span data-stu-id="548bc-108">In the list, mark the selected row.</span></span>

    * <span data-ttu-id="548bc-110">Rediģējiet darbinieka sākuma datumu.</span><span class="sxs-lookup"><span data-stu-id="548bc-110">Edit the start date for the worker.</span></span>  
    * <span data-ttu-id="548bc-111">Rediģējiet darbinieka beigu datumu.</span><span class="sxs-lookup"><span data-stu-id="548bc-111">Edit the end date for the worker.</span></span>  
    * <span data-ttu-id="548bc-112">Mainiet darbinieka tipu, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="548bc-112">Change the worker type for the worker, if needed.</span></span>  
3. <span data-ttu-id="548bc-113">Laukā Vārds ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="548bc-113">In the First name field, type a value.</span></span>

    * <span data-ttu-id="548bc-116">Mainiet darbinieka darbu, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="548bc-116">Change the job for the worker, if needed.</span></span>  
    * <span data-ttu-id="548bc-117">Mainiet darbinieka nodaļu, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="548bc-117">Change the department for the worker, if needed.</span></span>  
    * <span data-ttu-id="548bc-118">Mainiet darbinieka pilnas slodzes ekvivalenta statusu, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="548bc-118">Change the full-time equivalent status for the worker, if needed.</span></span>  
6. <span data-ttu-id="548bc-119">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="548bc-119">Click OK.</span></span>
