<span data-ttu-id="671a7-104">Jūs varat izveidot mērķus, pievienot aktivitātes mērķiem, izsekot mērījumiem un saistīt mērķus ar pārskatu.</span><span class="sxs-lookup"><span data-stu-id="671a7-104">You can create goals, add activities to goals, track measurements, and link the goals to a review.</span></span> <span data-ttu-id="671a7-105">Šī procedūra tika izveidota, izmantojot demonstrācijas uzņēmuma USMF datus.</span><span class="sxs-lookup"><span data-stu-id="671a7-105">This procedure was created using the USMF demo data company.</span></span> <span data-ttu-id="671a7-106">Šī procedūra ir paredzēta līdzeklim, kas tika pievienots Dynamics 365 for Operations versijā 1611.</span><span class="sxs-lookup"><span data-stu-id="671a7-106">This procedure is for a feature that was added in Dynamics 365 for Operations, version 1611.</span></span>

2. <span data-ttu-id="671a7-108">Noklikšķiniet uz Skatīt visus mērķus</span><span class="sxs-lookup"><span data-stu-id="671a7-108">Click View all goals</span></span>
3. <span data-ttu-id="671a7-109">Noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="671a7-109">Click New.</span></span>
4. <span data-ttu-id="671a7-110">Laukā Nosaukums ierakstiet 'Pabeigt Contoso projektu'.</span><span class="sxs-lookup"><span data-stu-id="671a7-110">In the Name field, type 'Complete the Contoso project'.</span></span>
5. <span data-ttu-id="671a7-111">Pārskata laukā ievadiet 'Es pabeigšu Contoso projektu 30 dienās'.</span><span class="sxs-lookup"><span data-stu-id="671a7-111">In the Overview field, enter 'I will finish the Contoso project in 30 days'.</span></span>
    * <span data-ttu-id="671a7-112">Mērķus var pielietot individuāli, grupai vai pat korporācijai.</span><span class="sxs-lookup"><span data-stu-id="671a7-112">Goals can apply to an individual, a team, or even a corporation.</span></span> <span data-ttu-id="671a7-113">Noklusējuma iestatījums Darbinieka mērķiem ir Individuāli.</span><span class="sxs-lookup"><span data-stu-id="671a7-113">The default setting for Employee goals is Individual.</span></span>

7. <span data-ttu-id="671a7-115">Laukā Sākuma datums mainiet norādīto datumu.</span><span class="sxs-lookup"><span data-stu-id="671a7-115">In the Start date field, change the date shown.</span></span>
8. <span data-ttu-id="671a7-116">Laukā Beigu datums mainiet norādīto datumu.</span><span class="sxs-lookup"><span data-stu-id="671a7-116">In the End date field, change the date shown.</span></span>
    * <span data-ttu-id="671a7-117">Pabeidzot mērķi, ievadiet datumu, kad mērķis tika izpildīts šeit.</span><span class="sxs-lookup"><span data-stu-id="671a7-117">When you complete a goal, enter the date that you completed the goal here.</span></span>

10. <span data-ttu-id="671a7-119">Izvērsiet sadaļu Mērījumi.</span><span class="sxs-lookup"><span data-stu-id="671a7-119">Expand the Measurements section.</span></span> <span data-ttu-id="671a7-120">Ja tā jau ir izvērsta, divreiz noklikšķiniet bultiņu Izvērst.</span><span class="sxs-lookup"><span data-stu-id="671a7-120">If it is already expanded, click the Expand arrow twice.</span></span>
11. <span data-ttu-id="671a7-121">Noklikšķiniet uz Pievienot mērījumu, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="671a7-121">Click Add measurement to open the drop dialog.</span></span>

13. <span data-ttu-id="671a7-123">Laukā mērķa summa ievadiet skaitli.</span><span class="sxs-lookup"><span data-stu-id="671a7-123">In the Target amount field, enter a number.</span></span>
14. <span data-ttu-id="671a7-124">Noklikšķiniet uz Labi.</span><span class="sxs-lookup"><span data-stu-id="671a7-124">Click OK.</span></span>
15. <span data-ttu-id="671a7-125">Izvērsiet sadaļu Aktivitātes.</span><span class="sxs-lookup"><span data-stu-id="671a7-125">Expand the Activities section.</span></span> <span data-ttu-id="671a7-126">Ja tā jau ir izvērsta, divreiz noklikšķiniet bultiņu Izvērst.</span><span class="sxs-lookup"><span data-stu-id="671a7-126">If it is already expanded, click the Expand arrow twice.</span></span>
16. <span data-ttu-id="671a7-127">Noklikšķiniet uz Pievienot.</span><span class="sxs-lookup"><span data-stu-id="671a7-127">Click Add.</span></span>
17. <span data-ttu-id="671a7-128">Laukā Virsraksts ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="671a7-128">In the Title field, type a value.</span></span>
18. <span data-ttu-id="671a7-129">Apraksta laukā ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="671a7-129">In the Description field, type a value.</span></span>
19. <span data-ttu-id="671a7-130">Klikšķiniet Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="671a7-130">Click Save.</span></span>
20. <span data-ttu-id="671a7-131">Noklikšķiniet uz Rādīt sadaļas, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="671a7-131">Click Show sections to open the drop dialog.</span></span>
21. <span data-ttu-id="671a7-132">Atlasiet Nē laukā Rādīt mērījumus.</span><span class="sxs-lookup"><span data-stu-id="671a7-132">Select No in the Show measurements field.</span></span>
22. <span data-ttu-id="671a7-133">Klikšķiniet Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="671a7-133">Click Save.</span></span>
23. <span data-ttu-id="671a7-134">Noklikšķiniet uz Rādīt sadaļas, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="671a7-134">Click Show sections to open the drop dialog.</span></span>
24. <span data-ttu-id="671a7-135">Atlasiet Jā laukā Rādīt mērījumus.</span><span class="sxs-lookup"><span data-stu-id="671a7-135">Select Yes in the Show measurements field.</span></span>
25. <span data-ttu-id="671a7-136">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="671a7-136">Click Save.</span></span>
    * <span data-ttu-id="671a7-137">Bieži ir mērķi, kas ir vienādi vai līdzīgi grupas dalībniekiem vai visiem darbiniekiem.</span><span class="sxs-lookup"><span data-stu-id="671a7-137">There are often goals that are the same or similar for team members or all employees.</span></span>     <span data-ttu-id="671a7-138">Jūsu personāla vadības administrators vai vadītājs var izveidot mērķa veidnes šādiem mērķiem.</span><span class="sxs-lookup"><span data-stu-id="671a7-138">Your HR administrator or manager can create goal templates for those goals.</span></span> <span data-ttu-id="671a7-139">Jūs varat pievienot mērķi, izmantojot veidni, un pēc tam modificēt to, lai tas atbilstu jūsu kalendāram.</span><span class="sxs-lookup"><span data-stu-id="671a7-139">You can add a goal from a template, and then modify it to match your timelines.</span></span>
