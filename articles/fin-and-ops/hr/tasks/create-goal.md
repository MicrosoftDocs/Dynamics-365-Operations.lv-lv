15. <span data-ttu-id="6bf53-125">Izvērsiet sadaļu Aktivitātes.</span><span class="sxs-lookup"><span data-stu-id="6bf53-125">Expand the Activities section.</span></span> <span data-ttu-id="6bf53-126">Ja tā jau ir izvērsta, divreiz noklikšķiniet bultiņu Izvērst.</span><span class="sxs-lookup"><span data-stu-id="6bf53-126">If it is already expanded, click the Expand arrow twice.</span></span>

17. <span data-ttu-id="6bf53-128">Laukā Virsraksts ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="6bf53-128">In the Title field, type a value.</span></span>

19. <span data-ttu-id="6bf53-130">Klikšķiniet Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="6bf53-130">Click Save.</span></span>
20. <span data-ttu-id="6bf53-131">Noklikšķiniet uz Rādīt sadaļas, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="6bf53-131">Click Show sections to open the drop dialog.</span></span>

22. <span data-ttu-id="6bf53-133">Klikšķiniet Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="6bf53-133">Click Save.</span></span>
23. <span data-ttu-id="6bf53-134">Noklikšķiniet uz Rādīt sadaļas, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="6bf53-134">Click Show sections to open the drop dialog.</span></span>
