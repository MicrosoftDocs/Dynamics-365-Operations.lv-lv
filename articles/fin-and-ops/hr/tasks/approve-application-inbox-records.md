# <a name="approve-application-inbox-records"></a>Pieteikumu iesūtnes ierakstu apstiprināšana

[!include [task guide banner](../../includes/task-guide-banner.md)]

Šajā procedūrā parādīts, kā pārskatīt pieteikumus, kas saņemti no darbinieku pašapkalpošanās lapām. Papildus pieteikumu pārskatīšanai varat apstiprināt pieteikumu jūsu atlasītajos iesūtnes ierakstos. Pieteikuma iesūtnes ieraksti atspoguļo darba pieteikumus, kas uzņēmumam iesniegti izskatīšanai. Pēc ieraksta apstiprināšanas, personai, kas iesniegusi pieteikumu, tiks izveidots kandidāta ieraksts. Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.


## <a name="review-application-inbox-record"></a>Pārskatīt pieteikumu iesūtnes ierakstu
1. Dodieties uz Personāla vadība > Personāla atlase > Pieteikumi > Pieteikumu iesūtne.
2. Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.
3. Izvērsiet sadaļu Adreses.
4. Izvērsiet sadaļu Kontaktinformācija.
5. Izvērsiet sadaļu Pielikumi.

## <a name="approve-application-inbox-record"></a>Apstiprināt pieteikumu iesūtnes ierakstu
1. Sarakstā atzīmējiet atlasīto rindu.
2. Ievērojiet vērtību laukā Nosaukums vēlākai atsaucei
3. Noklikšķiniet uz Apstiprināt.
4. Noklikšķiniet uz OK.
5. Aizvērt lapu.

## <a name="view-the-newly-created-application-record"></a>Skatīt tikai jaunizveidoto pieteikuma ierakstu
1. Pārejiet uz sadaļu Personāla vadība > Personāla atlase > Pieteikumi > Pieteikumi.
2. Sarakstā noklikšķiniet uz saites atlasītajā rindā.
3. Izvērsiet sadaļu Pielikumi.
