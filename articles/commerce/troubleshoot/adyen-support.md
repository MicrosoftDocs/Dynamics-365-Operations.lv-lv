# <a name="troubleshoot-dynamics-365-payment-connector-for-adyen-issues"></a>Dynamics 365 Payment Connector Adyen problēmu novēršana

[!include [banner](../../includes/banner.md)]

Šajā tēmā sniegti norādījumi problēmu novēršanai, kas var palīdzēt, ja jums radušās ar Adyen Microsoft Dynamics 365 Payment Connector.

## <a name="description"></a>Apraksts

Jums ir radušās problēmas ar Adyen Dynamics 365 Payment Connector šādās jomās, un jums ir nepieciešams atbalsts no Adyen komandas:

- Pārdošanas punkta (POS), modernā pārdošanas punkta (MPOS), zvanu centra vai Dynamics 365 Commerce konfigurēšana
- Adyen maksājumu pakalpojumu sniedzēja (PSP) atsauces numurs (piemēram, ja jums radušies jautājumi par atteikumiem, tostarp attiecībā uz manuālo atslēgu ievadi \[MKE\])
- Jebkas, kas nedarbojas testa vai tiešo pārdevēju vidēs

## <a name="resolution"></a>Novēršana

Izmantojiet šo e-pasta veidni, lai sāktu atbalsta procesu ar Adyen komandu. Lai paātrinātu problēmu novēršanu, nodrošiniet, ka e-pastā ir visa nepieciešamā informācija.

| Lauks        | Vērtība |
|--------------|-------|
| Kam           | `support@adyen.com` |
| Kopija           | |
| Tēmas rindiņa | Microsoft Dynamics atbalsta pieprasījums |
| Pamatteksts         | <p>Sveicināti, Atbalsta komanda,</p><p>Lūdzu, sniedziet atbalstu saistībā ar šādu problēmu:</p><ul><li>Tirgotāja konts</li><li>Vide (Testa/Ražošanas)</li><li>Kanāls (POS/zvanu centrs/Commerce e-komercija)</li><li>PSP atsauces numurs, ja rēķins iesaistīts konkrētā transakcijā (PSP numuru varat atrast uz kvīts, Adyen klientu zonā vai POS termināla transakciju izvēlnē.)</li><li>Kļūdas ziņojuma ekrānuzņēmums vai fotoattēls, ja piemērojami.</li><li>Event Viewer žurnāli (.txt formātā)</li><li>Problēmas apraksts un jūsu izmēģinātās problēmu novēršanas darbības</li></ul> |
