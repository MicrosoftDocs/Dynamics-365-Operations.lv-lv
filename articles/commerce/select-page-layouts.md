> [!NOTE]
> <span data-ttu-id="8d294-106">Esošai lapai var izveidot izkārtojumus tikai tad, ja šajā lapā ir vismaz divi moduļi galvenajā slotā.</span><span class="sxs-lookup"><span data-stu-id="8d294-106">You can create layouts for an existing page only if that page has at least two modules under the main slot.</span></span>

1. <span data-ttu-id="8d294-110">Atlasiet lapu, atlasiet **Rediģēt**, lai to pārbaudītu, un pēc tam atlasiet lapas nosaukumu, lai to atvērtu.</span><span class="sxs-lookup"><span data-stu-id="8d294-110">Select the page, select **Edit** to check it out, and then select the page name to open it.</span></span> <span data-ttu-id="8d294-111">Pierakstiet moduļu secību.</span><span class="sxs-lookup"><span data-stu-id="8d294-111">Make a note of the module order.</span></span>

1. <span data-ttu-id="8d294-114">Atlasiet **Pārvērst par iegulto izkārtojumu**.</span><span class="sxs-lookup"><span data-stu-id="8d294-114">Select **Convert to Embedded Layout**.</span></span>
1. <span data-ttu-id="8d294-115">Mainiet moduļu secību, kā tas ir nepieciešams, un pierakstiet jauno secību.</span><span class="sxs-lookup"><span data-stu-id="8d294-115">Change the order of the modules as you require, and make a note of the new order.</span></span>

1. <span data-ttu-id="8d294-117">Ievadiet izkārtojuma nosaukumu un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="8d294-117">Enter a name for the layout, and then select **OK**.</span></span>

1. <span data-ttu-id="8d294-121">Atlasiet **Pabeigt rediģēšanu**, lai to pārbaudītu lapā, un pēc tam atlasiet **Publicēt**, lai publicētu to.</span><span class="sxs-lookup"><span data-stu-id="8d294-121">Select **Finish editing** to check in the page, and then select **Publish** to publish it.</span></span>
