## <a name="add-a-copyright-notice"></a><span data-ttu-id="0cf99-109">Autortiesību paziņojuma pievienošana</span><span class="sxs-lookup"><span data-stu-id="0cf99-109">Add a copyright notice</span></span>

<span data-ttu-id="0cf99-110">Lai pievienotu autortiesību paziņojumu katras lapas apakšdaļā, kurā tiek izmantota noteikta veidne, veiciet šādas darbības.</span><span class="sxs-lookup"><span data-stu-id="0cf99-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="0cf99-111">Dodieties uz **Fragmenti** un pēc tam atlasiet **Jaunas lapas fragments**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="0cf99-112">Dialoglodziņā atlasiet moduli **Kājene** un piešķiriet fragmentam nosaukumu.</span><span class="sxs-lookup"><span data-stu-id="0cf99-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="0cf99-113">Piemēram, ievadiet **Kājenes autortiesības**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="0cf99-114">Atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-114">Select **OK**.</span></span>
1. <span data-ttu-id="0cf99-115">Navigācijas rūtī atlasiet daudzpunktes pogu (**...**) blakus **Kājene** un pēc tam atlasiet **Pievienot moduli**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="0cf99-116">Dialoglodziņā atlasiet **Kājenes kategorija** un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="0cf99-117">Navigācijas rūtī atlasiet daudzpunktes pogu blakus **Kājenes kategorija** un pēc tam atlasiet **Pievienot moduli**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="0cf99-118">Dialoglodziņā atlasiet **Bagātinātā satura bloka elements** un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-118">In the dialog box, select **Content rich block item**, and then select **OK**.</span></span>
1. <span data-ttu-id="0cf99-119">Navigācijas rūtī atlasiet **Satura bagātinātā bloka elements**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-119">In the navigation pane, select **Content rich block item**.</span></span>
1. <span data-ttu-id="0cf99-120">Rekvizītu rūts labajā pusē laukā **Rindkopa** pievienojiet savu autortiesību ziņojumu.</span><span class="sxs-lookup"><span data-stu-id="0cf99-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="0cf99-121">Piemēram, ievadiet **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="0cf99-122">Atlasiet **Saglabāt**, atlasiet **Pārbaudīt** un pēc tam atlasiet **Publicēt**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="0cf99-123">Dodieties uz **Veidnes**, atlasiet veidni un pēc tam atlasiet **Norēķināties**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="0cf99-124">Sadaļā **Lapas strukturējums** izvērsiet **Pamatteksts** un pēc tam izvērsiet **Noklusējuma lapa**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="0cf99-125">Atlasiet daudzpunktes pogu blakus **Kājenes slots** un pēc tam atlasiet **Pievienot fragmentu**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="0cf99-126">Atlasiet iepriekš izveidoto fragmentu un pēc tam izvēlieties **Atlasīt**.</span><span class="sxs-lookup"><span data-stu-id="0cf99-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="0cf99-127">Pārbaudiet veidni un publicējiet to.</span><span class="sxs-lookup"><span data-stu-id="0cf99-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="0cf99-128">Kājene, kas satur autortiesību paziņojumu, automātiski tiek parādīta visu to lapu apakšā, kuras izmanto atlasīto veidni.</span><span class="sxs-lookup"><span data-stu-id="0cf99-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
