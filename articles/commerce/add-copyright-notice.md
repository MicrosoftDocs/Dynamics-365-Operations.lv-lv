## <a name="add-a-copyright-notice"></a><span data-ttu-id="c55b9-109">Autortiesību paziņojuma pievienošana</span><span class="sxs-lookup"><span data-stu-id="c55b9-109">Add a copyright notice</span></span>

<span data-ttu-id="c55b9-110">Lai pievienotu autortiesību paziņojumu katras lapas apakšdaļā, kurā tiek izmantota noteikta veidne, veiciet šādas darbības.</span><span class="sxs-lookup"><span data-stu-id="c55b9-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="c55b9-111">Dodieties uz **Fragmenti** un pēc tam atlasiet **Jaunas lapas fragments**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="c55b9-112">Dialoglodziņā atlasiet moduli **Kājene** un piešķiriet fragmentam nosaukumu.</span><span class="sxs-lookup"><span data-stu-id="c55b9-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="c55b9-113">Piemēram, ievadiet **Kājenes autortiesības**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="c55b9-114">Atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-114">Select **OK**.</span></span>
1. <span data-ttu-id="c55b9-115">Navigācijas rūtī atlasiet daudzpunktes pogu (**...**) blakus **Kājene** un pēc tam atlasiet **Pievienot moduli**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="c55b9-116">Dialoglodziņā atlasiet **Kājenes kategorija** un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="c55b9-117">Navigācijas rūtī atlasiet daudzpunktes pogu blakus **Kājenes kategorija** un pēc tam atlasiet **Pievienot moduli**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="c55b9-118">Dialoglodziņā atlasiet **Teksta bloks** un pēc tam atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="c55b9-119">Navigācijas rūtī atlasiet **Teksta bloks**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="c55b9-120">Rekvizītu rūts labajā pusē laukā **Rindkopa** pievienojiet savu autortiesību ziņojumu.</span><span class="sxs-lookup"><span data-stu-id="c55b9-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="c55b9-121">Piemēram, ievadiet **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="c55b9-122">Atlasiet **Saglabāt**, atlasiet **Pārbaudīt** un pēc tam atlasiet **Publicēt**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="c55b9-123">Dodieties uz **Veidnes**, atlasiet veidni un pēc tam atlasiet **Norēķināties**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="c55b9-124">Sadaļā **Lapas strukturējums** izvērsiet **Pamatteksts** un pēc tam izvērsiet **Noklusējuma lapa**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="c55b9-125">Atlasiet daudzpunktes pogu blakus **Kājenes slots** un pēc tam atlasiet **Pievienot fragmentu**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="c55b9-126">Atlasiet iepriekš izveidoto fragmentu un pēc tam izvēlieties **Atlasīt**.</span><span class="sxs-lookup"><span data-stu-id="c55b9-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="c55b9-127">Pārbaudiet veidni un publicējiet to.</span><span class="sxs-lookup"><span data-stu-id="c55b9-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="c55b9-128">Kājene, kas satur autortiesību paziņojumu, automātiski tiek parādīta visu to lapu apakšā, kuras izmanto atlasīto veidni.</span><span class="sxs-lookup"><span data-stu-id="c55b9-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
