# <a name="crop-images"></a><span data-ttu-id="8125a-103">Attēlu apgriešana</span><span class="sxs-lookup"><span data-stu-id="8125a-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="8125a-104">Šajā tēmā ir aprakstīts, kā apgriezt attēlus programmas Microsoft Dynamics 365 Commerce vietnes veidotājā.</span><span class="sxs-lookup"><span data-stu-id="8125a-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="8125a-105">Pārskats</span><span class="sxs-lookup"><span data-stu-id="8125a-105">Overview</span></span>

<span data-ttu-id="8125a-106">Commerce vietnes veidotāja multivides bibliotēka ļauj apgriezt attēlus, lai optimizētu tos dažādiem moduļu tipiem un skatvietām.</span><span class="sxs-lookup"><span data-stu-id="8125a-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="8125a-107">Attēla apgriešana</span><span class="sxs-lookup"><span data-stu-id="8125a-107">Crop an image</span></span>

<span data-ttu-id="8125a-108">Lai apgrieztu attēlu vietnes veidotājā, veiciet tālāk norādītās darbības.</span><span class="sxs-lookup"><span data-stu-id="8125a-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="8125a-109">Commerce vietnes veidotāja kreisajā navigācijas rūtī atlasiet **Multivides bibliotēka**.</span><span class="sxs-lookup"><span data-stu-id="8125a-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="8125a-110">Galvenajā logā atlasiet attēlu, kuru vēlaties pārveidot.</span><span class="sxs-lookup"><span data-stu-id="8125a-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="8125a-111">Komandjoslā atlasiet **Rediģēt**.</span><span class="sxs-lookup"><span data-stu-id="8125a-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="8125a-112">Atlasiet attēlu, kuru vēlaties ievadīt **Rediģēšanas režīmā**.</span><span class="sxs-lookup"><span data-stu-id="8125a-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="8125a-113">Sadaļā **Rediģēšanas režīms** atlasiet **Rediģēt skatu pēc moduļa**.</span><span class="sxs-lookup"><span data-stu-id="8125a-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="8125a-114">**Moduļa** nolaižamajā izvēlnē atlasiet moduļa veidu.</span><span class="sxs-lookup"><span data-stu-id="8125a-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="8125a-115">**Skata veida** nolaižamajā izvēlnē atlasiet skata veidu.</span><span class="sxs-lookup"><span data-stu-id="8125a-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="8125a-116">Nolaižamajā izvēlnē **Novietojums** atlasiet attēla novietojumu.</span><span class="sxs-lookup"><span data-stu-id="8125a-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="8125a-117">Nolaižamajā izvēlnē **Skatvieta** atlasiet skatvietas izmēru.</span><span class="sxs-lookup"><span data-stu-id="8125a-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="8125a-118">Attēls tiek pārklāts ar apgabalu, kas attēlo apgriešanas reģionu.</span><span class="sxs-lookup"><span data-stu-id="8125a-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="8125a-119">Pārvietojiet un mainiet apgriešanas reģiona izmērus pēc nepieciešamības.</span><span class="sxs-lookup"><span data-stu-id="8125a-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="8125a-120">Proporcijas tiks saglabātas automātiski.</span><span class="sxs-lookup"><span data-stu-id="8125a-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="8125a-121">Kad esat pabeidzis, komandjoslā atlasiet **Saglabāt** un pēc tam atlasiet **Pabeigt rediģēšanu**.</span><span class="sxs-lookup"><span data-stu-id="8125a-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="8125a-122">Pēc pielāgotās apgriešanas pabeigšanas attēla izmaiņas stāsies spēkā gandrīz nekavējoties.</span><span class="sxs-lookup"><span data-stu-id="8125a-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
