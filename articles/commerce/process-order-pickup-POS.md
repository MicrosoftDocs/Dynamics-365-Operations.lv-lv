<span data-ttu-id="714d9-121">Ja ir ieslēgts līdzeklis **Uzlabota lietotāja pieredze izdošanas pasūtījuma apstrādei pārdošanas punktā**, un jūs atlasāt operāciju **Izdošana**, tiks parādīts **Izdošanas** dialoglodziņš.</span><span class="sxs-lookup"><span data-stu-id="714d9-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="714d9-122">Tajā varat atlasīt krājumus un daudzumus, kas tiks izdoti.</span><span class="sxs-lookup"><span data-stu-id="714d9-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="714d9-123">Pēc noklusējuma visi krājumu pasūtītie daudzumi, kuri ir izdošanas vai iepakošanas stāvoklī, tiek uzskatīti par piemērotiem izdošanai.</span><span class="sxs-lookup"><span data-stu-id="714d9-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="714d9-124">Pēc noklusējuma šis daudzums tiek iestatīts kā izdošanas daudzums.</span><span class="sxs-lookup"><span data-stu-id="714d9-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="714d9-125">Ievadīto daudzumu var mainīt, ja daudzums nav 0 (nulle) un nepārsniedz atlasītās rindas kopējo atvērto (t.i., rēķinā neiekļauto) daudzumu.</span><span class="sxs-lookup"><span data-stu-id="714d9-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>
