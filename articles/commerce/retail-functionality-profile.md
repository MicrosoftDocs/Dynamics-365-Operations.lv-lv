## <a name="create-a-functionality-profile"></a><span data-ttu-id="4d614-108">Izveidot funkcionalitātes profilu</span><span class="sxs-lookup"><span data-stu-id="4d614-108">Create a functionality profile</span></span>

<span data-ttu-id="4d614-109">Lai izveidotu jaunu funkcionalitāti, veiciet šādas darbības.</span><span class="sxs-lookup"><span data-stu-id="4d614-109">To create a functionality profile, follow these steps.</span></span>

1. <span data-ttu-id="4d614-110">Navigācijas rūtī atveriet **Moduļi \> Kanāla iestatīšana \> POS profili \> Funkcionalitātes profili**.</span><span class="sxs-lookup"><span data-stu-id="4d614-110">In the navigation pane, go to **Modules \> Channel setup \> POS profiles \> Functionality profiles**.</span></span>
1. <span data-ttu-id="4d614-111">Darbību rūtī atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="4d614-111">On the action pane, select **New**.</span></span>
1. <span data-ttu-id="4d614-112">Laukā **Profils** ievadiet profila ID ("FN006" zemāk esošā attēla piemērā).</span><span class="sxs-lookup"><span data-stu-id="4d614-112">In the **Profile** field, enter an ID for the profile ("FN006" in the example image below).</span></span>
1. <span data-ttu-id="4d614-113">Laukā **Apraksts** ievadiet vērtību ("Adventure Works profilu" piemērā zemāk esošajā attēlā).</span><span class="sxs-lookup"><span data-stu-id="4d614-113">In the **Description** field, enter a value ("Adventure Works Profile" in the example image below).</span></span>
1. <span data-ttu-id="4d614-114">Sadaļā **Vispārīgi** atlasiet **ISO** lokalizācijas valsti.</span><span class="sxs-lookup"><span data-stu-id="4d614-114">In the **General** section, select a country for the **ISO** locale.</span></span>
1. <span data-ttu-id="4d614-115">Sadaļā **Vispārīgi** pēc nepieciešamības modificējiet citus iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="4d614-115">In the **General** section, modify other settings, as needed.</span></span>
1. <span data-ttu-id="4d614-116">Sadaļā **Vispārīgi** atlasiet **Saņemšanas profila ID** e-pasta kvītīm.</span><span class="sxs-lookup"><span data-stu-id="4d614-116">In the **General** section, select a **Receipt profile ID** for email receipts.</span></span>
1. <span data-ttu-id="4d614-117">Sadaļā **Funkcijas** pēc nepieciešamības modificējiet iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="4d614-117">In the **Functions** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="4d614-118">Sadaļā **Summa** pēc nepieciešamības modificējiet iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="4d614-118">In the **Amount** section, modify settings as, needed.</span></span>
1. <span data-ttu-id="4d614-119">Sadaļā **Informācijas kodi** pēc nepieciešamības modificējiet iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="4d614-119">In the **Info Codes** section, modify settings, as needed.</span></span>
1. <span data-ttu-id="4d614-120">Sadaļā **Kvīšu numerācija** pēc nepieciešamības modificējiet citus iestatījumus.</span><span class="sxs-lookup"><span data-stu-id="4d614-120">In the **Receipt numbering** section, modify settings, as needed.</span></span>
