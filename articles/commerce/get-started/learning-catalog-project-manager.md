# <a name="learning-catalog-for-implementation-project-managers-working-with-dynamics-365-commerce"></a>Apmācības katalogs ieviešanas projektu pārvaldniekiem, kas strādā ar Dynamics 365 Commerce

Vai atbildat par to, ka jūsu uzņēmuma ieviešana norit raiti?

Tālāk minētais katalogs ir sakārtots no pamatzināšanām uz noteiktiem domēniem kārtībā no visvienkāršākā uz vismodernāko. Ja saturs pastāv vairākos formātos, mēs jūs informēsim, lai varētu izvēlēties apmācības formātu, kas vislabāk atbilst jūsu vajadzībām.

Lai iegūtu apmācību par līdzekļiem, kas koplietoti ar citām finanšu un operāciju programmām, skatiet [Dynamics 365 Finanšu mācību katalogā](../../finance/get-started/learning-catalog-project-manager.md).
