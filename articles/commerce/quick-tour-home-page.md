# <a name="home-page-overview"></a><span data-ttu-id="e7962-103">Sākumlapas pārskats</span><span class="sxs-lookup"><span data-stu-id="e7962-103">Home page overview</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="e7962-104">Šajā tēmā ir sniegts pārskats par programmas Microsoft Dynamics 365 Commerce mājas lapu.</span><span class="sxs-lookup"><span data-stu-id="e7962-104">This topic provides an overview of the home page in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="e7962-105">Mājas lapa ir noklusējuma lapa, kur pircēji dodas, kad tie apmeklē e-tirdzniecības vietni.</span><span class="sxs-lookup"><span data-stu-id="e7962-105">The home page is the default page that shoppers go to when they visit an e-Commerce site.</span></span> <span data-ttu-id="e7962-106">Parasti šī lapa parāda preces un veicināšanas pasākumus, izmantojot mārketinga moduļu kombināciju.</span><span class="sxs-lookup"><span data-stu-id="e7962-106">Typically, this page showcases products and promotions by using a combination of marketing modules.</span></span> <span data-ttu-id="e7962-107">Mājas lapai jābūt papildinātai ar attēliem un tekstu, lai iesaistītu pircējus.</span><span class="sxs-lookup"><span data-stu-id="e7962-107">The home page should be rich with images and text to keep shoppers engaged.</span></span>

<span data-ttu-id="e7962-108">Tālāk redzamajā attēlā ir parādīts mājas lapas piemērs, kura izveidota, izmantojot moduļu bibliotēku un “Fabrikam” tēmu.</span><span class="sxs-lookup"><span data-stu-id="e7962-108">The following illustration shows an example of a home page that was built by using the module library and the "Fabrikam" theme.</span></span>

![Mājas lapas piemērs](./media/Homepage2.PNG)

<span data-ttu-id="e7962-110">Mājas lapas augšpusē ir virsraksts, kas parāda visas preču kategorijas un citas lapas, kuras mazumtirgotājs vēlas, lai klienti pārlūkotu.</span><span class="sxs-lookup"><span data-stu-id="e7962-110">The top of the home page has a header that shows all the product categories and other pages that the retailer wants customers to browse.</span></span> <span data-ttu-id="e7962-111">Mājas lapas apakšpusē ir kājene, kas ietver ātras saites uz dažādām tēmām, kas varētu interesēt pircējus.</span><span class="sxs-lookup"><span data-stu-id="e7962-111">The bottom of the home page has a footer that contains quick links to various topics that might interest customers.</span></span>

<span data-ttu-id="e7962-112">Mājas lapas galvenā sadaļa var izcelt preces, kategorijas vai veicināšanas pasākumus, izmantojot dažādus Dynamics 365 Commerce moduļus.</span><span class="sxs-lookup"><span data-stu-id="e7962-112">The main section of the home page can highlight products, categories, or promotions by using various Dynamics 365 Commerce modules:</span></span>

- <span data-ttu-id="e7962-113">**Hero** — parasti pirmā prece galvenās sadaļas augšpusē parāda vienu vai vairākus "hero" attēlus, kas veikalā izceļ jaunas preces un veicināšanas pasākumus.</span><span class="sxs-lookup"><span data-stu-id="e7962-113">**Hero** – Typically, the first item at the top of the main section shows one or more "hero" images that highlight new products and promotions in the store.</span></span> <span data-ttu-id="e7962-114">Ja ir vairāki hero attēli, tie tiek viesoti karuseļa modulī, lai lietotāji varētu tos pārlūkot.</span><span class="sxs-lookup"><span data-stu-id="e7962-114">If there are multiple hero images, they are hosted in a carousel module so that users can browse them.</span></span>

    <span data-ttu-id="e7962-115">Tālāk redzamajā attēlā parādīts mājas lapas piemērs, kur pirmā prece galvenajā sadaļā ir satura bloka izkārtojuma hero modulis ar nosaukumu "Jaunumi".</span><span class="sxs-lookup"><span data-stu-id="e7962-115">The following illustration shows an example of a home page where the first item in the main section is a hero layout of a content block module that is named "New Arrival."</span></span>

    ![Hero moduļa piemērs](./media/Hero.PNG)

- <span data-ttu-id="e7962-117">**Līdzeklis** — līdzekļa izkārtojums satura bloka modulim tiek izmantots, lai reklamētu preces vai veicināšanas pasākumus, izmantojot attēlu un teksta kombināciju.</span><span class="sxs-lookup"><span data-stu-id="e7962-117">**Feature** – A feature layout of a content block module is used to market products or promotions by using a combination of images and text.</span></span> <span data-ttu-id="e7962-118">Līdzekļu izkārtojumus var izmantot neatkarīgi, vai arī tos var viesot karuseļa modulī.</span><span class="sxs-lookup"><span data-stu-id="e7962-118">Features layouts can be used independently, or they can be hosted in a carousel module.</span></span>

    <span data-ttu-id="e7962-119">Tālāk esošajā attēlā redzams satura bloka moduļa līdzekļa izkārtojuma piemērs mājas lapā.</span><span class="sxs-lookup"><span data-stu-id="e7962-119">The following illustration shows an example of feature layout of a content block module on a home page.</span></span>

    ![Līdzekļu moduļu piemēri](./media/Feature.PNG)

- <span data-ttu-id="e7962-121">**Elements** — satura bloka elementa izkārtojuma modulis tiek izmantots, lai parādītu vairākas preces vai preču kategorijas, izmantojot attēlu un teksta kombināciju daudzkolonnu izkārtojumā.</span><span class="sxs-lookup"><span data-stu-id="e7962-121">**Tile** – A tile layout of a content block module is used to showcase multiple products or category of products by using a combination of images and text in a multicolumn layout.</span></span> <span data-ttu-id="e7962-122">Mājas lapas attēlā, kas parādīts iepriekš šajā tēmā, tiek lietots elementa izkārtojums modulis, kas tiek izmantots elementu **Iepirkties sievietēm**, **Iepirkties vīriešiem** un **Pirkt aksesuārus** trīs kolonnu atveidē.</span><span class="sxs-lookup"><span data-stu-id="e7962-122">In the illustration of a home page that appears earlier in this topic, a tile  layout is used for the three-column rendering of the **Shop Women**, **Shop Men**, and **Shop Accessories** items.</span></span>
- <span data-ttu-id="e7962-123">**Video atskaņotājs** — video atskaņotāja modulis var tikt izmantots, lai mājas lapā parādītu video saturu.</span><span class="sxs-lookup"><span data-stu-id="e7962-123">**Video player** – A video player module can be used to showcase video content on the home page.</span></span> <span data-ttu-id="e7962-124">Mājas lapas ilustrācija, kas parādīta iepriekš šajā tēmā, ietver video atskaņotāja moduli.</span><span class="sxs-lookup"><span data-stu-id="e7962-124">The illustration of a home page that appears earlier in this topic includes a video player module.</span></span>
- <span data-ttu-id="e7962-125">**Teksta bloks** — ar saturu papildināta bloka modulis var tikt izmantots, lai attēlotu mājas lapas teksta saturu vienas vai vairāku kolonnu izkārtojumā.</span><span class="sxs-lookup"><span data-stu-id="e7962-125">**Text block** – A content rich block module can be used to present text content on the home page in a single-column or multicolumn layout.</span></span>
- <span data-ttu-id="e7962-126">**Preču ieteikumi** — preču ieteikumu moduļi tiek izmantoti, lai mājas lapā parādītu sarakstus, piemēram, **Jauns**, **Populārs** un **Vispirktākais**.</span><span class="sxs-lookup"><span data-stu-id="e7962-126">**Product recommendations** – Product recommendations modules are used to show lists, such as **New**, **Trending**, and **Best Selling** on the home page.</span></span> <span data-ttu-id="e7962-127">Šie saraksti parāda preces, pamatojoties uz iepirkšanās tendencēm, un tos var ģenerēt algoritmiski vai pārraudzīt manuāli.</span><span class="sxs-lookup"><span data-stu-id="e7962-127">These lists showcase products based on shopping trends, and they can be algorithmically generated or manually curated.</span></span> <span data-ttu-id="e7962-128">Tie palīdz klientiem ātri atklāt labākās preces un tad turpināt iepirkties.</span><span class="sxs-lookup"><span data-stu-id="e7962-128">They help customers quickly discover top products and then continue to shop.</span></span>

    <span data-ttu-id="e7962-129">Tālāk redzamajā attēlā parādīts preču ieteikuma moduļu piemērs mājas lapā.</span><span class="sxs-lookup"><span data-stu-id="e7962-129">The following illustration shows an example of product recommendations modules on a home page.</span></span>

    ![Preču ieteikumu moduļu piemēri](./media/Recommendations.PNG)

> [!NOTE]
> <span data-ttu-id="e7962-131">Visus šeit uzskaitītos moduļus var izmantot jebkurā vietnes lapā.</span><span class="sxs-lookup"><span data-stu-id="e7962-131">All the modules that are listed here can be used on any site page.</span></span> <span data-ttu-id="e7962-132">Tomēr to izvietošana mājas lapā ir svarīga, jo šī lapa ir vieta, kur klienti pirmo reizi mijiedarbojas ar jūsu vietni.</span><span class="sxs-lookup"><span data-stu-id="e7962-132">However, their placement on the home page is important because that page is where customers first interact with your site.</span></span>
