# <a name="add-fields-to-an-excel-workbook-to-edit-retail-transactions"></a>Lauku pievienošana Excel darbgrāmatai, lai rediģētu mazumtirdzniecības darījumus

[!include [banner](../includes/banner.md)]

Šajā tēmā ir aprakstīts, kā pievienot laukus Microsoft Excel darbgrāmatai, lai programmā Microsoft Dynamics 365 Commerce varat rediģēt mazumtirdzniecības darījumus.

## <a name="overview"></a>Pārskats

Ģenerējot Excel failu, lai varētu rediģēt mazumtirdzniecības darījumus, fails tiek aizpildīts ar dažiem noklusējuma laukiem. Ja atjaunināmais lauks ģenerētajā Excel failā nav redzams pēc noklusējuma, varat to pievienot.

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a>Lauku pievienošana darblapai Excel darbgrāmatā

Lai pievienotu laukus Excel darbgrāmatai un varētu rediģēt mazumtirdzniecības darījumus, veiciet tālāk norādītās darbības.

1. Lejupielādējiet un atveriet Excel failu lapā **Pārskati**, lapā **Mazumtirdzniecības darījumi** vai elementā **Darījumu validācijas kļūmes** darbvietā **Veikala finanses**.
1. Atlasiet **Dizains**.
1. Atlasiet vajadzīgās tabulas zīmuļa simbolu un pēc tam pieejamo lauku sarakstā atrodiet un atlasiet pievienojamo lauku.
1. Atlasiet **Pievienot** un pēc tam atlasiet **Atjaunināt**. Laukus varat pārkārtot.
1. Kad atjaunināšana ir pabeigta, atlasiet **Atsvaidzināt**, lai ienestu datus jaunajā kolonnā.

Jaunajam laukam un tā datiem tagad ir jābūt pieejamiem rediģēšanai programmā Excel.
