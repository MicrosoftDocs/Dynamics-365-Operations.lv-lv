# <a name="manage-user-roles-in-microsoft-teams"></a>Lietotāju lomu pārvaldība Microsoft Teams

[!include [banner](includes/banner.md)]

Šajā tēmā aprakstīts, kā pārvaldīt Microsoft Dynamics 365 Commerce lietotāju lomas programmā Microsoft Teams.

Izveidojot komandu katram veikalam vai kanālam brigādes grupā, tiek izveidota grupai atbilstoša dalība grupā (piemēram, `HOUSTON_D365@<YourTenantAzureADDomain>.com`). Visiem veikala darbiniekiem, kas pieder grupas grupai, ir piešķirtas vienas no divām lietotāju lomām: **Īpašnieks** vai **Dalībnieks**. Veikala darbinieki, kuriem ir loma **Īpašnieks**, var veikt darbības, piemēram, pievienot privātu kanālu un pievienot vai dzēst dalībniekus. Parasti veikala vadītājiem ir **Īpašnieka** lietotāja loma.

Šajā attēlā parādīts grupas dalībnieku un viņu lietotāju lomu saraksta piemērs Microsoft Teams administratora centrā.

![Grupas dalībnieki un lietotāju lomas Microsoft Teams administrēšanas centrā.](media/d365-commerce-teams-integration-user-roles.png)

Plašāku informāciju skatiet sadaļā [Komandas īpašnieku un dalībnieku piešķiršana Microsoft Teams](/microsoftteams/assign-roles-permissions).
