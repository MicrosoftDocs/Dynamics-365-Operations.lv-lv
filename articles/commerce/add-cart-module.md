## <a name="cart-module-settings"></a><span data-ttu-id="4f39a-144">Groza moduļa iestatījumi</span><span class="sxs-lookup"><span data-stu-id="4f39a-144">Cart module settings</span></span>

<span data-ttu-id="4f39a-145">Groza moduļiem ir trīs iestatījumi, kurus var konfigurēt.</span><span class="sxs-lookup"><span data-stu-id="4f39a-145">Cart modules have three settings that can be configured:</span></span>
