# <a name="upload-files-other-than-images-and-videos"></a>Failu, kas nav attēli un videoklipi, augšupielāde

[!include [banner](includes/banner.md)]

Šajā rakstā ir aprakstīts, kā augšupielādēt failus, kas nav attēli un videoklipus vietu veidotājā Microsoft Dynamics 365 Commerce.

Commerce vietnes veidotāja multivides bibliotēka atbalsta jebkādu bināro līdzekļu augšupielādi, kas nav attēli vai videoklipi. Piemēram, jūs varētu vēlēties augšupielādēt Microsoft Excel, Microsoft Word, Microsoft PowerPoint vai PDF failus.

Tiek atbalstīti šādi dokumentu veidi:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>Augšupielādēt failu

Lai Commerce vietnes veidotāju augšupielādētu failu, veiciet tālāk norādītās darbības.

1. Kreisās puses navigācijas rūtī atlasiet **Mediju bibliotēka**.
1. Komandu joslā atlasiet **Augšupielādēt \> Augšupielādēt mediju vienumus**.
1. Failu pārlūkā atlasiet vienu vai vairākus failus un pēc tam atlasiet **Atvērt**.
1. Dialoglodziņā **Augšupielādēt multivides vienumu** pēc nepieciešamības ievadiet nosaukumu, aprakstu un atslēgvārdu metadatus.
1. Lai publicētu failus tūlīt pēc augšupielādes, atzīmējiet izvēles rūtiņu **Publicēt multivides vienumus pēc augšupielādes**.
1. Atlasiet **Labi**.
