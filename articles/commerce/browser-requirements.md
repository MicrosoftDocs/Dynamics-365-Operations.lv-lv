# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Dynamics 365 Commerce vietņu pārlūkprogrammas prasības

[!include [banner](includes/banner.md)]

Šajā tēmā izklāstītas pārlūku prasības Microsoft Dynamics 365 Commerce vietnēm.

Dynamics 365 Commerce vietnes vislabāk atbalsta šādas tīmekļa pārlūkprogrammas. Lai atrastu katras tīmekļa pārlūkprogrammas visjaunāko laidienu, dodieties uz programmatūras ražotāja vietni.

- [Microsoft Edge](https://www.microsoft.com/edge) (līdz pašreizējai versijai mīnus 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (līdz pašreizējai versijai mīnus 1)
- [Google Chrome](https://www.google.com/chrome) (līdz pašreizējai versijai mīnus 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (līdz pašreizējai versijai mīnus 1)

> [!NOTE]
> Internet Explorer netiek atbalstīts. Ja lietojat Internet Explorer, dažas funkcijas var nestrādāt, un lietotājiem var būt neatbilstoša pieredze.
