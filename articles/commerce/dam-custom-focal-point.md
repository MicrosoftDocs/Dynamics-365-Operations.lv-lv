# <a name="customize-image-focal-points"></a><span data-ttu-id="b9fb0-103">Attēlu fokusa punktu pielāgošana</span><span class="sxs-lookup"><span data-stu-id="b9fb0-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="b9fb0-104">Šajā tēmā aprakstīts, kā pielāgot attēla fokusa punktus Microsoft Dynamics 365 Commerce vietnes veidotājā.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="b9fb0-105">Kad attēls tiek augšupielādēts Commerce vietnes veidotāja multivides bibliotēkā, sistēma mēģina noteikt attēla fokusa punktu.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="b9fb0-106">Piemēram, ja attēlā ir persona, sistēma pēc noklusējuma iestatīs fokusa punktu uz personas seju.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="b9fb0-107">Vairumā gadījumu automātiski tiek iestatīts fokusa punkts visiem skatpunktiem, bet dažkārt var būt nepieciešams pielāgot fokusa punktu, lai nodrošinātu, ka noteikta attēla daļa vienmēr ir redzama.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="b9fb0-108">Definēt pielāgotu attēla fokusa punktu</span><span class="sxs-lookup"><span data-stu-id="b9fb0-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="b9fb0-109">Lai definētu pielāgotu attēla fokusa punktu, veiciet šīs darbības.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="b9fb0-110">Commerce vietnes veidotāja kreisajā navigācijas rūtī atlasiet **Multivides bibliotēka**.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="b9fb0-111">Galvenajā logā atlasiet attēlu, kuru vēlaties pārveidot.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="b9fb0-112">Komandjoslā atlasiet **Rediģēt**.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="b9fb0-113">Atlasiet attēlu, kuru vēlaties ievadīt **Rediģēšanas režīmā**.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="b9fb0-114">Sadaļā **Rediģēšanas režīms** atlasiet **Mainīt fokusa punktu**.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="b9fb0-115">Virs attēla parādās riņķveida fokusa punktu vadīkla.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="b9fb0-116">Atlasiet fokusa punktu vadīklu, lai pārvietotu to uz vēlamo fokusa punktu.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="b9fb0-117">Kad esat pabeidzis, komandjoslā atlasiet **Saglabāt** un pēc tam atlasiet **Pabeigt rediģēšanu**.</span><span class="sxs-lookup"><span data-stu-id="b9fb0-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
