[!include [banner](../includes/banner.md)]

<span data-ttu-id="d7aee-104">Šajā rakstā ir paskaidroti dažādie veidi, kādā varat skatīt žurnālu ierakstus un transakcijas.</span><span class="sxs-lookup"><span data-stu-id="d7aee-104">This article explains the various ways that you can view journal entries and transactions.</span></span>

## <a name="voucher-transactions"></a><span data-ttu-id="d7aee-107">Dokumentu transakcijas</span><span class="sxs-lookup"><span data-stu-id="d7aee-107">Voucher transactions</span></span>
<span data-ttu-id="d7aee-108">**Dokumentu transakcijas** ir uzziņu lapa, kurā varat atlasīt dažādas tabulas un laukus, lai norādītu kritērijus meklējamai bilancei vai transakcijai.</span><span class="sxs-lookup"><span data-stu-id="d7aee-108">**Voucher transactions** is an inquiry page where you can select from various tables and fields to specify criteria for the balance or transaction that you're searching for.</span></span> <span data-ttu-id="d7aee-109">Piemēram, varat skatīt visas transakcijas noteiktā datumā vai kontā vai visas transakcijas ar veidu **Lietošanas**, kas atrodas noteiktā grāmatošanas līmenī.</span><span class="sxs-lookup"><span data-stu-id="d7aee-109">For example, you can view all transactions for a specific date or account, or all transactions of the **Operating** type that are in a specific posting layer.</span></span> <span data-ttu-id="d7aee-110">Pēc noklusējuma lapā tiek rādīts žurnāla numurs, dokuments, datums un galvenais konts, bet varat pievienot papildu tabulas, laukus un kritērijus, lai sašaurinātu meklēšanas rezultātus.</span><span class="sxs-lookup"><span data-stu-id="d7aee-110">By default, the page shows the journal number, voucher, date, and main account, but you can add additional tables, fields, and criteria to narrow down your search.</span></span>

## <a name="audit-trail"></a><span data-ttu-id="d7aee-111">Auditācijas pieraksti</span><span class="sxs-lookup"><span data-stu-id="d7aee-111">Audit trail</span></span>
<span data-ttu-id="d7aee-112">**Auditācijas pieraksti** ir uzziņas lapa, kur tiek rādītītas dažāda veida transakcijas, apraksti, transakciju autors un izveides laiks.</span><span class="sxs-lookup"><span data-stu-id="d7aee-112">**Audit trail** is an inquiry page that shows the types of transactions, descriptions, who the transactions were created by, and when they were created.</span></span> <span data-ttu-id="d7aee-113">Uzziņu lapā **Auditācijas pieraksti** varat skatīt dokumentu transakcijas.</span><span class="sxs-lookup"><span data-stu-id="d7aee-113">From the **Audit trail** inquiry page, you can view the voucher transactions.</span></span>

## <a name="financial-reports"></a><span data-ttu-id="d7aee-114">Finanšu pārskati</span><span class="sxs-lookup"><span data-stu-id="d7aee-114">Financial reports</span></span>
