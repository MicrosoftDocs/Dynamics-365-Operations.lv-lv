# <a name="close-the-fiscal-year"></a><span data-ttu-id="af4c2-103">Finanšu gada slēgšana</span><span class="sxs-lookup"><span data-stu-id="af4c2-103">Close the fiscal year</span></span>

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="af4c2-104">Šajā procedūrā tiek aprakstīts gada beigu slēgšanas process, kas pārsūta bilances uz jaunu finanšu gadu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-104">This procedure steps through the year end close process that transfers balances to a new fiscal year.</span></span>


## <a name="validate-year-end-close-parameters"></a><span data-ttu-id="af4c2-105">Apstiprināt gada beigu slēgšanas parametrus</span><span class="sxs-lookup"><span data-stu-id="af4c2-105">Validate year-end close parameters</span></span>
1. <span data-ttu-id="af4c2-106">Dodieties uz Virsgrāmata > Virsgrāmatas iestatīšana > Virsgrāmatas parametri.</span><span class="sxs-lookup"><span data-stu-id="af4c2-106">Go to General ledger > Ledger setup > General ledger parameters.</span></span>
2. <span data-ttu-id="af4c2-107">Izvērsiet sadaļu Finanšu gada slēgšana.</span><span class="sxs-lookup"><span data-stu-id="af4c2-107">Expand the Fiscal year close section.</span></span>
3. <span data-ttu-id="af4c2-108">Atlasiet Jā vai Nē opcijai gada slēgšanas darbību dzēšana pārsūtīšanas laikā.</span><span class="sxs-lookup"><span data-stu-id="af4c2-108">Select Yes or No for the option Delete close-of-year transactions during transfer.</span></span>
    * <span data-ttu-id="af4c2-109">Ja finanšu gads jau ir slēgts, un gada beigu slēgšana tiek palaista vēlreiz, šis iestatījums ir svarīgs.</span><span class="sxs-lookup"><span data-stu-id="af4c2-109">If the fiscal year has already been closed and the year-end close is being run again, this setting is important.</span></span> <span data-ttu-id="af4c2-110">Ja iestatīts uz Jā, dokuments par iepriekšējā gada beigu slēgšanu tiks dzēsts, un tiks izveidots jauns dokuments visu kontu sākuma bilancēm.</span><span class="sxs-lookup"><span data-stu-id="af4c2-110">If set to Yes, the voucher for the previous year-end close will be deleted, and a new voucher will be created for all accounts beginning balances.</span></span> <span data-ttu-id="af4c2-111">Ja iestatīts uz Nē, iepriekšējais dokuments tiks saglabāts un tiks izveidots jauns dokuments tikai ierakstu pielāgošanai, kas tika grāmatoti pēc iepriekšējā gada beigu slēgšanas.</span><span class="sxs-lookup"><span data-stu-id="af4c2-111">If set to No, the previous voucher will remain and a new voucher will only be created for adjusting entries that were posted after the last year-end close.</span></span>  
4. <span data-ttu-id="af4c2-112">Atlasiet Jā vai Nē, vai veidot slēgšanas darbības pārsūtīšanas laikā.</span><span class="sxs-lookup"><span data-stu-id="af4c2-112">Select Yes or No for whether to Create closing transactions during transfer.</span></span>
    * <span data-ttu-id="af4c2-113">Ja iestatīts uz Jā, tiek izveidotas divas darbības.</span><span class="sxs-lookup"><span data-stu-id="af4c2-113">If set to Yes, two transactions are created.</span></span> <span data-ttu-id="af4c2-114">Viens dokuments ir izveidots finanšu gadā, kas tiek slēgts, lai pārnestu bilances uz PL virsgrāmatas kontiem, līdz nullei, un nākamajā finanšu gada tiek izveidots otrs dokuments sākuma bilancēm.</span><span class="sxs-lookup"><span data-stu-id="af4c2-114">One voucher is created in the fiscal year being closed to bring the balances of the P&L ledger accounts down to zero and a second voucher is created in the next fiscal year for the beginning balances.</span></span> <span data-ttu-id="af4c2-115">Ja iestatīts uz Nē, tiek izveidots viens dokuments nākamā finanšu gada sākuma bilancēm.</span><span class="sxs-lookup"><span data-stu-id="af4c2-115">If set to No, a single voucher is created in the next fiscal year for the beginning balances.</span></span>  
5. <span data-ttu-id="af4c2-116">Atlasiet Jā vai Nē, vai iestatīt finanšu gada statusu kā neatgriezeniski slēgtu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-116">Select Yes or No for whether to Set fiscal year status to permanently closed.</span></span>
    * <span data-ttu-id="af4c2-117">Ja iestatīts uz Jā, finanšu gada statuss tiek iestatīts uz Neatgriezeniski slēgts.</span><span class="sxs-lookup"><span data-stu-id="af4c2-117">If set to Yes, the fiscal year status will be set to Permanently closed.</span></span>  <span data-ttu-id="af4c2-118">Tā kā neatgriezeniski slēgtu gadu nevar atvērt atkārtoti, vislabāk ir iestatīt šo opciju uz Nē.</span><span class="sxs-lookup"><span data-stu-id="af4c2-118">Because a permanently closed year cannot be reopened, it would be a best practice to set this option to No.</span></span>  
6. <span data-ttu-id="af4c2-119">Atlasiet Jā vai Nē, vai dokumenta numurs ir jānorāda gada beigu slēgšanas laikā.</span><span class="sxs-lookup"><span data-stu-id="af4c2-119">Select Yes or No for whether a Voucher number must be filled in during the year end close.</span></span>
    * <span data-ttu-id="af4c2-120">Ja iestatīts uz Jā, Gada beigu slēgšanas procesa laikā, dokumenta numurs ir jāievada manuāli.</span><span class="sxs-lookup"><span data-stu-id="af4c2-120">If set to Yes, a voucher number must be manually entered during the year end close process.</span></span> <span data-ttu-id="af4c2-121">Numuru sērija netiek izmantota, lai ģenerētu šo dokumenta numuru.</span><span class="sxs-lookup"><span data-stu-id="af4c2-121">A number sequence is not used to generate this voucher number.</span></span> <span data-ttu-id="af4c2-122">Vislabāk ir iestatīt to uz Jā.</span><span class="sxs-lookup"><span data-stu-id="af4c2-122">It's a best practice to set this to Yes.</span></span>  
7. <span data-ttu-id="af4c2-123">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-123">Close the page.</span></span>
8. <span data-ttu-id="af4c2-124">Pārejiet uz sadaļu Virsgrāmata > Perioda noslēgšana > Gada beigu slēgšana.</span><span class="sxs-lookup"><span data-stu-id="af4c2-124">Go to General ledger > Period close > Year end close.</span></span>
9. <span data-ttu-id="af4c2-125">Noklikšķiniet uz Jauns, lai izveidotu gada beigu slēgšanas veidni.</span><span class="sxs-lookup"><span data-stu-id="af4c2-125">Click New to create a year-end close template.</span></span>
    * <span data-ttu-id="af4c2-126">Veidni var izveidot juridisku personu grupai, kurai vēlaties palaist gada beigu slēgšanu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-126">A template can be created for a group of legal entities for which to run the year-end close.</span></span> <span data-ttu-id="af4c2-127">Šo veidni var izmantot atkārtoti katru gadu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-127">This template can be reused year after year.</span></span>  
10. <span data-ttu-id="af4c2-128">Grupas nosaukuma laukā, ievadiet gada beigu slēgšanas veidnes nosaukumu..</span><span class="sxs-lookup"><span data-stu-id="af4c2-128">In the Group name field, Enter a year-end close template name..</span></span>
11. <span data-ttu-id="af4c2-129">Atlasiet finanšu gadu, kuram tiks izveidota veidne.</span><span class="sxs-lookup"><span data-stu-id="af4c2-129">Select the fiscal year for which the template will be created.</span></span>
    * <span data-ttu-id="af4c2-130">Tikai juridiskas personas, kas izmanto vienā finanšu gadu, var sagrupēt gada beigu slēgšanas veidnē.</span><span class="sxs-lookup"><span data-stu-id="af4c2-130">Only legal entities which use the same fiscal year can be grouped together in the year-end close template.</span></span> <span data-ttu-id="af4c2-131">Tas ir tāpēc, ka gada beigu slēgšana tiek veikta, izvēloties finanšu gadu, nevis datumu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-131">This is because the year end close is done by selecting a fiscal year, not a date.</span></span>  
12. <span data-ttu-id="af4c2-132">Lietojiet šo saīsni ieraksta saglabāšanai.</span><span class="sxs-lookup"><span data-stu-id="af4c2-132">Use the shortcut for saving a record.</span></span>

    * <span data-ttu-id="af4c2-134">Juridiskās personas var pievienot, atlasot juridiskās personas vai atlasot organizācijas hierarhiju.</span><span class="sxs-lookup"><span data-stu-id="af4c2-134">Legal entities can be added by either selecting the legal entities or by selecting an organizational hierarchy.</span></span>  <span data-ttu-id="af4c2-135">Tiks pievienotas tikai juridiskās personas ar organizācijas hierarhiju, ar atlasītu to pašu finanšu kalendāru.</span><span class="sxs-lookup"><span data-stu-id="af4c2-135">Only legal entities with the organizational hierarchy with the same fiscal calendar selected will be added.</span></span>  
14. <span data-ttu-id="af4c2-136">Atlasiet juridiskās personas vai organizācijas hierarhiju.</span><span class="sxs-lookup"><span data-stu-id="af4c2-136">Select either the legal entities or the organizational hierarchy.</span></span>
15. <span data-ttu-id="af4c2-137">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="af4c2-137">Click OK.</span></span>
16. <span data-ttu-id="af4c2-138">Atlasiet vai finanšu dimensijas tiks pārnestas uz nākamo finanšu gadu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-138">Select whether the financial dimensions will transfer to the next fiscal year.</span></span>
    * <span data-ttu-id="af4c2-139">Vislabākais ir iestatīt šo opciju Bilances kontiem uz Jā.</span><span class="sxs-lookup"><span data-stu-id="af4c2-139">It's a best practice to set this option to Yes for Balance sheet accounts.</span></span>  <span data-ttu-id="af4c2-140">Tas saglabās grāmatoto darbību finanšu dimensijas, veidojot bilances kontu sākuma atlikumus.</span><span class="sxs-lookup"><span data-stu-id="af4c2-140">This will maintain the financial dimensions on posted transactions when creating the beginning balances for the balance sheet accounts.</span></span>  <span data-ttu-id="af4c2-141">Peļņas un zaudējumu kontos jūs varat uzturēt finanšu dimensijas (Slēgt visu), kad bilances tiek pārvietotas uz Nesadalīto peļņu, vai jūs varat aizstāt finanšu dimensijas ar citu dimensijas vērtību (Slēgt vienu).</span><span class="sxs-lookup"><span data-stu-id="af4c2-141">For profit and loss accounts, you can select to maintain the financial dimensions (Close all) when the balances are moved to Retained earnings or you can select to have the financial dimensions replaced with a different dimension value (Close single).</span></span> <span data-ttu-id="af4c2-142">Atlasot opciju Slēgt vienu, jūs varat definēt konkrētu dimensijas vērtību katrai dimensijai vai pat atstāt to tukšu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-142">If you choose Close single, you can define a specific dimension value for each dimension or even choose to leave it blank.</span></span>  
17. <span data-ttu-id="af4c2-143">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="af4c2-143">Click Save.</span></span>
18. <span data-ttu-id="af4c2-144">Sāciet gada beigu slēgšanu, Darbību rūtī, atlasot palaist finanšu slēgšanu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-144">Start the year-end close by choosing Run fiscal close on the Action Pane.</span></span>
    * <span data-ttu-id="af4c2-145">Gada beigu slēgšana tiks veikta atlasītajai veidnei.</span><span class="sxs-lookup"><span data-stu-id="af4c2-145">The year-end close will be run for the selected template.</span></span>

    * <span data-ttu-id="af4c2-147">Pirmo reizi palaižot gada beigu slēgšanu, lai iegūtu sākuma bilances daļa organizāciju var izvēlēties veikt gada beigu slēgšanas visām juridiskajām personām veidnē.</span><span class="sxs-lookup"><span data-stu-id="af4c2-147">When first running the year-end close, to get beginning balances most organizations may choose to run the year-end close for all legal entities within the template.</span></span> <span data-ttu-id="af4c2-148">Ja pēc tam tiek veikti pielāgošanas ieraksti, jūs varat palaist gada beigu slēgšanu tikai juridiskām personām, kurām ir korekcijas.</span><span class="sxs-lookup"><span data-stu-id="af4c2-148">If adjusting entries are made after that, you may choose to run the year-end close for only the legal entities that have adjustments.</span></span>  
20. <span data-ttu-id="af4c2-149">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="af4c2-149">Click OK.</span></span>
21. <span data-ttu-id="af4c2-150">Atlasiet finanšu gadu, kuram vēlaties palaist gada beigu slēgšanu.</span><span class="sxs-lookup"><span data-stu-id="af4c2-150">Select the fiscal year for which to run the year-end close.</span></span>
22. <span data-ttu-id="af4c2-151">Laukā Dokuments ievadiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="af4c2-151">In the Voucher field, type a value.</span></span>
    * <span data-ttu-id="af4c2-152">Tas ir vislabākais veids, lai iekļautu finanšu gada dokumenta numuru, lai būtu vieglāk atrast gada beigu slēgšanas dokumentu, kas tiek izveidots.</span><span class="sxs-lookup"><span data-stu-id="af4c2-152">It's a best practice to include the fiscal year in the voucher number, to make it easier to find the year-end close voucher that is created.</span></span>

24. <span data-ttu-id="af4c2-156">Noklikšķiniet uz OK.</span><span class="sxs-lookup"><span data-stu-id="af4c2-156">Click OK.</span></span>
