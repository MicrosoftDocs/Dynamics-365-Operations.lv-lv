# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="51f0e-103">Žurnālu ierakstu vai transakciju skatīšana</span><span class="sxs-lookup"><span data-stu-id="51f0e-103">View journal entries or transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="51f0e-104">Šajā procedūrā parādīts, kā izmantot Dokumentu darbību vaicājumu, lai meklētu žurnālu ierakstus vai darbības.</span><span class="sxs-lookup"><span data-stu-id="51f0e-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="51f0e-105">Pārejiet uz sadaļu Virsgrāmata > Pieprasījumi un pārskati > Dokumentu transakcijas.</span><span class="sxs-lookup"><span data-stu-id="51f0e-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="51f0e-106">Atlasiet lauku, kuram vēlaties definēt filtra kritērijus.</span><span class="sxs-lookup"><span data-stu-id="51f0e-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="51f0e-107">Ievadiet jūsu filtra kritērijus atlasītajam laukam.</span><span class="sxs-lookup"><span data-stu-id="51f0e-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="51f0e-108">Varat veikt filtrēšanu, izmantojot vienu vērtību vai diapazonu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="51f0e-109">Definējot diapazonu, pārliecinieties, ka tiek lietota pareiza sintakse.</span><span class="sxs-lookup"><span data-stu-id="51f0e-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="51f0e-110">Vērtības jāatdala ar dubultu punktu (..).</span><span class="sxs-lookup"><span data-stu-id="51f0e-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="51f0e-111">Noklikšķiniet cilni Savienojumi, lai pievienotu papildu tabulas, kuras vēlaties filtrēt.</span><span class="sxs-lookup"><span data-stu-id="51f0e-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="51f0e-112">Kokā atlasiet 'Tabulas\Virsgrāmatas žurnāla ieraksts'.</span><span class="sxs-lookup"><span data-stu-id="51f0e-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="51f0e-113">Noklikšķiniet Pievienot savienojuma tabulu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-113">Click Add table join.</span></span>
7. <span data-ttu-id="51f0e-114">Noklikšķiniet uz Atcelt, ja nevēlaties pievienot papildu tabulu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="51f0e-115">Noklikšķiniet uz cilnes Diapazons.</span><span class="sxs-lookup"><span data-stu-id="51f0e-115">Click the Range tab.</span></span>
9. <span data-ttu-id="51f0e-116">Lai palaistu vaicājumu, noklikšķiniet uz Labi.</span><span class="sxs-lookup"><span data-stu-id="51f0e-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="51f0e-117">Noklikšķiniet uz Transakcijas izcelsme.</span><span class="sxs-lookup"><span data-stu-id="51f0e-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="51f0e-118">Dažādas pogas par režģi var izmantot, lai izpētītu papildu informāciju par atlasīto dokumenta ierakstu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="51f0e-119">Dažas pogas var nebūt pieejamas, atkarībā no darījuma veida un darbības īpašībām.</span><span class="sxs-lookup"><span data-stu-id="51f0e-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="51f0e-120">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-120">Close the page.</span></span>
12. <span data-ttu-id="51f0e-121">Noklikšķiniet uz Oriģinālais dokuments.</span><span class="sxs-lookup"><span data-stu-id="51f0e-121">Click Original document.</span></span>
13. <span data-ttu-id="51f0e-122">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="51f0e-122">Close the page.</span></span>
