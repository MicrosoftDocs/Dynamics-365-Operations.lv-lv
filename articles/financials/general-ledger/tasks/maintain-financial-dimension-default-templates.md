# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="be0ea-103">Uzturēt finanšu dimensijas noklusējuma veidnes</span><span class="sxs-lookup"><span data-stu-id="be0ea-103">Maintain financial dimension default templates</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="be0ea-104">Šajā procedūrā parādīts, kā iestatīt finanšu dimensijas noklusējuma veidnes.</span><span class="sxs-lookup"><span data-stu-id="be0ea-104">This procedure shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="be0ea-105">Veidnes var izmantot, lai ātri sadalītu summas no pirmdokumentiem.</span><span class="sxs-lookup"><span data-stu-id="be0ea-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="be0ea-106">Pārejiet uz sadaļu Virsgrāmata > Kontu plāns > Dimensijas > Finanšu dimensijas noklusējuma veidnes.</span><span class="sxs-lookup"><span data-stu-id="be0ea-106">Go to General ledger > Chart of accounts > Dimensions > Financial dimension default templates.</span></span>
2. <span data-ttu-id="be0ea-107">Noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="be0ea-107">Click New.</span></span>
3. <span data-ttu-id="be0ea-108">Laukā Veidnes ID, ievadiet veidnes ID.</span><span class="sxs-lookup"><span data-stu-id="be0ea-108">In the Template ID field, enter a Template ID..</span></span>
4. <span data-ttu-id="be0ea-109">Ievadiet procentuālo vērtību pirmās dimensijas vērtībām.</span><span class="sxs-lookup"><span data-stu-id="be0ea-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="be0ea-110">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="be0ea-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="be0ea-111">Noklikšķiniet uz Pievienot.</span><span class="sxs-lookup"><span data-stu-id="be0ea-111">Click Add.</span></span>
7. <span data-ttu-id="be0ea-112">Ievadiet citus procentus nākamajām dimensiju vērtībām.</span><span class="sxs-lookup"><span data-stu-id="be0ea-112">Enter another percent for the next dimension values.</span></span>
    * <span data-ttu-id="be0ea-113">Šajā piemērā ievadiet procentuālo vērtību, tāpēc jums vēl joprojām ir piešķirti mazāk nekā 100 %.</span><span class="sxs-lookup"><span data-stu-id="be0ea-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span>  
8. <span data-ttu-id="be0ea-114">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="be0ea-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="be0ea-115">Noklikšķiniet uz Pievienot.</span><span class="sxs-lookup"><span data-stu-id="be0ea-115">Click Add.</span></span>
10. <span data-ttu-id="be0ea-116">Ievadiet citus procentus, kas padarīs kopējo procentuālo vērtību vienādu ar 100%.</span><span class="sxs-lookup"><span data-stu-id="be0ea-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="be0ea-117">Ievadiet finanšu dimensijas vērtības, kas atbilst ievadītajiem procentiem.</span><span class="sxs-lookup"><span data-stu-id="be0ea-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="be0ea-118">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="be0ea-118">Click Save.</span></span>
