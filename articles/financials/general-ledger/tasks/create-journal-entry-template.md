1. <span data-ttu-id="cb614-106">Ejiet uz **Navigācijas rūts > Moduļi > Virsgrāmata > Žurnāla ieraksti > Vispārējie žurnāli**.</span><span class="sxs-lookup"><span data-stu-id="cb614-106">Go to **Navigation pane > Modules > General ledger > Journal entries > General journals**.</span></span>

11. <span data-ttu-id="cb614-117">Laukā **Konta veids** ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="cb614-117">In the **Account type** field, type a value.</span></span>
12. <span data-ttu-id="cb614-118">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="cb614-118">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="cb614-119">Laukā **Debets** ievadiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="cb614-119">In the **Debit** field, type a value.</span></span>

    - <span data-ttu-id="cb614-129">Procenti: summas vaučerī tiek pārveidotas par procentu koeficientiem, kas ļauj piemērot jebkuru summu, kad atlasīta vaučera veidne.</span><span class="sxs-lookup"><span data-stu-id="cb614-129">Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>
    - <span data-ttu-id="cb614-130">Summa: tiks saglabātas un piemērotas faktiskās summas.</span><span class="sxs-lookup"><span data-stu-id="cb614-130">Amount: The actual amounts will be stored and applied.</span></span>

22. <span data-ttu-id="cb614-132">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="cb614-132">Click **New**.</span></span>
23. <span data-ttu-id="cb614-133">Laukā **Nosaukums** noklikšķiniet uz nolaižamā saraksta pogas, lai atvērtu pārlūku.</span><span class="sxs-lookup"><span data-stu-id="cb614-133">In the **Name** field, click the drop-down button to open the lookup.</span></span>

26. <span data-ttu-id="cb614-136">Noklikšķiniet uz **Funkcijas**.</span><span class="sxs-lookup"><span data-stu-id="cb614-136">Click **Functions**.</span></span>
