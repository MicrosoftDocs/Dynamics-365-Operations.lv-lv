# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="3d89b-103">Virsgrāmatas grāmatošanas grupu izveide pievienotās vērtības nodoklim</span><span class="sxs-lookup"><span data-stu-id="3d89b-103">Set up Ledger posting groups for sales tax</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="3d89b-104">PVN tiek aprēķināts un iegrāmatots galvenajos kontos, kas ir norādīti Virsgrāmatas grāmatošanas grupās.</span><span class="sxs-lookup"><span data-stu-id="3d89b-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="3d89b-105">Virsgrāmatas grāmatošanas grupas ir piesaistītas katram PVN kodam.</span><span class="sxs-lookup"><span data-stu-id="3d89b-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="3d89b-106">Varat iestatīt atsevišķas Virsgrāmatas grāmatošanas grupas katram PVN kodam, izmantot vienu virsgrāmatas grāmatošanas grupu visiem PVN kodiem vai PVN kodiem piešķirt vairākas Virsgrāmatas grāmatošanas grupas.</span><span class="sxs-lookup"><span data-stu-id="3d89b-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="3d89b-107">Šis paraugs izmanto DEMF demonstrācijas uzņēmumu.</span><span class="sxs-lookup"><span data-stu-id="3d89b-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="3d89b-108">Pārejiet uz **Navigācijas rūts > Moduļi > Nodoklis > Iestatīšana > PVN > Virsgrāmatas grāmatošanas grupas**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="3d89b-109">Klikšķiniet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-109">Click **New**.</span></span>
3. <span data-ttu-id="3d89b-110">Laukā **Virsgrāmatas grāmatošanas grupas** ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="3d89b-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="3d89b-111">Laukā **Apraksts** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="3d89b-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="3d89b-112">Laukā **Maksājamais PVN** izvēlieties galveno kontu izejošajiem PVN, kas jāmaksā nodokļu administrācijai.</span><span class="sxs-lookup"><span data-stu-id="3d89b-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="3d89b-113">PVN tiek iekasēts nodokļu iestādes vietā, ja pārdodat ar nodokli apliekamas preces un pakalpojumus.</span><span class="sxs-lookup"><span data-stu-id="3d89b-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="3d89b-114">Laukā **Saņemamais PVN** izvēlieties galveno kontu ienākošajiem PVN, kas jāsaņem no nodokļu administrācijas.</span><span class="sxs-lookup"><span data-stu-id="3d89b-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="3d89b-115">Kreditori iekasē nodokli nodokļu iestādes vārdā, ja pērkat ar nodokli apliekamas preces un pakalpojumus.</span><span class="sxs-lookup"><span data-stu-id="3d89b-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="3d89b-116">Šis lauks nav pieejams, ja lapā **Virsgrāmatas parametri** ir atlasīta opcija Piemērot PVN nodokļu sistēmas kārtulas.</span><span class="sxs-lookup"><span data-stu-id="3d89b-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="3d89b-117">Tā vietā PVN, kas tiek maksāts kreditoriem, tiek debetēts tajā pašā kontā, kurā pirkums.</span><span class="sxs-lookup"><span data-stu-id="3d89b-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="3d89b-118">Laukā **Izmantošanas nodokļa izdevumi** atlasiet galveno kontu atskaitāmo izmantošanas nodokļu grāmatošanai, kurus pārdevēji nepieprasa vai par kuriem nodokļu administrācijai nav ziņots kā par daļu no ES apgrieztās iekasēšanas GST/HST.</span><span class="sxs-lookup"><span data-stu-id="3d89b-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="3d89b-119">Opcijai **Izmantošanas nodoklis** ir jābūt atlasītai, lai izmantotu lauku **PVN kods** sadaļā **PVN grupa**, kas tiek izmantota darījumā.</span><span class="sxs-lookup"><span data-stu-id="3d89b-119">The **Use tax** option needs to be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="3d89b-120">Šis lauks nav pieejams, ja lapā **Virsgrāmatas parametri** ir atlasīta opcija **Piemērot PVN nodokļu sistēmas kārtulas**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-120">This field is not be available if the **Apply sales tax taxation rules** option is selected in the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="3d89b-121">Laukā **Maksājamais izmantošanas nodoklis** izvēlieties galveno kontu ienākošo izmantošanas nodokļu grāmatošanai, kuri ir jāmaksā nodokļu administrācijai.</span><span class="sxs-lookup"><span data-stu-id="3d89b-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="3d89b-122">Opcijai **Izmantošanas nodoklis** ir jābūt atlasītai, lai izmantotu lauku **PVN kods** sadaļā **PVN grupa**, lai grāmatotu **Izmantošanas nodokli**</span><span class="sxs-lookup"><span data-stu-id="3d89b-122">The **Use tax** option needs to be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="3d89b-123">Ja opcija **Piemērot PVN nodokļu sistēmas kārtulas** ir atlasīta lapā **Virsgrāmatas parametri** nobīde tiek grāmatota darījuma izdevumu kontā.</span><span class="sxs-lookup"><span data-stu-id="3d89b-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction’s expense account.</span></span>   
9. <span data-ttu-id="3d89b-124">Laukā **Norēķinu konts** atlasiet galveno kontu, kas ir virsgrāmatu kontu neto atlikums, kas norādīts laukos **Maksājamais izmantošanas nodoklis** un **Saņemamais PVN**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="3d89b-125">Atlikums tiks izveidots, kad tiks nokārtots PVN un grāmatošanas uzdevums izpildīts.</span><span class="sxs-lookup"><span data-stu-id="3d89b-125">The balance will be created when the sales tax settle and post job is ran.</span></span>  <span data-ttu-id="3d89b-126">Ja nodokļu administrācija norēķinu periodā ir saistīta ar pārdevēja kontu, atlikums tiek nosūtīts uz pārdevēja kontu.</span><span class="sxs-lookup"><span data-stu-id="3d89b-126">If the tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>
10. <span data-ttu-id="3d89b-127">Laukā **Pārdevēja skaidras naudas atlaide** atlasiet galveno kontu, lai grāmatotu skaidras naudas atlaidi PVN kodiem, kas saistīti ar šīs virsgrāmatas grāmatošanas grupu.</span><span class="sxs-lookup"><span data-stu-id="3d89b-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="3d89b-128">Šī ir izvēles opcija, un, ja netiek ievadīts neviens konts, opcijai **Skaidras naudas atlaižu kodi** tiks izmantots galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="3d89b-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="3d89b-129">Var būt noderīgi katrai **Virsgrāmatas grāmatošanas grupai** izmantot citu kontu, ja tiek izmantots apgrieztais PVN skaidras naudas atlaides opcijai PVN grupās.</span><span class="sxs-lookup"><span data-stu-id="3d89b-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="3d89b-130">Laukā **Klienta pieteikuma atlaide** atlasiet galveno kontu, lai grāmatotu skaidras naudas atlaidi **PVN kodiem**, kas saistīti ar šīs **Virsgrāmatas grāmatošanas grupu**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="3d89b-131">Šī ir izvēles opcija, un, ja netiek ievadīts neviens konts, opcijai **Skaidras naudas atlaižu kodi** tiks izmantots galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="3d89b-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="3d89b-132">Var būt noderīgi katrai **Virsgrāmatas grāmatošanas grupai** izmantot citu kontu, ja tiek izmantots apgrieztais PVN skaidras naudas atlaides opcijai **PVN grupās**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="3d89b-133">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="3d89b-133">Click **Save**.</span></span>
