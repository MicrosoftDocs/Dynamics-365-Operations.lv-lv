# <a name="set-up-ledger-posting-groups-for-sales-tax"></a><span data-ttu-id="ce061-103">Virsgrāmatas PVN grāmatošanas grupu iestatīšana</span><span class="sxs-lookup"><span data-stu-id="ce061-103">Set up ledger posting groups for sales tax</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ce061-104">PVN tiek aprēķināts un iegrāmatots galvenajos kontos, kas ir norādīti Virsgrāmatas grāmatošanas grupās.</span><span class="sxs-lookup"><span data-stu-id="ce061-104">Sales tax is calculated and posted to main accounts that are specified in the Ledger posting groups.</span></span> <span data-ttu-id="ce061-105">Virsgrāmatas grāmatošanas grupas ir piesaistītas katram PVN kodam.</span><span class="sxs-lookup"><span data-stu-id="ce061-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="ce061-106">Varat iestatīt atsevišķas Virsgrāmatas grāmatošanas grupas katram PVN kodam, izmantot vienu virsgrāmatas grāmatošanas grupu visiem PVN kodiem vai PVN kodiem piešķirt vairākas Virsgrāmatas grāmatošanas grupas.</span><span class="sxs-lookup"><span data-stu-id="ce061-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="ce061-107">Šis paraugs izmanto DEMF demonstrācijas uzņēmumu.</span><span class="sxs-lookup"><span data-stu-id="ce061-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="ce061-108">Pārejiet uz sadaļu Nodokļi > Iestatījumi > PVN > Virsgrāmatas grāmatošanas grupas.</span><span class="sxs-lookup"><span data-stu-id="ce061-108">Go to Tax > Setup > Sales tax > Ledger posting groups.</span></span>
2. <span data-ttu-id="ce061-109">Noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="ce061-109">Click New.</span></span>
3. <span data-ttu-id="ce061-110">Ierakstiet vērtību laukā Virsgrāmatas grāmatošanas grupa.</span><span class="sxs-lookup"><span data-stu-id="ce061-110">In the Ledger posting group field, type a value.</span></span>
4. <span data-ttu-id="ce061-111">Apraksta laukā ierakstiet vērtību.</span><span class="sxs-lookup"><span data-stu-id="ce061-111">In the Description field, type a value.</span></span>
5. <span data-ttu-id="ce061-112">Laukā Maksājamais PVN atlasiet galveno kontu izejošiem PVN, kas ir jāmaksā nodokļu iestādei.</span><span class="sxs-lookup"><span data-stu-id="ce061-112">In the Sales tax payable field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span>
    * <span data-ttu-id="ce061-113">PVN tiek iekasēts nodokļu iestādes vietā, ja pārdodat ar nodokli apliekamas preces un pakalpojumus.</span><span class="sxs-lookup"><span data-stu-id="ce061-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="ce061-114">Laukā Saņemtais PVN atlasiet galveno kontu ienākošajiem nodokļiem, kas ir saņemti no nodokļu iestādes.</span><span class="sxs-lookup"><span data-stu-id="ce061-114">In the Sales tax receivable filed, select the main account for incoming taxes that are received from the tax authority.</span></span>
    * <span data-ttu-id="ce061-115">Kreditori iekasē nodokli nodokļu iestādes vārdā, ja pērkat ar nodokli apliekamas preces un pakalpojumus.</span><span class="sxs-lookup"><span data-stu-id="ce061-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="ce061-116">Šis lauks nav pieejams, ja lapā Virsgrāmatas parametri ir atlasīta opcija Lietot PVN nodokļu nosacījumus.</span><span class="sxs-lookup"><span data-stu-id="ce061-116">This field is not available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span> <span data-ttu-id="ce061-117">Tā vietā PVN, kas tiek maksāts kreditoriem, tiek debetēts tajā pašā kontā, kurā pirkums.</span><span class="sxs-lookup"><span data-stu-id="ce061-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="ce061-118">Laukā Importa nodokļa izdevumi atlasiet galveno kontu, lai grāmatotu ieturamos importa nodokļus, kurus kreditori nav paziņojuši vai snieguši nodokļu iestādei kā daļu no ES atgriezeniskā GST/HST.</span><span class="sxs-lookup"><span data-stu-id="ce061-118">In the Use tax expense field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span>
    * <span data-ttu-id="ce061-119">Transakcijā izmantotajam PVN grupas PVN kodam jāatlasa opcija Importa nodoklis.</span><span class="sxs-lookup"><span data-stu-id="ce061-119">The Use tax option needs to be selected for the Sales tax code in the Sales tax group that is used in the transaction.</span></span>  <span data-ttu-id="ce061-120">Šis lauks nav pieejams, ja lapā Virsgrāmatas parametri ir atlasīta opcija Lietot PVN nodokļu nosacījumus.</span><span class="sxs-lookup"><span data-stu-id="ce061-120">This field is not be available if the Apply sales tax taxation rules option is selected in the General ledger parameters page.</span></span>   
8. <span data-ttu-id="ce061-121">Laukā Maksājamais importa nodoklis atlasiet galveno kontu, kurā grāmatot ienākošos importa nodokļus, kas ir jāmaksā nodokļu iestādēm.</span><span class="sxs-lookup"><span data-stu-id="ce061-121">In the Use tax payable field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span>
    * <span data-ttu-id="ce061-122">Lai iegrāmatotu importa nodokli, ir jāatlasa opcija Importa nodoklis PVN grupas PVN kodā.</span><span class="sxs-lookup"><span data-stu-id="ce061-122">The Use tax option needs to be selected in the Sales tax code in the Sales tax group to post Use tax.</span></span> <span data-ttu-id="ce061-123">Ja opcija Lietot PVN nodokļu kārtulas ir atlasīta Virsgrāmatas parametros, nobīde tiek grāmatota transakcijas izdevumu kontā.</span><span class="sxs-lookup"><span data-stu-id="ce061-123">If Apply sales tax taxation rules option is selected in General ledger parameters, the offset is posted to the transaction’s expense account.</span></span>   
9. <span data-ttu-id="ce061-124">Laukā Apmaksas konts atlasiet galveno kontu, kurā tiks grāmatota neto bilance no Virsgrāmatas kontiem, kas norādīti laukos Maksājamais importa nodoklis un Saņemamais PVN.</span><span class="sxs-lookup"><span data-stu-id="ce061-124">In the Settlement account field, select the main account  that the net balance of the ledger accounts specified in the Use tax payable and Sales tax receivable fields will be posted.</span></span>
    * <span data-ttu-id="ce061-125">Bilance tiks izveidota, kad tiks palaists uzdevums PVN apmaksai un grāmatošanai.</span><span class="sxs-lookup"><span data-stu-id="ce061-125">The balance will be created when the Sales tax settle and post job is ran.</span></span>  <span data-ttu-id="ce061-126">Ja nodokļu iestāde maksājumu periodam ir saistīta ar kreditora kontu, bilance tiek grāmatota uz kreditora kontu.</span><span class="sxs-lookup"><span data-stu-id="ce061-126">If the Tax authority for the settlement period is associated with a vendor account, the balance is posted to the vendor account instead.</span></span>   
10. <span data-ttu-id="ce061-127">Laukā Kreditora termiņatlaide atlasiet galveno kontu, kurā grāmatot termiņatlaides PVN kodiem, kas saistīti ar šo Virsgrāmatas grāmatošanas grupu.</span><span class="sxs-lookup"><span data-stu-id="ce061-127">In the Vendor cash discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="ce061-128">Tas nav obligāti un, ja nav ievadīts neviens konts, tiks izmantots termiņatlaižu kodu galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="ce061-128">This is optional and if no account is entered,  the main account on Cash discount codes will be used.</span></span> <span data-ttu-id="ce061-129">Ja izmantojat PVN grupu opciju PVN atgriešana termiņatlaidēs, var būt noderīgi izmantot atšķirīgus kontus Virsgrāmatas grāmatošanas grupām.</span><span class="sxs-lookup"><span data-stu-id="ce061-129">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="ce061-130">Laukā Debitora termiņatlaide atlasiet galveno kontu, kurā grāmatot termiņatlaides PVN kodiem, kas saistīti ar šo Virsgrāmatas grāmatošanas grupu.</span><span class="sxs-lookup"><span data-stu-id="ce061-130">In the Customer case discount field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span>
    * <span data-ttu-id="ce061-131">Tas nav obligāti un, ja nav ievadīts neviens konts, tiks izmantots termiņatlaižu kodu galvenais konts.</span><span class="sxs-lookup"><span data-stu-id="ce061-131">This is optional and if no account is entered, the main account on the Cash discount codes will be used.</span></span> <span data-ttu-id="ce061-132">Ja izmantojat PVN grupu opciju PVN atgriešana termiņatlaidēs, var būt noderīgi izmantot atšķirīgus kontus Virsgrāmatas grāmatošanas grupām.</span><span class="sxs-lookup"><span data-stu-id="ce061-132">It can be useful to use different accounts per Ledger posting group if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
12. <span data-ttu-id="ce061-133">Noklikšķiniet uz Saglabāt.</span><span class="sxs-lookup"><span data-stu-id="ce061-133">Click Save.</span></span>
