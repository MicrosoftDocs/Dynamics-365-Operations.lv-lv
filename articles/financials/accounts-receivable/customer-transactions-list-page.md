## <a name="view-settlements"></a><span data-ttu-id="551f2-104">Skatīt nosegšanas darbības</span><span class="sxs-lookup"><span data-stu-id="551f2-104">View settlements</span></span>

<span data-ttu-id="551f2-105">Darbību rūts cilne **Skatīt segšanas** nodrošina ātru piekļuvi segšanas vēsturei un papildu informācijai par visu segšanas transakciju.</span><span class="sxs-lookup"><span data-stu-id="551f2-105">The **View settlements** tab on the action pane provides quick access to settlement history and more information about the whole settlement transaction.</span></span> <span data-ttu-id="551f2-106">Varat arī parādīt papildu transakcijas, kas ir saistītas ar atlasīto transakciju, vai nu tāpēc, ka tās ir šīs segšanas daļa, vai arī tie ir maksājumi, kas izveidoti tajā pašā maksājumu žurnālā.</span><span class="sxs-lookup"><span data-stu-id="551f2-106">You can also show additional transactions that are related to the selected transaction, either because they were part of the same settlement or because they are payments that were created in the same payment journal.</span></span>

1. <span data-ttu-id="551f2-107">Atlasiet **Debitoru parādi \> Debitori**.</span><span class="sxs-lookup"><span data-stu-id="551f2-107">Select **Accounts receivable \> Customers**.</span></span>
2. <span data-ttu-id="551f2-108">Atlasiet debitoru, kuram ir transakcijas, un pēc tam atlasiet cilni **Debitors \> Transakcija**.</span><span class="sxs-lookup"><span data-stu-id="551f2-108">Select a customer that has transactions, and then select **Customer \> Transactions**.</span></span>
3. <span data-ttu-id="551f2-109">Atlasiet meklējamo transakciju.</span><span class="sxs-lookup"><span data-stu-id="551f2-109">Select a transaction to research.</span></span>
4. <span data-ttu-id="551f2-110">Atlasiet darbību rūts cilni **Skatīt segšanas**.</span><span class="sxs-lookup"><span data-stu-id="551f2-110">Select the **View settlements** tab on the action pane.</span></span>

    <span data-ttu-id="551f2-111">Dialoglodziņa **Skatīt segšanas** ir redzama atlasītā transakcija un visi dokumenti, kas to sedz.</span><span class="sxs-lookup"><span data-stu-id="551f2-111">The **View settlements** dialog box shows the selected transaction and all documents that have been settled against it.</span></span> <span data-ttu-id="551f2-112">Šis dialoglodziņš līdzinās segšanas vēstures skatam, bet tas ietver visus saistītos dokumentus.</span><span class="sxs-lookup"><span data-stu-id="551f2-112">This dialog box resembles the settlement history view, but it includes all related documents.</span></span> 

5. <span data-ttu-id="551f2-113">No šī dialoglodziņa var veikt dažādus uzdevumus.</span><span class="sxs-lookup"><span data-stu-id="551f2-113">You can perform various tasks from this dialog box.</span></span> <span data-ttu-id="551f2-114">Atlasiet vienu vai vairākus vaučerus un pēc tam atlasiet vienu no tālāk norādītajām izvēlnēm.</span><span class="sxs-lookup"><span data-stu-id="551f2-114">Select one or more vouchers, and then select one of the following menus:</span></span>

   - <span data-ttu-id="551f2-115">**Skatīt uzskaiti** – tiek parādīti visi ar atlasītajiem dokumentiem saistītie vaučeri.</span><span class="sxs-lookup"><span data-stu-id="551f2-115">**View accounting** – All vouchers that are related to the selected documents appear.</span></span> <span data-ttu-id="551f2-116">Atlasiet **Aizvērt**, lai aizvērtu dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="551f2-116">Select **Close** to close the dialog box.</span></span>
   - <span data-ttu-id="551f2-117">**Eksportēt** – eksportēt atlasītos vaučerus uz programmu Microsoft Excel.</span><span class="sxs-lookup"><span data-stu-id="551f2-117">**Export** – Export the selected vouchers to Microsoft Excel.</span></span>
   - <span data-ttu-id="551f2-118">**Skatīt saistītos maksājumus** – tiek parādītas visas maksājumu žurnāla transakcijas, kas tika izveidotas maksājumu žurnālā, kas ir saistīts ar atlasīto dokumentu.</span><span class="sxs-lookup"><span data-stu-id="551f2-118">**View related payments** – All the payment journal transactions that were created in the payment journal that is related to the selected document appear.</span></span> <span data-ttu-id="551f2-119">Turklāt tiek parādītas visas segšanas, kas ir saistītas ar šiem maksājumiem.</span><span class="sxs-lookup"><span data-stu-id="551f2-119">In addition, all the settlements that are related to those payments appear.</span></span> <span data-ttu-id="551f2-120">Šīs izvēlnes marķējums arī tiek mainīts uz **Skatīt segšanas**.</span><span class="sxs-lookup"><span data-stu-id="551f2-120">The label of this menu also changes to **View settlements**.</span></span> <span data-ttu-id="551f2-121">Atlasiet **Skatīt segšanas**, lai parādītu tikai tās transakcijas, kuras tika parādītas, kad pirmoreiz atvērāt dialoglodziņu **Skatīt segšanas**.</span><span class="sxs-lookup"><span data-stu-id="551f2-121">Select **View settlements** to show only the transactions that were shown when you first opened the  **View settlements** dialog box.</span></span>
    - <span data-ttu-id="551f2-122">**Transakciju segšana** – šī izvēlne tiek parādīta, ja oriģinālais dokuments, kas tika atlasīts, netika pilnībā segts.</span><span class="sxs-lookup"><span data-stu-id="551f2-122">**Settle transactions** – This menu appears if the original document that was selected wasn't fully settled.</span></span> <span data-ttu-id="551f2-123">To atlasot, tiek parādīts dialoglodziņš **Transakciju segšana**, kurā var atlasīt transakcijas segšanai.</span><span class="sxs-lookup"><span data-stu-id="551f2-123">When you select it, the **Settle transactions** dialog box appears, where you can select transactions for settlement.</span></span>
    - <span data-ttu-id="551f2-124">**Transakciju atsaukšana** – šī izvēlne tiek parādīta, ja oriģinālais dokuments, kas tika atlasīts, tika pilnībā segts.</span><span class="sxs-lookup"><span data-stu-id="551f2-124">**Undo settlements** – This menu appears if the original document that was selected was fully settled.</span></span> <span data-ttu-id="551f2-125">To atlasot, tiek parādīts dialoglodziņš **Segšanu atsaukšana**, kurā var atsaukt segšanas darbības, kas tika veiktas šim dokumentam.</span><span class="sxs-lookup"><span data-stu-id="551f2-125">When you select it, the **Undo settlements** dialog box appears, where you can undo the settlements that were done for that document.</span></span>
