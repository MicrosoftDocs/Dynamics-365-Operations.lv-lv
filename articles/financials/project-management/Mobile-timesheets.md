## <a name="create-a-project-timesheet-on-your-mobile-device"></a><span data-ttu-id="04e11-117">Projekta darba laika uzskaites tabulas izveide mobilajā ierīcē</span><span class="sxs-lookup"><span data-stu-id="04e11-117">Create a project timesheet on your mobile device</span></span>

1.  <span data-ttu-id="04e11-118">Pievērsiet uzmanību Dynamics 365 URL lapai **Projektu pārvaldība un uzskaite** \> **Darba laika uzskaites tabulas** \> **Manas darba laika uzskaites tabulas (optimizētas mobilajām ierīcēm)**.</span><span class="sxs-lookup"><span data-stu-id="04e11-118">Note the Dynamics 365 URL for the **Project management and accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)** page.</span></span>

2.  <span data-ttu-id="04e11-119">Mobilās ierīces tīmekļa pārlūkprogrammā piekļūstiet URL, kas minēts iepriekšējā darbībā.</span><span class="sxs-lookup"><span data-stu-id="04e11-119">On your mobile device’s web browser, access the URL noted in the previous step.</span></span>
 
3.  <span data-ttu-id="04e11-120">Atlasiet **Projektu pārvaldība un uzskaite** \> **Darba laika uzskaites tabulas** \> **Manas darba laika uzskaites tabulas (optimizētas mobilajām ierīcēm)**.</span><span class="sxs-lookup"><span data-stu-id="04e11-120">Select **Project management and Accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)**.</span></span>

4.  <span data-ttu-id="04e11-121">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="04e11-121">Select **New**.</span></span>

5.  <span data-ttu-id="04e11-122">Atlasiet resursu.</span><span class="sxs-lookup"><span data-stu-id="04e11-122">Select a resource.</span></span>

6.  <span data-ttu-id="04e11-123">Tiek parādīti kalendāra datumi pašreizējai nedēļai.</span><span class="sxs-lookup"><span data-stu-id="04e11-123">The calendar dates for the current week are shown.</span></span>

7.  <span data-ttu-id="04e11-124">Noklikšķiniet uz **Labi**.</span><span class="sxs-lookup"><span data-stu-id="04e11-124">Click **OK**.</span></span>

8.  <span data-ttu-id="04e11-125">Noklikšķiniet uz **+**, lai pievienotu jaunu rindu.</span><span class="sxs-lookup"><span data-stu-id="04e11-125">Click **+** to add a new line.</span></span>

9.  <span data-ttu-id="04e11-126">Atlasiet debitoru un projekta informāciju par darba laika uzskaites tabulas rindu.</span><span class="sxs-lookup"><span data-stu-id="04e11-126">Select the customer and project information for the timesheet line.</span></span>

10. <span data-ttu-id="04e11-127">Ievadiet atlasītajai projekta rindai nostrādāto stundu skaitu.</span><span class="sxs-lookup"><span data-stu-id="04e11-127">Enter the hours that you worked for the selected project line.</span></span>

11. <span data-ttu-id="04e11-128">Pēc izvēles: ievadiet jebkādus ārējus un iekšējus komentārus par stundām.</span><span class="sxs-lookup"><span data-stu-id="04e11-128">Optional: Enter any external and internal comments for the hours.</span></span>

12. <span data-ttu-id="04e11-129">Atlasiet darbību Atpakaļ, lai saglabātu stundu informāciju un atgrieztos darba laika uzskaites tabulas detalizētajā skatā.</span><span class="sxs-lookup"><span data-stu-id="04e11-129">Select the back action to save the hour information and return to the timesheet details view.</span></span>

14. <span data-ttu-id="04e11-131">Kad ir pabeigta darba laika uzskaites tabulas rindu ievade, atlasiet **Darbplūsmas** \> darbību **Iesniegt**, lai iesniegtu darba laika uzskaites tabulu darbplūsmas apstiprināšanas procesā.</span><span class="sxs-lookup"><span data-stu-id="04e11-131">After you have completed entering the timesheet lines, select the **Workflow** \> **Submit** action to submit your timesheet to the workflow approval process.</span></span>
