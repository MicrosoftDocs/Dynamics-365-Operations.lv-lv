------------------------------------------------

1.  <span data-ttu-id="1967c-119">Pievērsiet uzmanību Dynamics 365 URL lapai **Projektu pārvaldība un uzskaite** \> **Darba laika uzskaites tabulas** \> **Manas darba laika uzskaites tabulas (optimizētas mobilajām ierīcēm)**.</span><span class="sxs-lookup"><span data-stu-id="1967c-119">Note the Dynamics 365 URL for the **Project management and accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)** page.</span></span>

2.  <span data-ttu-id="1967c-120">Mobilās ierīces tīmekļa pārlūkprogrammā piekļūstiet URL, kas minēts iepriekšējā darbībā.</span><span class="sxs-lookup"><span data-stu-id="1967c-120">On your mobile device’s web browser, access the URL noted in the previous step.</span></span>
 
3.  <span data-ttu-id="1967c-121">Atlasiet **Projektu pārvaldība un uzskaite** \> **Darba laika uzskaites tabulas** \> **Manas darba laika uzskaites tabulas (optimizētas mobilajām ierīcēm)**.</span><span class="sxs-lookup"><span data-stu-id="1967c-121">Select **Project management and Accounting** \> **Timesheets** \> **My timesheets(Optimized for mobile)**.</span></span>

4.  <span data-ttu-id="1967c-122">Atlasiet **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="1967c-122">Select **New**.</span></span>

5.  <span data-ttu-id="1967c-123">Atlasiet resursu.</span><span class="sxs-lookup"><span data-stu-id="1967c-123">Select a resource.</span></span>

6.  <span data-ttu-id="1967c-124">Tiek parādīti kalendāra datumi pašreizējai nedēļai.</span><span class="sxs-lookup"><span data-stu-id="1967c-124">The calendar dates for the current week are shown.</span></span>

7.  <span data-ttu-id="1967c-125">Noklikšķiniet uz **Labi**.</span><span class="sxs-lookup"><span data-stu-id="1967c-125">Click **OK**.</span></span>

8.  <span data-ttu-id="1967c-126">Noklikšķiniet uz **+**, lai pievienotu jaunu rindu.</span><span class="sxs-lookup"><span data-stu-id="1967c-126">Click **+** to add a new line.</span></span>

9.  <span data-ttu-id="1967c-127">Atlasiet debitoru un projekta informāciju par darba laika uzskaites tabulas rindu.</span><span class="sxs-lookup"><span data-stu-id="1967c-127">Select the customer and project information for the timesheet line.</span></span>

10. <span data-ttu-id="1967c-128">Ievadiet atlasītajai projekta rindai nostrādāto stundu skaitu.</span><span class="sxs-lookup"><span data-stu-id="1967c-128">Enter the hours that you worked for the selected project line.</span></span>

11. <span data-ttu-id="1967c-129">Pēc izvēles: ievadiet jebkādus ārējus un iekšējus komentārus par stundām.</span><span class="sxs-lookup"><span data-stu-id="1967c-129">Optional: Enter any external and internal comments for the hours.</span></span>

12. <span data-ttu-id="1967c-130">Atlasiet darbību Atpakaļ, lai saglabātu stundu informāciju un atgrieztos darba laika uzskaites tabulas detalizētajā skatā.</span><span class="sxs-lookup"><span data-stu-id="1967c-130">Select the back action to save the hour information and return to the timesheet details view.</span></span>

13. <span data-ttu-id="1967c-131">Pēc izvēles: ievadiet papildu darba laika uzskaites tabulas rindas.</span><span class="sxs-lookup"><span data-stu-id="1967c-131">Optional: Enter additional timesheet lines.</span></span>

14. <span data-ttu-id="1967c-132">Kad ir pabeigta darba laika uzskaites tabulas rindu ievade, atlasiet **Darbplūsmas** \> darbību **Iesniegt**, lai iesniegtu darba laika uzskaites tabulu darbplūsmas apstiprināšanas procesā.</span><span class="sxs-lookup"><span data-stu-id="1967c-132">After you have completed entering the timesheet lines, select the **Workflow** \> **Submit** action to submit your timesheet to the workflow approval process.</span></span>
