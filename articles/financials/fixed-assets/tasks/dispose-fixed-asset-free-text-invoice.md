# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="43442-103">Pamatlīdzekļa izslēgšana, izmantojot brīvā teksta rēķinu</span><span class="sxs-lookup"><span data-stu-id="43442-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="43442-104">Šajā sadaļā ir sniegta informācija par to, kā izslēgt pamatlīdzekli, izmantojot brīva teksta rēķinu.</span><span class="sxs-lookup"><span data-stu-id="43442-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="43442-105">Pārejiet uz sadaļu Debitori > Rēķini > Visi brīva teksta rēķini.</span><span class="sxs-lookup"><span data-stu-id="43442-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="43442-106">Noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="43442-106">Click New.</span></span>
3. <span data-ttu-id="43442-107">Laukā Debitora konts ievadiet vai atlasiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="43442-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="43442-108">Pārbaudiet noklusējuma rēķina datumu un labojiet, ja nepieciešams.</span><span class="sxs-lookup"><span data-stu-id="43442-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="43442-109">Pārbaudiet atlikušos noklusējuma virsraksta laukus, piemēram, Valūta, un labojiet, ja nepieciešamas.</span><span class="sxs-lookup"><span data-stu-id="43442-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="43442-110">Ievadiet aprakstu rēķina rindā.</span><span class="sxs-lookup"><span data-stu-id="43442-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="43442-111">Ievadiet vai atlasiet rēķina rindas galveno kontu.</span><span class="sxs-lookup"><span data-stu-id="43442-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="43442-112">Pārbaudiet noklusējuma PVN grupu un krājumu PVN grupu.</span><span class="sxs-lookup"><span data-stu-id="43442-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="43442-113">Ievadiet vienības cenu vai pamatlīdzekļa pārdošanas summu.</span><span class="sxs-lookup"><span data-stu-id="43442-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="43442-114">Noklikšķiniet uz Detalizēta informācija par rindu.</span><span class="sxs-lookup"><span data-stu-id="43442-114">Click Line details.</span></span>  
11. <span data-ttu-id="43442-115">Atlasiet pārdodamā pamatlīdzekļa numuru.</span><span class="sxs-lookup"><span data-stu-id="43442-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="43442-116">Noklikšķiniet uz Grāmatot.</span><span class="sxs-lookup"><span data-stu-id="43442-116">Click Post.</span></span>
