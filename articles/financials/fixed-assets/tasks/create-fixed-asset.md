# <a name="create-a-fixed-asset"></a><span data-ttu-id="59bd8-103">Pamatlīdzekļa izveide</span><span class="sxs-lookup"><span data-stu-id="59bd8-103">Create a fixed asset</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="59bd8-104">Šis uzdevuma ceļvedis izmanto USMF demonstrācijas uzņēmumu.</span><span class="sxs-lookup"><span data-stu-id="59bd8-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="59bd8-105">Tajā tiks izveidots jauns pamatlīdzeklis, izmantojot Pamatlīdzekļu saraksta lapu.</span><span class="sxs-lookup"><span data-stu-id="59bd8-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="59bd8-106">Ejiet uz **Navigācijas rūts > Moduļi > Pamatlīdzekļi > Pamatlīdzekļi > Pamatlīdzekļi**.</span><span class="sxs-lookup"><span data-stu-id="59bd8-106">Go to **Navigation pane > Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="59bd8-107">**Darbību rūtī** noklikšķiniet uz **Jauns**.</span><span class="sxs-lookup"><span data-stu-id="59bd8-107">On the **Action pane**, click **New**.</span></span>
3. <span data-ttu-id="59bd8-108">Ievadiet vai atlasiet vērtību laukā **Pamatlīdzekļa grupa**.</span><span class="sxs-lookup"><span data-stu-id="59bd8-108">In **the Fixed asset group** field, enter or select a value.</span></span> <span data-ttu-id="59bd8-109">Lauks **Numurs** izmantos noklusējuma vērtību, ja esat iespējojis **Pamatlīdzekļu funkcionalitātes automātiska numurēšana** **Pamatlīdzekļu parametros** un **Pamatlīdzekļu grupā**.</span><span class="sxs-lookup"><span data-stu-id="59bd8-109">The **Number** field will default if you have enabled **Autonumber fixed assets functionality** in the **Fixed assets parameters** and the **Fixed asset group**.</span></span>  <span data-ttu-id="59bd8-110">Ja tā nav, Jums jāievada unikāls kods pamatlīdzekļa identificēšanai.</span><span class="sxs-lookup"><span data-stu-id="59bd8-110">If not, you must enter a unique number to identify the fixed asset.</span></span>  
4. <span data-ttu-id="59bd8-111">Laukā **Nosaukums** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="59bd8-111">In the **Name** field, type a value.</span></span> <span data-ttu-id="59bd8-112">Ievadiet papildu informāciju, kas jūsu uzņēmumam ir nepieciešama par šo līdzekli.</span><span class="sxs-lookup"><span data-stu-id="59bd8-112">Enter the additional information that your business needs for this asset.</span></span>  
5. <span data-ttu-id="59bd8-113">**Darbību rūtī** noklikšķiniet uz **Grāmatas**.</span><span class="sxs-lookup"><span data-stu-id="59bd8-113">On the **Action pane**, click **Books**.</span></span>
6. <span data-ttu-id="59bd8-114">Laukā **Iegādes datums** ievadiet datumu.</span><span class="sxs-lookup"><span data-stu-id="59bd8-114">In the **Acquisition date** field, enter a date.</span></span>
7. <span data-ttu-id="59bd8-115">Laukā **Iegādes cena** ievadiet skaitli.</span><span class="sxs-lookup"><span data-stu-id="59bd8-115">In the **Acquisition price** field, enter a number.</span></span>
    - <span data-ttu-id="59bd8-116">Ievadiet papildinformāciju, kas jūsu uzņēmumam ir nepieciešama par šo grāmatu.</span><span class="sxs-lookup"><span data-stu-id="59bd8-116">Enter the additional information that your business needs for this book.</span></span>  
    - <span data-ttu-id="59bd8-117">Ievadiet papildinformāciju, kas jūsu uzņēmumam ir nepieciešama par atlikušajām grāmatām.</span><span class="sxs-lookup"><span data-stu-id="59bd8-117">Enter the additional information that your business needs for the remaining books.</span></span>  
8. <span data-ttu-id="59bd8-118">Aizvērt lapu.</span><span class="sxs-lookup"><span data-stu-id="59bd8-118">Close the page.</span></span>
