# <a name="eur-00015-party-search-using-vat-id"></a>EUR-00015 Pušu meklēšana pēc PVN ID

[!include [task guide banner](../../includes/task-guide-banner.md)]

Šajā procedūrā ir parādīts, kā pabeigt puses meklēšanu, izmantojot reģistrācijas ID. Lai varētu veikt šo procedūru, jums ir jāiestata PVN ID un jāievada PVN ID kreditoriem, debitoriem vai juridiskajām personām.

Šī procedūra attiecas uz visām Eiropas valstīm/reģioniem. Šī procedūra ir izveidota, izmantojot demonstrācijas datu uzņēmumu DEMF, kura primārā adrese ir Vācijā. Šī procedūra ir paredzēta kreditoriem maksājamo parādu vadītājam vai debitoru parādu vadītājam. Šī procedūra ir paredzēta līdzeklim, kas tika pievienots Dynamics 365 for Operations versijā 1611.

1. Dodieties uz Organizācijas administrēšana > Globālā adrešu grāmata > Globālā adrešu grāmata.
2. Noklikšķiniet uz Reģistrācijas ID meklēšana.
3. Noklikšķiniet uz Pievienot.
4. Laukā Reģistrācijas tips ievadiet vai atlasiet kādu vērtību.
    * Piemēram, ja vēlējāties atrastu puses, izmantojot PVN ID tipa reģistrācijas ID, atlasiet PVN ID.  
5. Laukā Valsts/reģions ievadiet vai atlasiet kādu vērtību.
    * Piemēram, ievadiet DEU.  
6. Laukā Reģistrācijas numurs ierakstiet vērtību.
7. Noklikšķiniet uz Meklēt.
    * Tiks parādītas visas puses ar šo reģistrācijas ID.
