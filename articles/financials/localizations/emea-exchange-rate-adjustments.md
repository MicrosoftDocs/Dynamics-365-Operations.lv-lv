[!include[banner](../includes/banner.md)]


<span data-ttu-id="99099-104">Šajā tēmā ir sniegta informācija par maiņas kursa korekciju juridiskām personām Igaunijā, Ungārijā, Čehijas Republikā, Latvijā, Lietuvā, Polijā un Krievijā.</span><span class="sxs-lookup"><span data-stu-id="99099-104">This topic provides information about the exchange rate adjustment functionality for users in legal entities in Estonia, Hungary, Czech Republic, Latvia, Lithuania, Poland, and Russia.</span></span>

-   <span data-ttu-id="99099-106">Maiņas kursa korekciju grāmatojumus var atcelt kā sākotnējo korekciju labojumus (negatīvas summas).</span><span class="sxs-lookup"><span data-stu-id="99099-106">Postings of exchange rate adjustments can be reversed as corrections (negative amounts) to the original adjustments.</span></span>
-   <span data-ttu-id="99099-107">Ja tiek grāmatotas secīgi nerealizētas korekcijas, neatkarīgi no tā, vai korekcijas pārstāv peļņu vai zaudējumus, tiks izmantots tas pats Virsgrāmatas grāmatošanas konts un darbības tips.</span><span class="sxs-lookup"><span data-stu-id="99099-107">When consecutive unrealized adjustments are posted, the same general ledger posting account and transaction type are used, regardless of whether the adjustments represent a gain or a loss.</span></span>
-   <span data-ttu-id="99099-108">Aprēķinātais maiņas kursa ienākums vienmēr tiek grāmatots ienākumu kontos un aprēķinātie maiņas kursa zaudējumi vienmēr tiek grāmatoti zaudējumu kontos.</span><span class="sxs-lookup"><span data-stu-id="99099-108">Calculated exchange rate gains are always posted to gain accounts, and calculated exchange rate losses are always posted to loss accounts.</span></span>

<span data-ttu-id="99099-109">Juridiskās personas, kuru primārā adrese ir Čehijas Republikā, var izmantot speciālās maiņas kursa korekciju metodes.</span><span class="sxs-lookup"><span feature introduces aren't applied.</span></span> <span data-ttu-id="99099-112">Nerealizētā un realizētā peļņa vai zaudējumi tiek aprēķināti pret pēdējo izmantoto maiņas kursu.</span><span
