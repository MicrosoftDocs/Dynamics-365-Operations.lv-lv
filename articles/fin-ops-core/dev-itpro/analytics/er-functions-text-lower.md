# <a name="LOWER">LOWER ER funkcija</a>

[!include [banner](../includes/banner.md)]

`LOWER` funkcija atgriež norādīto teksta virkni kā *Virknes* vērtību pēc tam, kad tā ir konvertēta mazajos burtos.

## <a name="syntax"></a>Sintakse

```
LOWER (text)
```

## <a name="arguments"></a>Argumenti

`text`: *Virkne*

*Virknes* vērtība, kas konkretizē tekstu.

## <a name="return-values"></a>Atgrieztās vērtības

*Virkne*

Iegūtā teksta vērtība.

## <a name="example"></a>Paraugs

`LOWER ("Sample")` atgriež **"sample"**.
