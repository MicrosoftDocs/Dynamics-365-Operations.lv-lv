# <a name="upper-er-function"></a>UPPER ER funkcija

[!include [banner](../includes/banner.md)]

`UPPER` funkcija atgriež norādīto teksta virkni kā *Virknes* vērtību pēc tam, kad tā ir konvertēta lielajos burtos.

## <a name="syntax"></a>Sintakse

```vb
UPPER (text )
```

## <a name="arguments"></a>Argumenti

`text`: *Virkne*

*Virknes* tipa datu avota datu tipa derīgs ceļš.

## <a name="return-values"></a>Atgrieztās vērtības

*Virkne*

Iegūtā teksta vērtība.

## <a name="example"></a>Paraugs

`UPPER ("Sample")` atgriež **"SAMPLE"**.
