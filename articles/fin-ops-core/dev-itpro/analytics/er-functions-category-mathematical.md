# <a name="list-of-er-functions-in-the-mathematical-category"></a>ER funkciju saraksts matemātikas kategorijā

[!include [banner](../includes/banner.md)]

Elektronisko pārskatu (ER) matemātiskās funkcijas var izmantot, lai veiktu daudzus vispārējus matemātiskos aprēķinus. Šajā tēmā ir sniegts šo funkciju kopsavilkums.

## <a name="list-of-supported-functions"></a>Atbalstīto funkciju saraksts

| Funkcija | Apraksts |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Šī funkcija atgriež norādītā numura absolūto vērtību (moduli) kā *Reālu* vērtību. Tas ir, tā atgriež skaitli bez tā zīmes. |
| [Jauda](er-functions-mathematical-power.md)         | Šī funkcija atgriež *Reālu* vērtību, kas atspoguļo rezultātu norādītā pozitīvā skaitļa palielināšanai līdz norādītajai jaudai. |
| [Apaļots](er-functions-mathematical-round.md)         | Šī funkcija atgriež norādīto numuru kā *Reālu* vērtību pēc tam, kad tas ir noapaļots līdz norādītajam ciparu skaitam aiz komata. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Šī funkcija atgriež norādīto numuru kā *Reālu* vērtību pēc tam, kad tas ir noapaļots uz leju līdz norādītajam ciparu skaitam aiz komata. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Šī funkcija atgriež norādīto numuru kā *Reālu* vērtību pēc tam, kad tas ir noapaļots uz augšu līdz norādītajam ciparu skaitam aiz komata. |
