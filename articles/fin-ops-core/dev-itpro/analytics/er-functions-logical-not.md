# <a name="not-er-function"></a>NOT ER funkcija

[!include [banner](../includes/banner.md)]

`NOT` funkcija atgriež norādītā nosacījuma apgriezto loģisko vērtību kā *Būla* vērtību.

## <a name="syntax"></a>Sintakse

```vb
NOT (condition)
```

## <a name="arguments"></a>Argumenti

`condition`: *Būla*

Derīga nosacījuma izteiksme, kas ir jāatsauc.

## <a name="return-values"></a>Atgrieztās vērtības

*Būla*

Iegūtā *Būla* vērtība.

## <a name="example"></a>Paraugs

`NOT (TRUE)` atgriež **FALSE**.
