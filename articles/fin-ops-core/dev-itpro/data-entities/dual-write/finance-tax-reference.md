# <a name="access-to-finance-and-tax-reference-data"></a>Piekļuve finanšu un nodokļu atsauces datiem

[!include [banner](../../includes/banner.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Katrs uzņēmums strādā ar finanšu datu pamata kopumu, piemēram, finanšu kalendāra gadu, valūtu, ar ko darījumi tiek veikti konti, kuros nauda, lai palaistu biznesu, nonāk vai izdziest, nodokļu likmes un pārskaitījums. Šie dati atrodas Finance and Operations lietojumprogrammās. Tomēr tas ir pakļauts Dataverse, lai modeļa darbinātās programmas Microsoft Dynamics 365 var būt viens avots finanšu un nodokļu datiem. Šādā veidā dati ir vienādi visā biznesa ekosistēmā. 

Finanšu un nodokļu dati ir integrēti, izmantojot šādus kartējumus:

+ [Integrētā virsgrāmata](ledger-mapping.md)
+ [Integrētie nodokļu pamatdati](tax-mapping.md)
