# <a name="assign-users-to-security-roles"></a><span data-ttu-id="caa00-103">Drošības lomu piešķiršana lietotājiem</span><span class="sxs-lookup"><span data-stu-id="caa00-103">Assign users to security roles</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="caa00-104">Lai izmantotu citas iespējas, kas nav kopīgas iespējas Finance and Operations programmās, lietotājiem jābūt piešķirtām drošības lomām.</span><span class="sxs-lookup"><span data-stu-id="caa00-104">To use anything other than common capabilities in Finance and Operations apps, users must be assigned to security roles.</span></span> <span data-ttu-id="caa00-105">Jūs varat piešķirt lietotājus lomām automātiski, pamatojoties uz kārtulām un biznesa datiem, izslēgt lietotājus no automātiskās lomas piešķires vai pievienot lietotājus lomām manuāli.</span><span class="sxs-lookup"><span data-stu-id="caa00-105">You can assign users to roles automatically, based on rules and business data, exclude users from automatic role assignment, or add users to roles manually.</span></span>

<span data-ttu-id="caa00-107">Šī procedūra skaidro kā sistēmas administratori var automātiski piešķirt lietotājiem lomas, pamatojoties uz biznesa datiem.</span><span class="sxs-lookup"><span data-stu-id="caa00-107">This procedure explains how system administrators can automatically assign users to roles, based on business data.</span></span> 
1. <span data-ttu-id="caa00-108">Dodieties uz **Navigācijas rūts > Moduļi > Sistēmas administrēšana > Drošība > Piešķirt lomas lietotājiem**.</span><span class="sxs-lookup"><span data-stu-id="caa00-108">Go to **Navigation pane > Modules > System administration > Security > Assign users to roles**.</span></span>
2. <span data-ttu-id="caa00-109">Kokā atlasiet 'Uzskaites supervizors'.</span><span class="sxs-lookup"><span data-stu-id="caa00-109">In the tree, select 'Accounting supervisor'.</span></span> <span data-ttu-id="caa00-110">Atlasiet lomu, kurai vēlaties konfigurēt kārtulu.</span><span class="sxs-lookup"><span data-stu-id="caa00-110">Select the role that you want to configure the rule for.</span></span> <span data-ttu-id="caa00-111">Šajā piemērā atlasiet Uzskaites supervizoru.</span><span class="sxs-lookup"><span data-stu-id="caa00-111">In this example, select Accounting supervisor.</span></span> 
3. <span data-ttu-id="caa00-112">Atlasiet **Pievienot kārtulu**, lai atvērtu dialoglodziņa izvēlni.</span><span class="sxs-lookup"><span data-stu-id="caa00-112">Select **Add rule** to open the dialog menu.</span></span>
4. <span data-ttu-id="caa00-113">Sarakstā **Vaicājuma atlase** atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="caa00-113">In the **Select a query** list, find and select the desired record.</span></span> <span data-ttu-id="caa00-114">Atlasiet vaicājumu, lai varētu izmantot šo kārtulu.</span><span class="sxs-lookup"><span data-stu-id="caa00-114">Select the query to use for this rule.</span></span>  
5. <span data-ttu-id="caa00-115">Sarakstā **Dalības kārtulas nosaukums** noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="caa00-115">In the **Membership rule name** list, click the link in the selected row.</span></span>
6. <span data-ttu-id="caa00-116">Atlasiet **Rediģēt vaicājumu**.</span><span class="sxs-lookup"><span data-stu-id="caa00-116">Select **Edit query**.</span></span> <span data-ttu-id="caa00-117">Rediģējiet vaicājumu pēc nepieciešamības.</span><span class="sxs-lookup"><span data-stu-id="caa00-117">Edit the query, as needed.</span></span>  
7. <span data-ttu-id="caa00-118">Atlasiet **Labi**.</span><span class="sxs-lookup"><span data-stu-id="caa00-118">Select **OK**.</span></span>
8. <span data-ttu-id="caa00-119">Atlasiet **Palaist automātisko lomu piešķiri**.</span><span class="sxs-lookup"><span data-stu-id="caa00-119">Select **Run automatic role assignment**.</span></span>
9. <span data-ttu-id="caa00-120">Doties uz **Navigācijas rūts > Moduļi > Sistēmas administrēšana >Lietotāji > Lietotāji** (ideālā gadījumā atsevišķā pārlūka cilnē).</span><span class="sxs-lookup"><span data-stu-id="caa00-120">Go to **Navigation pane > Modules > System administration > Users > Users** (ideally in a separate browser tab).</span></span>
10. <span data-ttu-id="caa00-121">Pārskatiet dažādiem lietotājiem piešķirtās lomas, lai apstiprinātu, ka lomas piešķires vaicājums ir bijis pareizs.</span><span class="sxs-lookup"><span data-stu-id="caa00-121">Review the roles assigned to various users to confirm that the role assignment query was correct.</span></span> <span data-ttu-id="caa00-122">Pielāgojiet un vajadzības gadījumā palaidiet atkārtoti.</span><span class="sxs-lookup"><span data-stu-id="caa00-122">Adjust and re-run if needed.</span></span>

## <a name="manually-assign-users-to-roles"></a><span data-ttu-id="caa00-138">Manuāla lomu piešķiršana lietotājiem</span><span class="sxs-lookup"><span data-stu-id="caa00-138">Manually assign users to roles</span></span>

1. <span data-ttu-id="caa00-141">Dodieties uz **Navigācijas rūts > Moduļi > Sistēmas administrēšana > Drošība > Piešķirt lomas lietotājiem**.</span><span class="sxs-lookup"><span data-stu-id="caa00-141">Go to **Navigation pane > Modules > System administration > Security > Assign users to roles**.</span></span>
2. <span data-ttu-id="caa00-142">Shēmā atlasiet lomu un izvēlnē **Lomai piešķirtie lietotāji** atlasiet **Manuāli piešķirt/izslēgt lietotājus**.</span><span class="sxs-lookup"><span data-stu-id="caa00-142">In the tree, select a role, and in the **Users assigned to role** menu, select **Manually assign / exclude users**.</span></span>
4. <span data-ttu-id="caa00-143">Sadaļā **Piešķirt lietotājus vai izslēgt lietotājus no lomas** lietotājiem, kuriem nav piešķirta loma, tiek uzskaitīti ar **Piešķires režīmu** iestatītu uz **Nav**.</span><span class="sxs-lookup"><span data-stu-id="caa00-143">In the **Assign users to or exclude users from role**, users that have not been assigned the role are listed with the **Assignment mode** set to **None**.</span></span> <span data-ttu-id="caa00-144">Atlasiet vienu vai vairākus lietotājus, kuriem jāpiešķir loma.</span><span class="sxs-lookup"><span data-stu-id="caa00-144">Select one or more users that should be assigned the role.</span></span>
