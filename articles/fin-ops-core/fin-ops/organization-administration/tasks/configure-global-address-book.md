[!include [banner](../../includes/banner.md)]

<span data-ttu-id="47407-104">Izmantojiet šo procedūru, lai iestatītu noklusētās vērtības un drošības politikas globālajai adrešu grāmatai.</span><span class="sxs-lookup"><span data-stu-id="47407-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="47407-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo uzdevumu, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="47407-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="47407-106">Šis uzdevums ir paredzēts plānošanas un konfigurācijas grupai.</span><span class="sxs-lookup"><span data-stu-id="47407-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="47407-107">Navigācijas rūtī dodieties uz **Moduļi > Organizācijas administrēšana > Globālā adrešu grāmata > Globālās adrešu grāmatas parametri**.</span><span class="sxs-lookup"><span data-stu-id="47407-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>

3. <span data-ttu-id="47407-109">**Dzēst puses, kurām nav lomu** atlasiet, vai dzēst puses, kurām nav piešķirta loma.</span><span class="sxs-lookup"><span data-stu-id="47407-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="47407-110">**Izmantot dublikātu pārbaudi** atlasiet, vai pārbaudīt ierakstu dublikātus.</span><span class="sxs-lookup"><span data-stu-id="47407-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="47407-111">**Parādīt DUNS numuru adresēs** atlasiet, vai parādīt DUNS numuru adresēs.</span><span class="sxs-lookup"><span data-stu-id="47407-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="47407-112">**Pārbaudīt unikālo DUNS numuru** atlasiet, vai pārbaudīt unikālos DUNS numurus.</span><span class="sxs-lookup"><span data-stu-id="47407-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="47407-113">Laukā **Puse** atlasiet opciju.</span><span class="sxs-lookup"><span data-stu-id="47407-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="47407-114">Laukā **Debitors** atlasiet opciju.</span><span class="sxs-lookup"><span data-stu-id="47407-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="47407-115">Laukā **Kreditors** atlasiet opciju.</span><span class="sxs-lookup"><span data-stu-id="47407-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="47407-116">Laukā **Potenciālais klients** atlasiet opciju.</span><span class="sxs-lookup"><span data-stu-id="47407-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="47407-117">Laukā **Konkurents** atlasiet opciju.</span><span class="sxs-lookup"><span data-stu-id="47407-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="47407-118">Noklikšķiniet uz cilnes **Privāto vietu drošība**.</span><span class="sxs-lookup"><span data-stu-id="47407-118">Click the **Private location security** tab.</span></span>

14. <span data-ttu-id="47407-121">Noklikšķiniet uz **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="47407-121">Click **Save**.</span></span>
