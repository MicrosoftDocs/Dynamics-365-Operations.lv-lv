# <a name="configure-address-books"></a>Adrešu grāmatu konfigurēšana

[!include [banner](../../includes/banner.md)]

Izmantojiet šo procedūru un lēmumus, ko pieņemāt globālās adrešu grāmatas un papildu adrešu grāmatas raksta konfigurācijas plānošanā, lai iestatītu papildu adrešu grāmatas jūsu organizācijai.

Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo uzdevumu, ir USMF.


## <a name="configure-address-books"></a>Adrešu grāmatu konfigurēšana
1. **Navigācijas rūtī** ejiet uz **Moduļi > Organizācijas administrēšana > Globālā adrešu grāmata > Adrešu grāmatas**.
2. Klikšķiniet **Jauns**.
3. Laukā **Nosaukums** ierakstiet kādu vērtību.
4. Laukā **Apraksts** ierakstiet kādu vērtību.
5. Noklikšķiniet uz **Saglabāt**.
6. Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.
7. Noklikšķiniet uz bultiņas, lai pievienotu atlasītās pieejamās grupas adrešu grāmatai.
8. Noklikšķiniet uz **Saglabāt**.
