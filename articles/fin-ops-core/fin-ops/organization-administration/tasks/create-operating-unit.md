# <a name="create-an-operating-unit"></a><span data-ttu-id="34bdd-103">Pārvaldības struktūrvienības izveide</span><span class="sxs-lookup"><span data-stu-id="34bdd-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="34bdd-104">Pārvaldības struktūrvienība ir organizācija, kas tiek lietota, lai sadalītu ekonomisko resursu un operatīvo procesu kontroli biznesā.</span><span class="sxs-lookup"><span data-stu-id="34bdd-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="34bdd-105">Cilvēkiem pārvaldības struktūrvienībā ir pienākums maksimāli izmantot ierobežotos resursus, uzlabot procesus un atskaitīties par savu sniegumu.</span><span class="sxs-lookup"><span data-stu-id="34bdd-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="34bdd-106">Pārvaldības struktūrvienību veidi ietver izmaksu centrus, biznesa vienības, nodaļas un vērtību plūsmas.</span><span class="sxs-lookup"><span data-stu-id="34bdd-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="34bdd-107">Lai izveidotu pārvaldības struktūrvienību, veiciet tālākminētās darbības.</span><span class="sxs-lookup"><span data-stu-id="34bdd-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="34bdd-108">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="34bdd-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="34bdd-109">Dodieties uz **Navigācijas rūts > Moduļi > Organizācijas administrēšana > Organizācijas > Pārvaldības struktūrvienības**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="34bdd-110">Noklikšķiniet uz **Jauns**, lai atvērtu nolaižamo dialoglodziņu.</span><span class="sxs-lookup"><span data-stu-id="34bdd-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="34bdd-111">Sarakstā atrodiet un atlasiet vajadzīgo ierakstu.</span><span class="sxs-lookup"><span data-stu-id="34bdd-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="34bdd-112">Atlasiet, kāda tipa pārvaldības struktūrvienību vēlaties veidot.</span><span class="sxs-lookup"><span data-stu-id="34bdd-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="34bdd-113">Sarakstā noklikšķiniet uz saites atlasītajā rindā.</span><span class="sxs-lookup"><span data-stu-id="34bdd-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="34bdd-114">Laukā **Nosaukums** ierakstiet kādu vērtību.</span><span class="sxs-lookup"><span data-stu-id="34bdd-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="34bdd-115">Ja nepieciešams, izvērsiet sadaļu **Vispārēji**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="34bdd-116">Sniedziet vispārēju informāciju par pārvaldības struktūrvienību, piemēram, identifikācijas numuru, DUNS numuru un vadītāju.</span><span class="sxs-lookup"><span data-stu-id="34bdd-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="34bdd-117">Ja nepieciešams, izvērsiet sadaļu **Adreses**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="34bdd-118">Ievadiet informāciju par adresi, tādu kā ielas nosaukumu un numuru, pasta indeksa kodu un pilsētu.</span><span class="sxs-lookup"><span data-stu-id="34bdd-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="34bdd-119">Noklikšķiniet uz **Pievienot**, lai ievadītu jaunu adreses ierakstu, vai noklikšķiniet uz Rediģēt, lai mainītu jau esošu adreses ierakstu.</span><span class="sxs-lookup"><span data-stu-id="34bdd-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="34bdd-120">Ja nepieciešams, izvērsiet sadaļu **Kontaktinformācija**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="34bdd-121">Ierakstiet informāciju par komunikācijas veidiem, piemēram, e-pasta adreses, URL un tālruņa numurus.</span><span class="sxs-lookup"><span data-stu-id="34bdd-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="34bdd-122">Lai ievadītu jaunu saziņas ierakstu, noklikšķiniet uz Jauns.</span><span class="sxs-lookup"><span data-stu-id="34bdd-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="34bdd-123">Lai modificētu esošo sakaru ierakstu, noklikšķiniet uz **Vairāk opciju > Detalizēti**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="34bdd-124">Ir arī iespēja mainīt **Operatīvās struktūrvienības numuru** pēc vajadzības.</span><span class="sxs-lookup"><span data-stu-id="34bdd-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="34bdd-125">Ņemiet vērā, ka šis numurs ir unikāls identifikators atbilstīgajam **Puses** ierakstam un nevar būt tāds pats kā cita operatīva struktūrvienība.</span><span class="sxs-lookup"><span data-stu-id="34bdd-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="34bdd-126">Atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="34bdd-126">Select **Save**.</span></span>
