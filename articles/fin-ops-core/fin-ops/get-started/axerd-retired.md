# <a name="axerd-page-has-been-retired"></a>Lapa AxErd ir noņemta

[!include [banner](../includes/banner.md)]

Diemžēl mums bija jānoņem AxErd tīmekļa lapa, kurā bija pieejamas daudzu Microsoft Dynamics AX 2012 tabulu entītiju relāciju diagrammas (Entity relationship diagram — ERD). Gan projekta pirmkods, gan tīmekļa lapas HTML faili tika publicēti GitHub repozitorijā: https://github.com/Microsoft/ax-2012-doc-tools.

HTML lapas atrodas repozitorijā šeit: [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm).

Aprakstu par ERD izveidi skatiet šeit: [https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm](https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm).

Risinājums atrodas AxErd mapes saknē: [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd).
