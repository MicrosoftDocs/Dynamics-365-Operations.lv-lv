# <a name="change-the-banner-or-logo"></a><span data-ttu-id="0072b-103">Reklāmkaroga vai logotipa maiņa</span><span class="sxs-lookup"><span data-stu-id="0072b-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0072b-104">Šajā procedūrā uzskaitītas darbības, kuras sistēmas administratori var izmantot, lai atjauninātu reklāmkarogu vai logotipa attēlu, kas tiek rādīts juridiskai personai.</span><span class="sxs-lookup"><span data-stu-id="0072b-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="0072b-105">Demonstrācijas datu uzņēmums, kas tiek izmantots, lai izveidotu šo procedūru, ir USMF.</span><span class="sxs-lookup"><span data-stu-id="0072b-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="0072b-106">Dodieties uz **Navigācijas rūts > Moduļi > Organizācijas administrēšana > Organizācijas > Juridiskas personas**.</span><span class="sxs-lookup"><span data-stu-id="0072b-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="0072b-107">Kreisajā pusē esošajā sarakstā atlasiet juridisko personu, kurai vēlaties augšupielādēt reklāmkarogu vai logotipa attēlu.</span><span class="sxs-lookup"><span data-stu-id="0072b-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="0072b-108">Ja tā ir jau atlasīta, pārejiet uz nākamo darbību.</span><span class="sxs-lookup"><span data-stu-id="0072b-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="0072b-109">Izvērsiet cilni **Informācijas paneļa attēls**.</span><span class="sxs-lookup"><span data-stu-id="0072b-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="0072b-110">Atlasiet **Mainīt**.</span><span class="sxs-lookup"><span data-stu-id="0072b-110">Select **Change**.</span></span>
    
    <span data-ttu-id="0072b-111">Ideālā reklāmkaroga attēla izšķirtspēja ir 1920 x 281 pikseļi.</span><span class="sxs-lookup"><span data-stu-id="0072b-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="0072b-112">Ideālais logotipa attēla platums ir 350 pikseļi.</span><span class="sxs-lookup"><span data-stu-id="0072b-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="0072b-113">Atlasiet **Saglabāt**.</span><span class="sxs-lookup"><span data-stu-id="0072b-113">Select **Save**.</span></span>
6. <span data-ttu-id="0072b-114">Dodieties uz **Navigācijas rūts > Moduļi > Standarta > Standarta > Noklusējuma informācijas panelis**.</span><span class="sxs-lookup"><span data-stu-id="0072b-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="0072b-115">Jaunajam reklāmkarogam vai logotipa attēlam būtu jābūt redzamam informācijas panelī.</span><span class="sxs-lookup"><span data-stu-id="0072b-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="0072b-116">Var būt nepieciešams mainīt savu uzņēmumu uz to, kuram augšupielādējāt reklāmkarogu, izmantojot uzņēmuma atlasītāju.</span><span class="sxs-lookup"><span data-stu-id="0072b-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
