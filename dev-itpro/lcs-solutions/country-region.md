# <a name="localization-and-regulatory-features"></a>Lokalizāciju un pārvaldes funkcijas

Microsoft Dynamics 356 operācijām iekļauj funkcionalitāti, kādu valsts/reģiona specifiskām regulām un procesiem. Šī funkcija tiek iespējota, pamatojoties uz aktīvo tiesību subjekta primārā adrese. Šajā tēmā ir ietverta sarakstos resursi, kas var palīdzēt jums uzzināt vairāk par valsts/reģiona specifisko funkcionalitāti.
